# C# 随机数课堂练习（趣味版）

## 练习1：今天谁值日？

### 题目

班级有 **8名学生**，编号为 **1~8**。

编写程序随机选出 **今天的值日生**。

### 运行示例

```
今天的值日生是：3号同学
```

### 要求

* 使用 `Random`
* 随机数范围：1~8

### 知识点

* `Random`
* `Next(min,max)`

### 参考代码

```csharp
Random r = new Random();
int num = r.Next(1, 9);

Console.WriteLine("今天的值日生是：" + num + "号同学");
```

# 练习3：6位随机密码生成器

### 题目

# 练习5：老虎机小游戏（课堂气氛神器）

### 题目

模拟一个 **简单老虎机**。

# Random 最重要知识总结（给学生）

```csharp
Random r = new Random();

//0~9
r.Next(10);

//1~10
r.Next(1,11);

//随机多个
int a = r.Next(1,10);
int b = r.Next(1,10);
```

记忆口诀：

```
Next(最大值)       → 从0开始
Next(最小,最大)    → 左闭右开
```

例如：

```
Next(1,11) → 1~10
```
