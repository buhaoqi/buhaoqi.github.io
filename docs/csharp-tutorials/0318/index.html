3月18日 do while循环语句

案例1：密码验证​

string password;

do
{
    Console.Write("请输入密码：");
    password = Console.ReadLine();

} while (password != "123456");

Console.WriteLine("登录成功！");

✔ 至少要输入一次

string username;
string password;
int count = 0;

do
{
    Console.Write("用户名：");
    username = Console.ReadLine();

    Console.Write("密码：");
    password = Console.ReadLine();

    if (username == "admin" && password == "123")
    {
        Console.WriteLine("登录成功！");
        break;
    }

    count++;
    Console.WriteLine("登录失败");

} while (count < 3);

if (count == 3)
{
    Console.WriteLine("账号已锁定");
}

一、do while语句是什么​

do while 是一种执行一次再判断的循环语句。

👉 核心特点：

do while 循环特别适合：当条件一开始就不成立

二、基本语法​

do
{
    // 循环体
} while (条件);

⚠️ 注意：

  • while (条件); 后面必须有分号 ;
  • 条件是布尔表达式（true / false）

三、执行流程​

执行顺序如下：

① 先执行一次代码块
② 再判断条件
③ 如果为 true → 继续循环
④ 如果为 false → 结束

四、与 while 的本质区别（重点）​

对比项whiledo while
判断时机先判断后判断
执行次数可能 0 次至少 1 次
使用场景条件驱动必须执行一次

案例 1：输出 Hello World​

int i = 10;

// while
while (i < 5)
{
    Console.WriteLine("Hello World");
}

// do while
do
{
    Console.WriteLine("Hello World");
} while (i < 5);

👉 输出结果：

Hello World

✔ while：一次都不执行 ✔ do while：执行了一次

案例3：游戏菜单​

int choice;

do
{
    // 菜单一定会先显示 1 次 → 这就是 do while 的优势
    Console.WriteLine("===== 游戏菜单 =====");
    Console.WriteLine("1. 开始游戏");
    Console.WriteLine("2. 退出");
    Console.Write("请输入你的选择：");
    
    choice = int.Parse(Console.ReadLine());

    // 根据选择做出回应，让程序更完整
    if (choice == 1)
    {
        Console.WriteLine("游戏开始！祝你好运～\n");
    }
    else if (choice == 2)
    {
        Console.WriteLine("退出成功，欢迎下次再来！");
    }
    else
    {
        Console.WriteLine("输入错误，请重新选择！\n");
    }

} while (choice != 2); // 选择不是2就一直循环

✔ 菜单必须先显示一次

案例 4：游戏循环（常见于小游戏）​

string answer;

do
{
    Console.WriteLine("继续游戏？(y/n)");
    answer = Console.ReadLine();

} while (answer == "y");

4️⃣ 累加求和​

int sum = 0;
int i = 1;

do
{
    sum += i;
    i++;
} while (i <= 5);

Console.WriteLine(sum);

5️⃣ 菜单 + break 控制​

int choice;

do
{
    Console.WriteLine("1.登录");
    Console.WriteLine("2.退出");
    choice = int.Parse(Console.ReadLine());

    if (choice == 2)
        break;

} while (true);

✔ 常见写法：do while(true) + break

六、进阶用法

1️⃣ 搭配 continue​

int i = 0;

do
{
    i++;

    if (i == 3)
        continue;

    Console.WriteLine(i);

} while (i < 5);

👉 输出：

1
2
4
5

2️⃣ 无限循环​

do
{
    Console.WriteLine("死循环");
} while (true);

✔ 必须用 break 才能退出

七、常见错误（新手必看）

❌ 错误 1：忘记分号​

} while (i < 5)   // ❌ 少了 ;

✔ 正确：

} while (i < 5);

❌ 错误 2：死循环​

int i = 1;

do
{
    Console.WriteLine(i);
} while (i < 5);   // ❌ i 没变

✔ 必须改变变量：

i++;

❌ 错误 3：不理解“至少执行一次”​

很多人误以为：

条件不成立就不会执行

❌ 错！ ✔ do while 一定执行一次

八、总结（教学版精简）

一句话总结​

👉 do while 是一种 先执行，再判断的循环结构

三个核心记忆点​

1️⃣ 至少执行一次 2️⃣ 条件在后面 3️⃣ 必须有分号

使用口诀（很实用）​

👉 “先干一遍，再看条件”

九、对比建议（实战经验）

什么时候用哪个？

场景推荐
不确定是否执行while
必须执行一次do while
菜单/输入验证do while

十、给你一个经典练习题（推荐做🔥）

👉 题目： 不断输入数字，直到输入 0 结束，统计总和

int num;
int sum = 0;

do
{
    Console.Write("请输入数字（0结束）：");
    num = int.Parse(Console.ReadLine());

    sum += num;

} while (num != 0);

Console.WriteLine("总和：" + sum);
