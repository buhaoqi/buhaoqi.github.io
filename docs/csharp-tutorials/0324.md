# Random 核心知识总结（给学生记忆）

```csharp
Random r = new Random();
```

### 生成随机数

```
r.Next(10)     → 0~9
r.Next(1,11)   → 1~10
```

口诀：

```
Next(最小,最大)
左闭右开
```

例如：

```
Next(1,11) = 1~10
```
