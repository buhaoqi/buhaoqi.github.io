C# 新手最容易犯错的基础规则

规则 1：严格区分大小写

```csharp
Console.WriteLine("Hello, World!");
console.WriteLine("Hello, World!");
```
C#的代码是严格区分大小写的。

这里的Console类的首字母C,WriteLine 方法的首字母 W，如果写成小写，编译器就无法识别。

规则 2：点号表示归属
```csharp
Console.WriteLine("Hello, World!");
```
这里的点号表示WriteLine()方法属于 Console 类，可以翻译为"的“，

也就是执行Console 类的 WriteLine 方法。

规则3：小括号 () 表示执行一个动作


规则 4：数据是有类型的
字符串：
"村长"
字符：
'A'
数字：
100
写错类型。
程序就容易报错。

规则 5：大部分语句后面要加分号
;
少一个。
满屏飘红。
尽量一行一条语句
别把代码写成一坨。
未来你自己都会看不懂。

规则 6：注释不会执行
注释是写给人看的。
不是写给电脑看的。
