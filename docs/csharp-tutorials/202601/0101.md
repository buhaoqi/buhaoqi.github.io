## **开场**  
```
大家好，我是村长。

学习C#，第一件要做的事就是安装Visual Studio。

Visual Studio是一款IDE，IDE翻译成中文就是"集成开发环境"，意思是它不仅仅是代码编辑器，还集成了编译环境。

从写代码、到编译、调试和运行，全都在这一个工具里搞定。

这期视频，我手把手带你从下载到安装，一步不落，轻松上手。

```
## 第一步：下载安装器
```
首先，打开豆包，在对话框中输入：“Visual Studio社区版下载地址”，向豆包提问。

在豆包的回复中，找到下载页面并打开

可以看到，官方提供了三个版本的Visual Studio。分别是社区版、专业版和企业版。

社区版是适合个人学习使用的免费版，功能也非常完整。

点击“免费下载“ 按钮，等待 1 秒钟后开始下载，如果这时弹出了“Cookie“对话框，请点击“接受”。

我这边用的是 Chrome浏览器，右上角点下载图标，可以看到安装器文件，名字叫：VisualStudioSetup。

体积很小，只有4兆。
```

## 第二步：安装Visual Studio主程序
```
下载完安装器之后，双击打开它。

点击“运行” → “继续”。

接下来，它会自动帮我们下载并安装Visual Studio主程序，

这个过程大概会占用 1GB 左右的硬盘空间。

等你看到“工作负荷”这个界面

说明主程序已经安装完成。
```


## 第三步：安装开发工具包
```
接下来是最关键的一步：安装工作负荷

工作负荷就是工具包，这里是系统提供的工具包列表。

不同方向的开发，需要安装对应的工具包，比如：

做网站开发需要安装.NET工具工具包

做C++应用程序开发，需要安装C++的工具包等等。

在这里，我们学习 C#，就必须勾选这个：.NET 桌面开发包

安装之后，你就可以开发：

- 控制台程序
- Windows 窗体程序

右边是它包含的组件，

- 保持默认就行，不用改。

右下角能看到工具包的大小，大概 6~7GB。

左下角可以修改安装位置，
如果你 C 盘空间足够，就不用修改。


然后，点击：“安装”
安装器会边下载边安装，
一般几分钟到十几分钟，看网速。


当你看到“启动”这个按钮

说明工具包已经装好了。
```
## 第四步：个性化设置
```
点击“启动”，进入设置界面。

这里有两个简单选项：
- 开发设置：保持默认
- 颜色主题：可以选择你喜欢的深色或浅色主题，后面还可以改。

虽然没有确凿的证据证明：“暗色模式更护眼”，但在这里，我还是推荐暗色模式。

点击：“启动 Visual Studio”

到这里，我们的VS 就安装完成了。
```
