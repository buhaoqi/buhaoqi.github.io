## **开场**  
大家好，我是村长，今天不废话，直接带你安装 Visual Studio。

## 第一步：下载安装器

> 首先，打开豆包，在对话框中输入：“Visual Studio社区版下载地址”，向豆包提问。
> 
> 在豆包的回复中，找到Visual Studio 2026 官方下载页地址

> 打开下载页，可以看到，官方提供了三个版本的Visual Studio。分别是社区版、专业版和企业版。我们选择第一个"Community"，也就是社区版。社区版是适合个人学习使用的免费版，功能也非常完整。点击“免费下载“ 按钮，等待 1 秒钟后开始下载，如果这时弹出了拦截对话框，请点击“允许“。
> 
> 现在下载的这个文件就是Visual Studio的安装器，名字叫 `VisualStudioSetup.exe`，只有 4.2 兆。
> 
> 下载完成，双击打开安装器，开始下载 Visual Studio的主程序。主程序大概占用不超过 2G的硬盘空间。

接下来是安装工作负荷，工作负荷就是开发工具包，这些都是 visual studio 提供的开发工具包，


## 安装.NET 桌面开发工具包

在这里，我们只需勾选.NET 桌面开发工具包即可，.NET 桌面开发工具包支持我们创建 C# 控制台程序、Windows窗体应用程序，如果做网站开发，还需要勾选ASP.NET 和 web 开发。

> 在右侧你能看到工具包的具体组成，这里保持默认就好。
>
> 通过界面左下角的位置，可以修改安装路径，在右下角可以看到，工具包的总大小是6.99G，如果你的C盘空间足够大，直接安装到C盘即可，不需更改安装路径。
> 
> 接下来，我直接点击 “**安装**” 按钮，开始下载并安装，整个过程需要大概 5 分钟左右，取决于你的网速和电脑配置，耐心等待安装完成。
> 
>
> 当你看到这个启动按钮，就说明我们已经安装完成。

点击启动按钮

---

## 启动 Visual Studio

> 安装完成后，Visual Studio 会自动打开。
> 
> 它可能会提示你注册或登录已有账号，这一步 **可以跳过或稍后再说**。
> 
> 接下来配置你想要的 **开发环境主题**，比如“深色”、“浅色”等。选你喜欢的风格即可，后面也可以更改。
> 
> 最后点击“开始 Visual Studio”按钮。
> 
> 安装完成！

## 认识Visual Studio

Visual Studio 是微软公司在1997 年发布的一款号称宇宙最强大的 IDE。

IDE就是集成开发环境，意思是 VS 不仅是一款代码编辑器，更是集成了编译环境的代码编辑器。

学习C#，第一件要做的事就是安装Visual Studio。
