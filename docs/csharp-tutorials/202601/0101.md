## **开场**  
大家好，我是村长。

学习C#，第一件要做的事就是安装Visual Studio。

VS 不仅是一款代码编辑器，它还集成了编译环境。我们称这种集成了编译环境的编辑器叫IDE。

这一期，我手把手带你，把 Visual Studio 从下载到安装，一步一步搞定。

就算你是零基础，也能跟着装好。

## 第一步：下载安装器

首先，打开豆包，在对话框中输入：“Visual Studio社区版下载地址”，向豆包提问。

在豆包的回复中，找到下载页面并打开

可以看到，官方提供了三个版本的Visual Studio。分别是社区版、专业版和企业版。社区版是适合个人学习使用的免费版，功能也非常完整。点击“免费下载“ 按钮，等待 1 秒钟后开始下载，如果这时弹出了拦截对话框，请点击“允许”。

我这边用的是 Chrome浏览器，右上角点下载图标，可以看到这个文件：VisualStudioSetup。注意，这个只是安装器，体积很小，只有几兆。


## 第二步：安装Visual Studio主程序

下载完安装器之后，双击打开它。
点击“运行” → “继续”。
接下来，它会自动帮我们下载并安装主程序，
这个过程大概会占用 1GB 左右的硬盘空间。
等你看到这个界面👇
👉 “工作负荷”
说明主程序已经安装完成了。

下载完成后，双击打开安装器，点击“运行“，点击"继续“，开始下载并安装 Visual Studio的主程序。主程序将会占用1个G左右的硬盘空间。



## 第三步：安装开发工具包

接下来是最关键的一步。
什么是“工作负荷”？
你可以理解为：
👉 不同开发方向的“工具包”
比如你想写 C#，就必须装对应的工具。
这里我们只需要勾选一个：
✅ .NET 桌面开发
勾选之后，你就可以开发：
👉 控制台程序
👉 Windows 窗体程序
右边是它包含的组件，
👉 保持默认就行，不用改。
左下角可以修改安装路径，
如果你 C 盘空间够，就不用动。
右下角能看到大小，大概 6~7GB。
然后，点击：
👉 “安装”
接下来就等它下载 + 安装，
一般几分钟到十几分钟，看网速。
等你看到这个按钮👇
👉 “启动”
说明工具包已经装好了。

## 第四步：个性化设置

点击“启动”，进入设置界面。
这里有两个简单选项：
👉 开发设置：保持默认
👉 颜色主题：随便选（推荐深色，护眼）
后面都可以改，不用纠结。
点击：
👉 “启动 Visual Studio”

到这里，我们的VS 就安装完成了。
