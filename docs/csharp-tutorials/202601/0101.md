## **开场**  
```
大家好，我是村长。

本期视频，我们一起来学习安装 Visual Studio。

 。

其实安装软件本身并不难，

真正让初学者困惑的，是为什么一定要装 Visual Studio。

今天，我的目标就是帮助你搞懂这两件事：1.怎么安装，2.以及为什么要安装。

```
## 第一步：下载安装器
```
首先，打开豆包，在对话框中输入：“Visual Studio社区版下载”，向豆包提问。

在豆包的回复中，找到"官方下载页"并打开链接

官方下载页提供了Visual Studio的下载，还提供了Visual Studio Code的下载

在这里，我们下载的是第一个：Visual Studio

Visual Studio分为Community社区版、Professional专业版和Enterprise企业版。

其中，社区版是免费的，适合个人学习使用，功能也非常完整。

点击“免费下载“ 按钮，等待 1 秒钟后开始下载，如果这时弹出了“Cookie“对话框，请点击“接受”。

我这边用的是 Chrome浏览器，右上角点下载图标，可以看到安装器文件，名字叫：VisualStudioSetup。

它的体积很小，只有4.2兆。
```

## 第二步：安装Visual Studio主程序
```
下载完安装器之后，双击打开它。

点击“继续”。

接下来，它会自动帮我们下载并安装主程序，

这个过程大概会占用 1GB 左右的硬盘空间。

等你看到“工作负荷”这个界面

说明主程序已经安装完成。
```


## 第三步：安装开发工具包
```
接下来是最关键的一步：安装工作负荷

记住：工作负荷就是微软提供的开发工具包，

不同方向的开发，需要安装不同的工具包

这是工具包列表。

比如：

做C#网站开发需要安装ASP.NET工具包

做Python开发，需要安装Python工具包

做Node开发，需要安装Node.JS等等。

在这里，我们学习 C#，就必须勾选：.NET 桌面开发包

安装之后，你就可以做控制台程序和Windows窗体程序的开发了。

右边是它包含的组件，保持默认就行，不用改。

右下角能看到工具包的大小，6.93GB。

左下角可以修改安装位置，
如果你 C 盘剩余空间少于20G，建议改到其他盘。
我在这里就不做修改了。

然后，点击：“安装”
安装器会边下载边安装，
一般几分钟到十几分钟，取决于你的网速。


当你看到“启动”这个按钮

说明工具包已经装好了。
```
## 第四步：个性化设置
```
点击“启动”，进入设置界面。

这里有两个简单选项：
- 开发设置：保持默认
- 颜色主题：选择你喜欢的深色或浅色主题，后面随时可以修改

尽管没有确凿证据证明深色模式护眼，这里还是推荐深色主题，看起来更舒服。

点击：“启动 Visual Studio”

到这里，我们的VS 就安装完成了。
```

## 为什么要安装Visual Studio

```
程序员写程序，本质上就是写指令，让计算机按照我们的想法去工作。

但我们编写C#语句使用的是英语单词，计算机是看不懂的，它只认识 0 和 1。

这时，就需要一个 “翻译官”，把我们用英语编写的代码翻译成机器能理解的0和1代码，这个工具就叫编译器。

而安装 Visual Studio，不仅仅安装了代码编辑器，同时还安装了完善的编译器。

像这种把代码编辑器和编译器等工具集成在一起的开发软件，我们称之为 IDE（也就是集成开发环境）。所以：

- Visual Studio 是标准的 IDE
- VS Code 只是代码编辑器，本身不属于 IDE。
```
