大家好，我是村长。零基础学C#，看懂这一期，你就彻底搞懂变量和数据类型！

很多新手写代码一脸懵，不知道什么时候用int、什么时候用string、小数和布尔值又该怎么写。其实特别简单，记住一句话：变量就是装数据的盒子，数据类型就是规定这个盒子只能装什么东西。
C#是强类型语言，必须先规定类型，再存数据。今天我们一次性吃透开发中最常用的五种核心数据类型：int、string、char、decimal、bool。
首先第一个，int 整型，最常用的数字类型。
int专门用来存没有小数点的整数，比如年龄、人数、序号、分数、数量，全部都用int。
给大家看标准写法：int 变量名 = 数值；
举个例子，int age = 20；代表我定义了一个叫age的变量，里面存的就是整数20。
大家记住关键点：int只能存整数，不能存小数、不能存文字。如果你写int a = 3.14；代码直接报错，这是新手最常踩的坑，一定要避开。
第二个，string 字符串类型，也是开发使用率最高的类型。
所有的文字、句子、中文、英文、符号，全部用string。比如姓名、手机号、地址、账号密码、提示文本，通通都是string。
它的核心特征超级好记：必须包裹双引号。
举例：string name = “小明”；string msg = “欢迎学习C#编程”；
这里重点提醒大家：双引号是string的标志，没有双引号、用单引号，都会直接报错。哪怕你存的是数字，只要加了双引号，它就是文本，不能用来数学计算，这一点一定要分清。
第三个，char 字符类型，很多新手容易和string搞混。
大家记住铁律：char只存单个字符，必须用单引号包裹。
什么是单个字符？一个汉字、一个字母、一个数字、一个符号，都可以。
比如：char sex = ‘男’；char num = ‘5’；char letter = ‘A’；
它和string最大的区别：第一，char是单引号，string是双引号；第二，char只能有一个内容，多一个字、多一个符号全部报错。想存多个文字，老老实实使用string。
第四个，decimal 高精度小数类型，重点用于金额、价格。
很多新手会用double存钱，这是绝对错误的！因为double会有精度丢失，计算金额会出错。
记住固定规则：只要是金钱、价格、账单、汇率计算，一律用decimal。
它的写法有个专属小细节，数值后面必须加小写m。
举例：decimal price = 99.9m；decimal money = 199.99m；
这是C#的语法规定，不加m编译器会报错，大家写代码的时候千万别漏掉。日常普通小数计算可以用其他类型，但涉及钱，只认decimal，安全精准不翻车。
最后一个，bool 布尔类型，逻辑判断的核心。
bool是最简单的类型，只有两个值：true 真、false 假，没有第三种情况。
所有的判断逻辑都靠它实现：比如账号是否登录、是否成年、按钮是否点击、数据是否为空，全部用bool。
举例：bool isLogin = true；bool isAdult = false；
大家注意，true和false是C#关键字，全小写、不加引号，直接写就行，这也是新手常写错的地方。
最后我给大家做一个一秒速记总结，大家直接截图保存：
存整数、数量用int；
存文字、句子用string（双引号）；
存单个字符用char（单引号）；
存金额价格用decimal（后缀加m）；
存对错、状态判断用bool（true/false）。
以上这五种类型，覆盖了我们C#入门90%的开发场景。只要你把这几个类型用熟练，后续的变量运算、条件判断、循环代码，你都能轻松看懂。

## 一、数据是什么

### 数据是信息

在生活中，数据，就是信息。我们身边到处都是数据。

（画面1：奇奇在超市）

超市里：

- 商品名称是数据。它告诉你：你买的是什么。
- 商品价格和数量也是数据。它们可以计算：一共要花多少钱。

超市靠这些数据收钱。我们靠这些数据决定：买，还是不买。

（画面2：奇奇在体检）

体检报告上的：姓名、年龄、身高、体重。这些也都是数据。

它们记录了：小朋友的成长情况。

（画面3：奇奇在教室）

成绩单上的分数也是数据。

根据每门课的分数，可以计算总分、排名。

这些数字，记录了小朋友努力的结果。

所以你会发现：

无论是价格、年龄、身高、体重，还是考试成绩——这些符号化的数据，

都有一个共同的作用：我们依靠数据传递信息，并根据数据做出选择。


### 程序中的数据是什么

在程序世界里，数据有一个更专业的定义：

数据是程序运行过程中操作的对象。

程序的运行，离不开数据。

处理数据贯穿于程序的整个生命周期。

比如：

- 计算器程序。需要你输入数字，才能计算结果。

- 游戏程序。需要知道：玩家的血量、分数、金币，才能决定输赢。

没有数据。程序就像：没有食材的厨房。根本没法工作。

不过这里有个非常关键的问题。

我们看到的数据，在计算机眼里，其实全都是：0 和 1

比如：

我们看到的文本“Hello“，在计算机内部，实际可能长这样：

```text
01001000 01100101 01101100 01101100 01101111
```

存储在你硬盘里的影片，本质上也是：一大串 0 和 1。

计算机根本不懂：什么叫文本。什么叫整数和小数。

它只认识：0 和 1。

那么问题来了：

同样是一串 0 和 1。计算机怎么知道：

它到底表示的是：文本、还是整数或小数

答案就是：数据类型

## 二、数据类型（Data Type）是什么


变量就是装数据的盒子，数据类型就是规定这个盒子只能装什么东西

C#是强类型语言，必须先规定类型，再存数据。今天我们一次性吃透开发中最常用的五种核心数据类型：int、string、char、decimal、bool。

数据类型就是数据的：“说明书”。

数据类型，其实就是一套识别数据，为数据分类规则。

本质上就是：告诉计算机：“这份数据，应该按什么方式处理。”

在 C# 中,最常见的数据类型有五种：

|数据类型| 数据	|说明|
|---|---|---|
|int | 18|整型|
|double|9.9|浮点型|
|string|"Hello"|字符串型|
|char|'男'|字符型|
|bool|true|布尔型|

不同的数据，特点不同，处理方式也不同。

比如：

- 数值，可以进行加减乘除的数学运算
- 文本，可以拼接组合、传递信息。

同样一个：100

如果把它当成 int 类型，也就是数值，它就是数值100。

但如果：把它当成 char 类型。它就表示小写字母 d，因为小写字母 d 对应的字符编码是 100。

所以：内容没变。变化的是：“解释方式”。


## 变量是什么

好。现在我们终于来到：

今天最核心的概念：变量是什么？

变量有很多种定义，比如：

- 变量就是变化的量(量就是容器的意思)。
- 所以，变量就是一个存储数据的容器。
- 变量也可以理解为数据的别名。你可以理解为给数据起外号
- 变量是引用数据的一种方式。

但是，从本质上说，变量是一个有名字的内存空间。

你可以把内存想象成一个宾馆，内存存储的是数据，宾馆存储的是人。

内存被划分为一排存储数据小格子，就像宾馆里的房间。
不同类型的数据，占用的内存大小也不同，就像不同大小的房间，有小房间、中房间、大房间。
客人，就像数据。
变量，就是宾馆里有房间号的房间。

不同变量占用不同大小的内存格子




不同类型的变量，就像不同大小的房间。
数据，就像住进房间里的客人。

## 声明变量与初始化变量

很多同学容易把：

“声明变量”

和

“初始化变量”

搞混。

这里一定要分清。

---

# 1. 声明变量

声明。

意思是：

# 只创建变量，不放数据。

例如：

```csharp
int age;
```

这句话只是告诉计算机：

> “我要创建一个叫 age 的整数变量。”

但里面还没有正式数据。

---

# 2. 初始化变量

初始化。

意思是：

# 第一次给变量赋值。

例如：

```csharp
age = 18;
```

这才是真正：

第一次把数据放进去。

---

当然。

实际开发中。

我们经常会：

声明和初始化一起写。

例如：

```csharp
int age = 18;
```

这句话同时完成了：

* 创建变量
* 第一次赋值

---

整句话翻译成人话就是：

> “创建一个叫 age 的整数变量，
> 并把 18 存进去。”

---

再比如：

```csharp
string name = "奇奇";
```

意思就是：

> “创建一个叫 name 的字符串变量，
> 里面保存文字‘奇奇’。”
