# 三、核心知识讲解（10分钟）

## 1 创建随机数对象

板书代码：

```csharp
Random r = new Random();
```

解释：

| 代码           | 含义      |
| ------------ | ------- |
| Random       | 随机数类    |
| r            | 对象名字    |
| new Random() | 创建随机数对象 |

---

## 2 生成随机数

### 写法1

```csharp
r.Next(10)
```

结果：

```text
0 ~ 9
```

---

### 写法2

```csharp
r.Next(1,11)
```

结果：

```text
1 ~ 10
```

口诀（必须强调）：

```text
Next(最小值, 最大值)

左闭右开
```

# 四、第一个程序（课堂演示）（5分钟）

## 随机点名

老师现场写代码：

```csharp
using System;

class Program
{
    static void Main()
    {
        Random r = new Random();

        int num = r.Next(1, 31);

        Console.WriteLine("今天回答问题的是：" + num + "号同学");
    }
}
```

运行几次，让学生观察：

```text
5号同学
17号同学
29号同学
```

课堂互动：

```text
老师：电脑真的随机吗？
```

学生会觉得有趣。

# 十、老师常见错误提醒

错误写法：

```csharp
Random r;
int num = r.Next(10);
```

原因：

```text
对象没有创建
```

正确：

```csharp
Random r = new Random();
```
