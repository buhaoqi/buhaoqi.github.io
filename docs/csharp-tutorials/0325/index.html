3月25日 数组

很好 👍 下面是一套 “课堂气氛版”C# 随机数练习题（递进式）。 特点：

  • 难度 逐步递增
  • 有趣、贴近生活
  • 每题都能练到一个知识点
  • 特别适合中职课堂互动

建议顺序： 1 → 5

C# 随机数课堂练习（趣味版）

练习1：今天谁值日？​

题目​

班级有 8名学生，编号为 1~8。

编写程序随机选出 今天的值日生。

运行示例​

今天的值日生是：3号同学

要求​

  • 使用 Random
  • 随机数范围：1~8

知识点​

  • Random
  • Next(min,max)

参考代码​

Random r = new Random();
int num = r.Next(1, 9);

Console.WriteLine("今天的值日生是：" + num + "号同学");

练习2：随机点奶茶

题目​

奶茶店有以下饮品：

1  珍珠奶茶
2  椰果奶茶
3  抹茶奶茶
4  草莓奶茶
5  巧克力奶茶

编写程序 随机推荐一杯奶茶。

运行示例​

今天推荐：草莓奶茶

要求​

  • 生成 1~5 的随机数
  • 使用 switch 输出奶茶名称

知识点​

  • switch
  • 随机选择

练习3：6位随机密码生成器

题目​

编写程序生成 6位随机数字密码。

示例：

生成的密码：583926

要求​

  • 使用 循环
  • 每一位随机 0~9

知识点​

  • for循环
  • 随机数
  • 字符串拼接

参考思路​

循环6次
每次生成一个数字
拼接成密码

练习4：猜数字游戏（进阶）

题目​

电脑生成 1~100 的随机数。 玩家一直猜，直到猜对。

程序提示​

请输入你猜的数字：50
猜小了

请输入你猜的数字：80
猜大了

请输入你猜的数字：73
恭喜猜对了！

要求​

  • 使用 while
  • 判断大小
  • 猜对结束程序

知识点​

  • while
  • 随机数
  • 条件判断

练习5：老虎机小游戏（课堂气氛神器）

题目​

模拟一个 简单老虎机。

随机生成 3个数字（1~3）

例如：

2 2 2

规则：

情况结果
三个相同恭喜中奖
不相同再来一次

运行示例​

老虎机结果：

3 3 3

恭喜中奖！

或

1 3 2

很遗憾，再试一次

要求​

  • 生成 3个随机数
  • 判断是否相等

知识点​

  • 多个随机数
  • 逻辑判断

课堂升级版（老师可以现场演示）

如果课堂时间允许，可以再加一个 超有趣的小练习：

练习6：今天吃什么？​

随机推荐：

1 兰州拉面
2 炒饭
3 汉堡
4 盖浇饭
5 麻辣烫
6 水饺

运行：

今天中午吃：麻辣烫

学生会很有参与感 😂

Random 最重要知识总结（给学生）

Random r = new Random();

//0~9
r.Next(10);

//1~10
r.Next(1,11);

//随机多个
int a = r.Next(1,10);
int b = r.Next(1,10);

记忆口诀：

Next(最大值)       → 从0开始
Next(最小,最大)    → 左闭右开

例如：

Next(1,11) → 1~10
