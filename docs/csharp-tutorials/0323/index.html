3月23日 方法

案例：老虎机​

using System;
using System.Threading;

class Program
{
    static void Main()
    {
        Random rand = new Random();
        int money = 100;

        Console.WriteLine("====== 老虎机小游戏 ======");

        while (money >= 10)
        {
            Console.WriteLine();
            Console.WriteLine("当前金币：" + money);
            Console.WriteLine("按 Enter 拉动老虎机 (10金币)");
            Console.WriteLine("输入 0 退出");

            string input = Console.ReadLine();

            if (input == "0")
                break;

            money -= 10;

            Console.WriteLine("🎰 老虎机转动中...");

            int a = 0, b = 0, c = 0;

            // 滚动动画
            for (int i = 0; i < 10; i++)
            {
                a = rand.Next(1, 5);
                b = rand.Next(1, 5);
                c = rand.Next(1, 5);

                Console.Write("\r");
                Console.Write(GetIcon(a) + " | " + GetIcon(b) + " | " + GetIcon(c));

                Thread.Sleep(150);
            }

            Console.WriteLine();
            Console.WriteLine();

            // 判断中奖
            if (a == b && b == c)
            {
                Console.WriteLine("🎉 三个相同！大奖 +50");
                money += 50;
            }
            else if (a == b || a == c || b == c)
            {
                Console.WriteLine("✨ 两个相同！+20");
                money += 20;
            }
            else
            {
                Console.WriteLine("很遗憾，没有中奖");
            }
        }

        Console.WriteLine();
        Console.WriteLine("游戏结束，剩余金币：" + money);
    }

    static string GetIcon(int n)
    {
        switch (n)
        {
            case 1: return "🍒";
            case 2: return "🍋";
            case 3: return "🔔";
            case 4: return "⭐";
            default: return "?";
        }
    }
}

C# Random 随机数课堂教案（中职版）

一、教学目标​

学生通过本节课能够：

  1. 理解什么是 随机数
  2. 掌握 Random 类的基本用法
  3. 会使用 Next() 生成随机数
  4. 能结合 if、switch、循环 做简单小游戏

二、课堂导入（5分钟）

老师提问：

老师：如果我想让电脑随机点一个同学回答问题，
电脑怎么做到“随机”？

学生可能回答：

  • 随便选
  • 抽签
  • 摇号

老师总结：

电脑通过“随机数”来实现随机。

引出今天的主题：

C# 如何生成随机数？

三、核心知识讲解（10分钟）

1 创建随机数对象​

板书代码：

Random r = new Random();

解释：

代码含义
Random随机数类
r对象名字
new Random()创建随机数对象

2 生成随机数​

写法1​

r.Next(10)

结果：

0 ~ 9

写法2​

r.Next(1,11)

结果：

1 ~ 10

口诀（必须强调）：

Next(最小值, 最大值)

左闭右开

四、第一个程序（课堂演示）（5分钟）

随机点名​

老师现场写代码：

using System;

class Program
{
    static void Main()
    {
        Random r = new Random();

        int num = r.Next(1, 31);

        Console.WriteLine("今天回答问题的是：" + num + "号同学");
    }
}

运行几次，让学生观察：

5号同学
17号同学
29号同学

课堂互动：

老师：电脑真的随机吗？

学生会觉得有趣。

十、老师常见错误提醒

错误写法：

原因：

对象没有创建

正确：

Random r = new Random();
