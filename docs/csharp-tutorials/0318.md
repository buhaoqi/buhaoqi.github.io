# 5道 **C# continue 关键字专项练习编程题**
（专门练习：**跳过本次循环，直接进入下一次**）
`continue` 作用：**跳过当前循环剩下的代码，立刻开始下一次循环**
适合课堂专项训练，**每道题必须用 continue 才能完成**

---

# 第 1 题：跳过数字 5（基础 continue）
**题目**：输出 1~10，但**遇到 5 时跳过不输出**。
**要求**：必须使用 `continue`。

**答案**
```csharp
using System;
class Program
{
    static void Main()
    {
        int i = 0;
        while (i < 10)
        {
            i++;
            if (i == 5)
            {
                continue; // 跳过5，不打印
            }
            Console.WriteLine(i);
        }
    }
}
```

# 第 2 题：只打印奇数，跳过偶数（continue 经典）
**题目**：输出 1~20 数字，**遇到偶数就跳过，只打印奇数**。
**要求**：必须用 `continue`。

**答案**
```csharp
using System;
class Program
{
    static void Main()
    {
        int i = 0;
        while (i < 20)
        {
            i++;
            if (i % 2 == 0)
            {
                continue; // 偶数跳过
            }
            Console.WriteLine(i);
        }
    }
}
```

# 第 3 题：跳过负数，只累加正数（continue + 计算）
**题目**：用户循环输入数字，输入 0 结束。
**规则**：
- 遇到**负数**，使用 `continue` 跳过，不累加
- 只累加**正数**
- 最后输出总和

**答案**
```csharp
using System;
class Program
{
    static void Main()
    {
        int sum = 0;
        int num = 0;

        while (true)
        {
            Console.Write("输入数字(0结束)：");
            num = int.Parse(Console.ReadLine());

            if (num == 0) break;
            if (num < 0)
            {
                Console.WriteLine("负数已跳过！");
                continue; // 跳过负数
            }
            sum += num;
        }
        Console.WriteLine("正数总和：" + sum);
    }
}
```

# 第 4 题：输入“跳过”则不打印（continue + 字符串）
**题目**：用户循环输入文字。
**规则**：
- 输入**“跳过”** → 使用 `continue` 不打印
- 输入**“结束”** → 退出程序
- 其他内容正常打印

**答案**
```csharp
using System;
class Program
{
    static void Main()
    {
        while (true)
        {
            Console.Write("输入内容：");
            string str = Console.ReadLine();

            if (str == "跳过")
            {
                continue; // 跳过本次，不输出
            }
            if (str == "结束")
            {
                break;
            }
            Console.WriteLine("你输入了：" + str);
        }
    }
}
```

# 第 5 题：1~50 跳过能被 10 整除的数字（综合练习）
**题目**：输出 1~50，**跳过 10、20、30、40、50**，其余都输出。
**要求**：必须用 `continue`。

**答案**
```csharp
using System;
class Program
{
    static void Main()
    {
        int i = 0;
        while (i < 50)
        {
            i++;
            if (i % 10 == 0)
            {
                continue; // 跳过10的倍数
            }
            Console.Write(i + " ");
        }
    }
}
```

# 核心知识点总结（课堂必讲）
- **break**：直接**退出整个循环**
- **continue**：**只跳过本次循环**，继续下一次
- 5道题覆盖：跳过数字、跳过偶数、跳过负数、跳过字符串、跳过倍数
