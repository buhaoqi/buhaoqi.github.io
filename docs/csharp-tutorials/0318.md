## 案例1：密码验证



```csharp
string password;

do
{
    Console.Write("请输入密码：");
    password = Console.ReadLine();

} while (password != "123456");

Console.WriteLine("登录成功！");
```

✔ 至少要输入一次

```csharp
string username;
string password;
int count = 0;

do
{
    Console.Write("用户名：");
    username = Console.ReadLine();

    Console.Write("密码：");
    password = Console.ReadLine();

    if (username == "admin" && password == "123")
    {
        Console.WriteLine("登录成功！");
        break;
    }

    count++;
    Console.WriteLine("登录失败");

} while (count < 3);

if (count == 3)
{
    Console.WriteLine("账号已锁定");
}
```

## 一、do while语句是什么

`do while` 是一种**执行一次再判断的循环语句**。

👉 核心特点：

do while 循环特别适合：**当条件一开始就不成立**


## 二、基本语法

```csharp
do
{
    // 循环体
} while (条件);
```

⚠️ 注意：

* `while (条件);` 后面必须有分号 `;`
* 条件是布尔表达式（true / false）

## 三、执行流程

执行顺序如下：

```
① 先执行一次代码块
② 再判断条件
③ 如果为 true → 继续循环
④ 如果为 false → 结束
```

## 四、与 while 的本质区别（重点）

| 对比项  | while  | do while |
| ---- | ------ | -------- |
| 判断时机 | 先判断    | 后判断      |
| 执行次数 | 可能 0 次 | 至少 1 次   |
| 使用场景 | 条件驱动   | 必须执行一次   |


## 案例 1：输出 Hello World

```csharp
int i = 10;

// while
while (i < 5)
{
    Console.WriteLine("Hello World");
}

// do while
do
{
    Console.WriteLine("Hello World");
} while (i < 5);
```

👉 输出结果：

```
Hello World
```

✔ while：一次都不执行
✔ do while：执行了一次




## 案例3：游戏菜单

```csharp
int choice;

do
{
    // 菜单一定会先显示 1 次 → 这就是 do while 的优势
    Console.WriteLine("===== 游戏菜单 =====");
    Console.WriteLine("1. 开始游戏");
    Console.WriteLine("2. 退出");
    Console.Write("请输入你的选择：");
    
    choice = int.Parse(Console.ReadLine());

    // 根据选择做出回应，让程序更完整
    if (choice == 1)
    {
        Console.WriteLine("游戏开始！祝你好运～\n");
    }
    else if (choice == 2)
    {
        Console.WriteLine("退出成功，欢迎下次再来！");
    }
    else
    {
        Console.WriteLine("输入错误，请重新选择！\n");
    }

} while (choice != 2); // 选择不是2就一直循环
```

✔ 菜单必须先显示一次

    Console.WriteLine("2.退出");

# 七、常见错误（新手必看）

## ❌ 错误 1：忘记分号

```csharp
} while (i < 5)   // ❌ 少了 ;
```

✔ 正确：

```csharp
} while (i < 5);
```

---

## ❌ 错误 2：死循环

```csharp
int i = 1;

do
{
    Console.WriteLine(i);
} while (i < 5);   // ❌ i 没变
```

✔ 必须改变变量：

```csharp
i++;
```

---

## ❌ 错误 3：不理解“至少执行一次”

很多人误以为：

> 条件不成立就不会执行

❌ 错！
✔ do while 一定执行一次
