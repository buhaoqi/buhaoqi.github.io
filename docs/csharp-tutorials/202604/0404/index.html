2026技能操作真题

2026年河北省对口升学计算机类 技能操作真题（可打印版）

满分：150分 考试时间：60分钟

一、Windows 基本操作（30分）​

在考生文件夹下完成以下操作：

  1. 新建文件夹 Examinee，并在其中新建子文件夹：winkt、wordkt、excelkt、pptkt、cskt。
  2. 为 winkt\cup.bmp 创建快捷方式，命名为“杯子”，移动到 Examinee 文件夹。
  3. 将 winkt 中所有 .txt 格式文件移动到 Examinee 文件夹。
  4. 将 winkt\cake.jpg 移动到 Examinee，并重命名为“蛋糕.jpg”。
  5. 删除 winkt 中大小为 2KB 的文件。
  6. 将 winkt\run.gif 复制到 Examinee，并重命名为“奔跑.gif”。
  7. 删除考生文件夹下的 Examdel 文件夹。

二、Word 操作（40分）​

打开 wordkt\Word1.doc，完成如下设置：

  1. 查找替换：将全文“计算机”替换为“Computer”，字体设置为：红色、加粗、四号字。
  2. 页面设置：页边距上下 2.5cm，左右 2cm；纸张 A4，方向纵向。
  3. 段落格式：全文首行缩进 2 字符，行间距 1.5 倍，段前间距 0.5 行。
  4. 表格操作：在文档末尾插入 3 行 4 列表格；合并第一行所有单元格，输入“学生成绩表”；表格边框 1.5 磅实线，底纹浅灰色。
  5. 图文混排：插入图片 winkt\cup.bmp，环绕方式为四周型，大小缩放 50%，放置在文档右侧。

三、Excel 操作（30分）​

打开 excelkt\Excel1.xlsx，完成如下操作：

  1. 在 A1:D1 依次输入：姓名、语文、数学、总分。 在 A2:A4 输入：张三、李四、王五。 成绩： 张三：语文85 数学92 李四：语文78 数学88 王五：语文90 数学85
  2. 用公式计算每位学生总分；计算语文平均分、数学最高分。
  3. A1:D1 合并居中，黑体、14号、蓝色；数据区域居中对齐并加细实线边框；总分列底纹设为黄色。
  4. 选取 A1:C4 数据，插入簇状柱形图，图表标题为“学生成绩对比”，显示数据标签，放在表格右侧。

四、PowerPoint 操作（20分）​

新建演示文稿，完成如下操作：

  1. 第1张幻灯片（标题幻灯片）： 主标题：计算机技能考试 副标题：2026年河北省对口升学 标题：黑体44号、蓝色；副标题：宋体24号、黑色。
  2. 第2张幻灯片（标题+内容）： 标题：考试模块 内容项目符号：Windows、Word、Excel、PPT、C#编程。
  3. 第3张幻灯片：插入图片 winkt\cake.jpg，应用图片样式“棱台亚光”，添加动画“飞入”（自左侧）。
  4. 所有幻灯片切换效果设为“淡出”，换片方式为单击鼠标时。
  5. 文件保存到 pptkt，命名为“技能演示.pptx”。

五、C# 程序设计（30分）​

在 cskt 文件夹下创建控制台应用程序 ScoreApp，实现如下功能：

  1. 存储3名学生姓名、语文成绩、数学成绩。
  2. 计算每个学生总分、全班平均分。
  3. 输出每名学生的姓名、各科成绩、总分。
  4. 输出全班平均分。
  5. 找出总分最高的学生并输出姓名和总分。

参考答案与操作步骤（打印版）

一、Windows 操作步骤​

  1. 在考生文件夹空白处右键→新建文件夹，命名 Examinee，进入后依次新建5个文件夹。
  2. 找到 cup.bmp→右键→创建快捷方式→重命名“杯子”→剪切到 Examinee。
  3. 打开 winkt，搜索 *.txt，全选→剪切→粘贴到 Examinee。
  4. 剪切 cake.jpg 到 Examinee，按 F2 重命名为“蛋糕.jpg”。
  5. 切换到“详细信息”视图，按大小排序，删除大小为2KB的文件。
  6. 复制 run.gif 到 Examinee，重命名为“奔跑.gif”。
  7. 选中 Examdel 文件夹，按 Delete 删除。

二、Word 操作步骤​

  1. 开始→替换→查找：计算机，替换为：Computer→更多→格式→字体：红、加粗、四号→全部替换。
  2. 布局→页边距→自定义边距：上下2.5，左右2；纸张A4。
  3. 全选文字→段落→首行缩进2字符，行距1.5倍，段前0.5行。
  4. 插入3行4列表格→合并第一行→输入标题→表格工具设置边框1.5磅、底纹浅灰。
  5. 插入图片→环绕：四周型→大小缩放50%→拖到右侧。

三、Excel 操作步骤​

  1. 按题目要求在对应单元格录入表头、姓名、成绩。
  2. D2输入 =B2+C2 下拉填充； B5输入 =AVERAGE(B2:B4)； C5输入 =MAX(C2:C4)。
  3. 合并居中A1:D1，设置字体颜色；数据居中加边框；总分列底纹黄色。
  4. 选中A1:C4→插入簇状柱形图→添加标题与数据标签→移动到右侧。

四、PPT 操作步骤​

  1. 新建PPT，第1张版式标题幻灯片，输入文字并设置字体。
  2. 新建幻灯片，输入标题与项目符号列表。
  3. 插入图片→应用样式“棱台亚光”→动画→飞入，自左侧。
  4. 切换→淡出→单击鼠标换片→全部应用。
  5. 另存到 pptkt，命名“技能演示.pptx”。

五、C# 参考代码​

using System;

class ScoreApp
{
    static void Main()
    {
        string[] names = { "张三", "李四", "王五" };
        int[] chinese = { 85, 78, 90 };
        int[] math = { 92, 88, 85 };
        int[] total = new int[3];
        int maxTotal = 0;
        string maxName = "";

        for (int i = 0; i < 3; i++)
        {
            total[i] = chinese[i] + math[i];
            if (total[i] > maxTotal)
            {
                maxTotal = total[i];
                maxName = names[i];
            }
        }

        double avg = (total[0] + total[1] + total[2]) / 3.0;

        Console.WriteLine("学生成绩表：");
        for (int i = 0; i < 3; i++)
        {
            Console.WriteLine($"{names[i]} 语文:{chinese[i]} 数学:{math[i]} 总分:{total[i]}");
        }

        Console.WriteLine($"全班平均分：{avg:F2}");
        Console.WriteLine($"最高分：{maxName} {maxTotal}分");
    }
}
