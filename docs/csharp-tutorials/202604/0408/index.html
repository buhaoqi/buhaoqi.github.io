对口升学计算机类考试真题

2026年河北省对口升学计算机类考试真题（可打印·题目+答案分离版） 说明：2026年河北省对口升学计算机类考试（3月14-15日）官方真题暂未完全公开，本试卷为全网精选高频真题，覆盖理论、技能全部考点，适配备考刷题、模拟测试。 考试分值：理论部分100分 + 技能部分150分 适用考生：河北省对口升学计算机类考生

第一部分：专业理论真题（纯题目卷） 一、单项选择题（每题2分，共12分）

  1. 在 Word 中，拼写检查在哪个选项卡？（） A. 文件 B. 开始 C. 审阅 D. 视图
  2. 以下属于面向对象语言的是？（） A. C B. Python C. 汇编 D. HTML
  3. Excel 中求 A1:A10 平均值的函数是？（） A. =SUM (A1:A10) B. =AVERAGE(A1:A10) C. =MAX (A1:A10) D. =MIN (A1:A10)
  4. TCP/IP 中，80 端口通常用于？（） A. FTP B. SMTP C. HTTP D. DNS
  5. HTML5 中定义文档标题的标签是？（） A. head B. title C. header D. meta
  6. 以下哪个不是操作系统功能？（） A. 管理硬件 B. 提供用户界面 C. 编译程序 D. 控制程序运行 二、多项选择题（每题3分，共6分）
  7. 计算机网络的主要功能包括？（） A. 资源共享 B. 数据通信 C. 分布式处理 D. 提高可靠性
  8. 关系型数据库有？（） A. MySQL B. Oracle C. MongoDB D. Redis 三、判断题（每题1分，共2分）
  9. 磁盘既是输入设备也是输出设备。（）
  10. IP 地址和域名都是唯一的。（） 四、简答题（每题10分，共20分）
  11. 简述计算机网络的定义与分类。
  12. 简述 HTML 基本结构与常用标签。

第二部分：专业技能实操真题（纯题目卷） 一、Windows 基本操作（30分）

  1. 在考生文件夹下创建test文件夹。
  2. 复制winkt\cup.bmp到test文件夹，并为该文件创建快捷方式，命名为 “杯子”。
  3. 将考生文件夹内所有.txt格式文件移动到test文件夹中。
  4. 删除文件夹中大小为 2KB 的文件。 二、Word 操作（30分）
  5. 查找替换：将文档中所有“计算机”替换为“Computer”，字体格式设置为红色、加粗。
  6. 文档排版：设置页面页边距上下2.5cm、左右2cm；全文设置首行缩进2字符。
  7. 表格制作：在文档指定位置插入 3 行 4 列表格，合并第一行所有单元格，将表格边框设置为1.5磅实线。 三、Excel 操作（20分）
  8. 录入基础数据，对数据区域设置统一字体、居中对齐、添加表格边框。
  9. 使用SUM求和函数、AVERAGE平均值函数、MAX最大值函数完成数据计算。
  10. 根据数据表制作柱状图，为图表添加标题和数据标签。 四、网络/编程基础操作（20分）
  11. 完成电脑本地IP地址、子网掩码、网关的配置操作。
  12. 编写Python基础代码，实现变量定义、输入输出、条件判断、循环基础功能。
  13. 编写HTML+CSS代码，搭建简单网页页面并完成基础样式设置。

第三部分：参考答案及详细解析（单独答案卷） 一、专业理论答案+解析 （一）单项选择题答案（每题2分）

  1. 答案：C 解析：Word中审阅选项卡包含拼写和语法检查、批注、修订等功能。
  2. 答案：B 解析：Python是典型的面向对象编程语言；C是面向过程语言，汇编是低级语言，HTML是标记语言，不属于编程语言。
  3. 答案：B 解析：AVERAGE函数为平均值函数；SUM是求和、MAX是最大值、MIN是最小值。
  4. 答案：C 解析：80端口是HTTP超文本传输协议默认端口；FTP默认21端口、SMTP默认25端口、DNS默认53端口。
  5. 答案：B 解析：<title>标签用于定义网页文档标题；<head>是头部容器，<header>是页面头部区块，<meta>用于定义元数据。
  6. 答案：C 解析：操作系统核心功能为硬件管理、用户交互、程序调度；编译程序是编译器的功能，不属于操作系统功能。 （二）多项选择题答案（每题3分）
  7. 答案：ABCD 解析：计算机网络四大核心功能：资源共享、数据通信、分布式处理、提高系统可靠性与可用性。
  8. 答案：AB 解析：MySQL、Oracle为经典关系型数据库；MongoDB、Redis属于非关系型数据库（NoSQL）。 （三）判断题答案（每题1分）
  9. 答案：对 解析：磁盘既可写入数据（输出），也可读取数据（输入），属于双向设备。
  10. 答案：对 解析：公网IP地址、域名具有全球唯一性，可精准定位网络设备与网站。 （四）简答题参考答案（每题10分）
  11. 计算机网络的定义与分类 答：计算机网络是将多台地理位置独立的计算机及终端设备，通过通信线路、网络设备和通信协议相互连接，实现数据通信、资源共享、分布式处理的计算机系统。 按覆盖范围可分为三类： ① 局域网（LAN）：覆盖小范围区域，如校园网、办公室网络； ② 城域网（MAN）：覆盖城市范围的网络； ③ 广域网（WAN）：覆盖远距离、跨区域的大型网络，核心代表为互联网。
  12. HTML 基本结构与常用标签 答：HTML标准基本结构：
1. <!DOCTYPE html>：文档声明，定义网页文档类型；
2. <html>：网页根标签，包裹所有网页内容；
3. <head>：网页头部，存放网页配置信息；
4. <body>：网页主体，存放所有可见页面内容。



常用核心标签：
- <title>：定义网页标题；
- <p>：定义段落文本；
- <a>：定义超链接；
- <img>：插入图片；
-<h1>~<h6>：六级标题标签，用于页面标题排版。

二、专业技能实操参考答案（操作步骤） （一）Windows 操作操作步骤

  1. 打开考生文件夹，空白处右键【新建-文件夹】，命名为test；
  2. 进入winkt文件夹，右键cup.bmp文件，选择【创建快捷方式】，将快捷方式重命名为“杯子”，复制粘贴到test文件夹；
  3. 在考生文件夹搜索框输入*.txt，全选所有文本文件，剪切后粘贴至test文件夹；
  4. 文件夹切换为【详细信息】视图，按文件大小排序，选中2KB大小的文件，按Delete键删除。 （二）Word 操作操作步骤
  5. 替换操作：点击【开始-替换】，查找内容输入“计算机”，替换为输入“Computer”，点击【更多-格式-字体】，设置红色、加粗，全部替换；
  6. 页面设置：点击【布局-页边距】，自定义边距，设置上下2.5cm、左右2cm；全选文本，段落设置首行缩进2字符；
  7. 表格操作：【插入-表格】，选择3行4列；选中第一行所有单元格，【表格工具-布局-合并单元格】；选中整个表格，设置边框为1.5磅实线。 （三）Excel 操作操作步骤
  8. 按要求录入表格数据，全选数据区域，设置统一字体、水平居中对齐，添加全部边框；
  9. 对应单元格输入公式：求和=SUM()、平均值=AVERAGE()、最大值=MAX()，回车后下拉填充公式；
  10. 选中有效数据区域，【插入-簇状柱状图】，点击图表添加标题，勾选数据标签选项。 （四）网络/编程基础操作要点
  11. 网络配置：打开电脑网络适配器属性，手动设置IPv4地址、子网掩码、默认网关，保存配置并验证连通性；
  12. Python基础：定义变量存储数据，使用print()输出、input()输入，结合if条件判断、for/while循环完成基础逻辑；
  13. HTML+CSS：搭建标准HTML基础结构，通过CSS样式设置字体、颜色、边距、布局等基础页面样式。

第四部分：备考建议 1.紧扣考试大纲：核心考点包含计算机基础、Windows系统操作、Office办公软件（Word/Excel/PPT）、计算机网络、编程（Python/C#）、HTML/CSS/前端基础、数据库基础； 2. 真题模拟结合：反复刷高频真题，熟悉题型、考点和出题规律，针对性弥补薄弱模块； 3. 侧重实操训练：技能考试占分比重极高，重点熟练系统操作、Office全套实操、简单代码编写和网络基础配置，保证操作零失误。
