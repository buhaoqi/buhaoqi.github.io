## 题目1：判断字符串中是否包含指定单词

编写一个C#程序，从控制台输入一句话，再输入一个单词，判断该句子中是否包含这个单词（使用 `Contains()` 方法）。

**示例输入**：

```
请输入一句话：I love programming
请输入一个单词：love
```

**示例输出**：

```
包含该单词
```

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入一句话：");
        string sentence = Console.ReadLine();

        Console.Write("请输入一个单词：");
        string word = Console.ReadLine();

        if (sentence.Contains(word))
        {
            Console.WriteLine("包含该单词");
        }
        else
        {
            Console.WriteLine("不包含该单词");
        }
    }
}
```

</details>

## 题目2：查找字符第一次出现的位置

编写一个C#程序，从控制台输入一个字符串，再输入一个字符，使用 `IndexOf()` 查找该字符第一次出现的位置。

**示例输入**：

```
请输入字符串：banana
请输入字符：a
```

**示例输出**：

```
第一次出现的位置是：1
```

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入字符串：");
        string input = Console.ReadLine();

        Console.Write("请输入字符：");
        char ch = Convert.ToChar(Console.ReadLine());

        int index = input.IndexOf(ch);

        if (index != -1)
        {
            Console.WriteLine("第一次出现的位置是：" + index);
        }
        else
        {
            Console.WriteLine("未找到该字符");
        }
    }
}
```

</details>

## 题目4：统计某个字符出现的次数

编写一个C#程序，从控制台输入一个字符串和一个字符，使用 `IndexOf()` 循环查找该字符出现的次数。

**示例输入**：

```
请输入字符串：banana
请输入字符：a
```

**示例输出**：

```
字符出现的次数是：3
```

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入字符串：");
        string input = Console.ReadLine();

        Console.Write("请输入字符：");
        char ch = Convert.ToChar(Console.ReadLine());

        int count = 0;
        int index = input.IndexOf(ch);

        while (index != -1)
        {
            count++;
            index = input.IndexOf(ch, index + 1);
        }

        Console.WriteLine("字符出现的次数是：" + count);
    }
}
```

</details>

## 题目5：过滤指定前缀的单词

编写一个C#程序，从控制台输入一句话（多个单词用空格分隔），输出所有以指定前缀开头的单词（使用 `StartsWith()`）。

**示例输入**：

```
请输入一句话：apple banana apricot orange
请输入前缀：ap
```

**示例输出**：

```
符合条件的单词：
apple
apricot
```

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入一句话：");
        string sentence = Console.ReadLine();

        Console.Write("请输入前缀：");
        string prefix = Console.ReadLine();

        string[] words = sentence.Split(' ');

        Console.WriteLine("符合条件的单词：");

        foreach (string word in words)
        {
            if (word.StartsWith(prefix))
            {
                Console.WriteLine(word);
            }
        }
    }
}
```

</details>

### 题目1：查找字符的所有位置
编写一个C#程序，从控制台输入一个字符串和一个字符，使用 `IndexOf()` 方法找到该字符在字符串中出现的所有位置（索引），并逐行输出这些索引。如果没有找到，输出“未找到”。

**示例输入**：  
`abracadabra`  
`a`

**示例输出**：  
0  
3  
5  
7  
10

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入字符串：");
        string input = Console.ReadLine();
        Console.Write("请输入要查找的字符：");
        char ch = Console.ReadKey().KeyChar;
        Console.WriteLine();

        int index = -1;
        bool found = false;
        while ((index = input.IndexOf(ch, index + 1)) != -1)
        {
            Console.WriteLine(index);
            found = true;
        }

        if (!found)
            Console.WriteLine("未找到");
    }
}
```
</details>

### 题目2：单词首字母大写的秘密
编写一个C#程序，从控制台输入一个句子，使用 `IndexOf()` 找到每个单词的起始位置，并将每个单词的首字母大写（其他字母保持不变），输出修改后的句子。假设单词之间以空格分隔。

**示例输入**：  
`hello world from csharp`

**示例输出**：  
`Hello World From Csharp`

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入句子：");
        string sentence = Console.ReadLine();
        char[] result = sentence.ToCharArray();

        // 首字母大写
        if (result.Length > 0 && char.IsLetter(result[0]))
            result[0] = char.ToUpper(result[0]);

        // 查找空格后的字母
        int index = -1;
        while ((index = sentence.IndexOf(' ', index + 1)) != -1)
        {
            if (index + 1 < result.Length && char.IsLetter(result[index + 1]))
                result[index + 1] = char.ToUpper(result[index + 1]);
        }

        Console.WriteLine(new string(result));
    }
}
```
</details>

### 题目3：敏感词屏蔽器
编写一个C#程序，从控制台输入一段文本和一个敏感词列表（敏感词之间用逗号分隔），使用 `IndexOf()` 查找文本中是否包含敏感词。如果包含，将所有敏感词替换为 `***`，输出替换后的文本。注意：敏感词不区分大小写。

**示例输入**：  
`I love C# but hate VB.NET and hate Java`  
`hate, VB.NET`

**示例输出**：  
`I love C# but *** *** and *** Java`

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入文本：");
        string text = Console.ReadLine();
        Console.Write("请输入敏感词（用逗号分隔）：");
        string[] sensitiveWords = Console.ReadLine().Split(',');

        foreach (string word in sensitiveWords)
        {
            string trimmedWord = word.Trim();
            int index = -1;
            while ((index = text.IndexOf(trimmedWord, index + 1, StringComparison.OrdinalIgnoreCase)) != -1)
            {
                text = text.Remove(index, trimmedWord.Length).Insert(index, "***");
            }
        }

        Console.WriteLine("屏蔽后：" + text);
    }
}
```
</details>

### 题目4：最长连续相同字符段
编写一个C#程序，从控制台输入一个字符串，使用 `IndexOf()` 找到字符串中最长连续相同字符的子串，并输出该字符及其重复次数。如果有多个相同长度的，输出第一个。

**示例输入**：  
`aaabbccccdd`

**示例输出**：  
`c 4`

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入字符串：");
        string str = Console.ReadLine();

        int maxLength = 0;
        char maxChar = ' ';
        int i = 0;

        while (i < str.Length)
        {
            char current = str[i];
            // 找到下一个不同字符的位置
            int nextDiff = str.IndexOf(current, i + 1);
            if (nextDiff == -1)
                nextDiff = str.Length;

            int length = nextDiff - i;
            if (length > maxLength)
            {
                maxLength = length;
                maxChar = current;
            }

            i = nextDiff;
        }

        Console.WriteLine($"{maxChar} {maxLength}");
    }
}
```
</details>

### 题目5：字符串片段查找器（无内置函数）
编写一个C#程序，模拟 `IndexOf()` 方法的功能，但不能使用 `string.IndexOf()` 或 `string.Contains()` 等现成查找方法。从控制台输入一个主字符串和一个子字符串，输出子字符串在主字符串中第一次出现的位置索引（从0开始），如果未找到输出 -1。

**示例输入**：  
`abcdefghijklmn`  
`defg`

**示例输出**：  
3

<details>
<summary>参考代码</summary>

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.Write("请输入主字符串：");
        string mainStr = Console.ReadLine();
        Console.Write("请输入子字符串：");
        string subStr = Console.ReadLine();

        int result = MyIndexOf(mainStr, subStr);
        Console.WriteLine(result);
    }

    static int MyIndexOf(string source, string target)
    {
        if (target.Length == 0) return 0;
        if (target.Length > source.Length) return -1;

        for (int i = 0; i <= source.Length - target.Length; i++)
        {
            bool found = true;
            for (int j = 0; j < target.Length; j++)
            {
                if (source[i + j] != target[j])
                {
                    found = false;
                    break;
                }
            }
            if (found) return i;
        }
        return -1;
    }
}
```
</details>
