解析：Python是典型的面向对象编程语言；C是面向过程语言，汇编是低级语言，HTML是标记语言，不属于编程语言。
3. 答案：B
解析：AVERAGE函数为平均值函数；SUM是求和、MAX是最大值、MIN是最小值。

解析：80端口是HTTP超文本传输协议默认端口；FTP默认21端口、SMTP默认25端口、DNS默认53端口。
5. 答案：B
解析：`<title>`标签用于定义网页文档标题；`<head>`是头部容器，`<header>`是页面头部区块，`<meta>`用于定义元数据。

解析：操作系统核心功能为硬件管理、用户交互、程序调度；编译程序是编译器的功能，不属于操作系统功能。

（四）简答题参考答案（每题10分）
1. 计算机网络的定义与分类
答：计算机网络是将多台地理位置独立的计算机及终端设备，通过通信线路、网络设备和通信协议相互连接，实现数据通信、资源共享、分布式处理的计算机系统。
按覆盖范围可分为三类：
① 局域网（LAN）：覆盖小范围区域，如校园网、办公室网络；
② 城域网（MAN）：覆盖城市范围的网络；
③ 广域网（WAN）：覆盖远距离、跨区域的大型网络，核心代表为互联网。
2. HTML 基本结构与常用标签
答：HTML标准基本结构：

```html
1. <!DOCTYPE html>：文档声明，定义网页文档类型；
2. <html>：网页根标签，包裹所有网页内容；
3. <head>：网页头部，存放网页配置信息；
4. <body>：网页主体，存放所有可见页面内容。



常用核心标签：
- <title>：定义网页标题；
- <p>：定义段落文本；
- <a>：定义超链接；
- <img>：插入图片；
-<h1>~<h6>：六级标题标签，用于页面标题排版。
```

二、专业技能实操参考答案（操作步骤）
（一）Windows 操作操作步骤
1. 打开考生文件夹，空白处右键【新建-文件夹】，命名为test；
2. 进入winkt文件夹，右键cup.bmp文件，选择【创建快捷方式】，将快捷方式重命名为“杯子”，复制粘贴到test文件夹；
3. 在考生文件夹搜索框输入*.txt，全选所有文本文件，剪切后粘贴至test文件夹；
4. 文件夹切换为【详细信息】视图，按文件大小排序，选中2KB大小的文件，按Delete键删除。
（二）Word 操作操作步骤
1. 替换操作：点击【开始-替换】，查找内容输入“计算机”，替换为输入“Computer”，点击【更多-格式-字体】，设置红色、加粗，全部替换；
2. 页面设置：点击【布局-页边距】，自定义边距，设置上下2.5cm、左右2cm；全选文本，段落设置首行缩进2字符；
3. 表格操作：【插入-表格】，选择3行4列；选中第一行所有单元格，【表格工具-布局-合并单元格】；选中整个表格，设置边框为1.5磅实线。
（三）Excel 操作操作步骤
1. 按要求录入表格数据，全选数据区域，设置统一字体、水平居中对齐，添加全部边框；
2. 对应单元格输入公式：求和=SUM()、平均值=AVERAGE()、最大值=MAX()，回车后下拉填充公式；
3. 选中有效数据区域，【插入-簇状柱状图】，点击图表添加标题，勾选数据标签选项。
（四）网络/编程基础操作要点
1. 网络配置：打开电脑网络适配器属性，手动设置IPv4地址、子网掩码、默认网关，保存配置并验证连通性；
2. Python基础：定义变量存储数据，使用print()输出、input()输入，结合if条件判断、for/while循环完成基础逻辑；
3. HTML+CSS：搭建标准HTML基础结构，通过CSS样式设置字体、颜色、边距、布局等基础页面样式。
