for 循环的**三大核心细节**：

- 「循环起点」（初始化表达式）
- 「循环终点」（条件表达式）
- 「循环步长」（增量/减量表达式）

```csharp
     //①       ②⑤       ④
for (int i = 1; i <= 3; i = i + 1) 
{
    // ③
    Console.WriteLine(i * 3);
}
```

- for 循环：就是重复做
- ①变量 i = 0;设置循环的起点
- ② i < 10; 设置循环的终点
