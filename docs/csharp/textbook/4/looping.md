> 一个 `for` 循环一定要想清楚三件事：

```csharp
for (初始化; 条件; 变化)
{
    // 循环体
}
```

* **初始化**：循环从哪开始？
* **条件**：什么时候停止？
* **变化**：每次怎么走？
