**常量**就是在程序运行过程中，其值不会改变的量。常量可以分为**直接常量**和**符号常量**两种形式。  

### （一）直接常量  
#### （1）整型常量  
对于一个整数值，默认的类型就是能保存它的最小整数类型，其类型分为 `int`、`uint`、`long` 或 `ulong`。如果默认类型不是需要的类型，可以在常量后面加上后缀（`U` 或 `L`）来明确指定其类型。

- 在常量后面加 `L` 或 `l`（不区分大小写）表示长整型。  
  例如：  
  `16`      // 这是一个 `int` 类型  
  `16L`     // 这是一个 `long` 类型  

- 在常量后面加 `U` 或 `u`（不区分大小写）表示无符号整型。  
  例如：  
  `127U`    // 这是一个 `uint` 类型  
  `127UL`   // 这是一个 `ulong` 类型  

整型常量既可以采用十进制数也可以采用十六进制数（默认为十进制数）表示，在数值前面加 `0x`（或 `0X`）则表示十六进制数，基数用 `0~9`、`A~F`（或 `a~f`），例如：  
`0x20`     // 十六进制数 20，相当于十进制数 32  
`0x1F`     // 十六进制数 1F，相当于十进制数 31  

#### （2）浮点常量  
一般带小数点的数或用科学记数法表示的数都被认为是浮点数，它的数据类型默认为 `double` 类型，但也可以加上后缀符表明三种不同的浮点格式数。

- 在数字后面加上 `F`（`f`）表示是 `float` 类型。  
  例如：`3.14F`、`2.718f`  
- 在数字后面加上 `D`（`d`）表示是 `double` 类型。  
  例如：`3.14D`、`2.718d`  
- 在数字后面加上 `M`（`m`）表示是 `decimal` 类型。  
  例如：`3.14M`、`2.718m`  

例如：  
`3.14`、`3.14e2`、`2.718E-2`    // 这些都是 `double` 类型常量，其中 `3.14e2` 相当于 \(3.14 \times 10^2\)，`2.718E-2` 相当于 \(2.718 \times 10^{-2}\)

#### （3）字符常量  
字符常量是用单引号括起来的单个字符（如 `'a'`），它占 16 位，以无符号整型数的形式存储这个字符所对应的 Unicode 代码。字符常量的表达有若干种形式：  

- 单引号括起的一个字符，如 `'a'`。  
- 十六进制的换码系列，以 `\x` 或 `\X` 开始，后面跟 4 位十六进制数，如 `\X0032`。  
- Unicode 码表示形式，以 `\U` 或 `\u` 开始，后面跟 4 位十六进制数，如 `\U0032`。  
- 显式转换整数字符代码，如 `(char)67`。  
- 转义字符系列（见下表）：

| 转义字符 | 含义         | Unicode 码 |
|----------|--------------|------------|
| `\'`     | 单引号       | `\u0027`   |
| `\"`     | 双引号       | `\u0022`   |
| `\\`     | 反斜线字符   | `\u005C`   |
| `\0`     | 空字符       | `\u0000`   |
| `\a`     | 警铃符       | `\u0007`   |
| `\b`     | 退格符       | `\u0008`   |
| `\f`     | 走纸换页符   | `\u000C`   |
| `\n`     | 换行符       | `\u000A`   |
| `\r`     | 回车符       | `\u000D`   |
| `\t`     | 水平制表符   | `\u0009`   |
| `\v`     | 垂直制表符   | `\u000B`   |

#### （4）字符串常量  
字符串常量表示若干个 Unicode 字符组成的字符序列，使用一对双引号标记，例如 `"12"`、`"abc"`、`"C#程序设计"` 都是字符串，可以包括转义字符。  
例如：  
`"Hello\n"`  
`"C:\\Program Files\\Microsoft Visual Studio"`    // 表示字符串 `C:\Program Files\Microsoft Visual Studio`

#### （5）布尔型常量  
布尔型常量只有两个：  
- `true`：表示逻辑真  
- `false`：表示逻辑假
