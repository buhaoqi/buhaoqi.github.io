任务一 C#语言简介

1.C#是什么​

编程语言

  • 面向对象的
  • 运行中.net平台上的
  • 微软开发

2.C#发展简史​

  • 1998年 开始设计
  • 2000年 实现标准化 ECMA-334
  • 2002年 发展壮大

3. C#特点​

  • 简洁
  • 面向对象：
  • 支持web：支持网站应用
  • 安全
  • 兼容
  • 错误处理机制

4..NET是什么​

是C#、C++、VB、F#等语言的开发环境。

5..NET的组成​

由两部分组成：

  • 公共语言运行时（CLR)：负责在运行时管理代码
    • JIT：.NET的编译器(编译阶段)
  • 类库

6. .NET发展史​

  • 2000 ：提出构想
  • 2002：1.0发布
  • 2005-2017 发展、成熟
  • 2019 .NET Cor 3.0发布
  • 2020：.NET 5.0发布
  • 2025年12月：.NET 10.0

6. VS是什么​

VS是一款集成开发环境。

VS是一款IDE。(IDE: Integer Develop Eviroment)

  • CLR: 代码编译器
  • 代码编辑器

7. VS如何管理项目？​

VS通过两个"容器"管理项目，分别：

  • 解决方案
    • 是一个应用程序(APP)。
    • 比喻：是一个"书包"
  • 项目
    • 是组成应用程序的模块单元。
    • 比喻：项目是一本本书

8. .NET的常见文件类型​

  • .sln：解决方案文件（打开解决方案的入口）
  • .csproj：项目配置文件
  • .cs： c#源代码文件
  • .resx
  • .suo等

9.VS的工作界面由几部分组成​

  • 标题栏
  • 菜单栏
  • 工具栏
  • 工作区
    • 左：工具箱
    • 中：代码编辑器窗口和窗体设计器窗口
    • 右：资源管理器
  • 状态栏

10.手写出C#文件的模板代码​

//命名空间：是代码单元容器，比如类
namespace LessonOne
{
  //类：属性和方法的容器
  class Program
  {
    //Main:程序的人口，任何项目都必须有一个Main
    static void Main()
    {
      System.Console.WriteLine("Hello World");
    }
  }
}
