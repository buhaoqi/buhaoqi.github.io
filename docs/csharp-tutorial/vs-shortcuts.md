## 选择类

|快捷键|说明|
||选择一行|

CTRL + K + C  注释
CTRL + K + U  取消注释

CTRL + K,CTRL + D  格式化

CTRL + ,  代码搜索
CTRL + Q  功能搜索

CTRL + G  跳转

Alt + /: 调用Copilot

CTRL  + K,S 插入代码

CTRL + R + R 改全部名

CTRL + SPACE： Suggestiong

ctour：创建构造函数

CTRL + X 删除一行

CTRL + L 删除一行

CTRL + Shift + L  删除一行（清剪切板)

ALT + up/down: 移动代码行

CTRL + Alt + B  打开断点

CTRL + F9 删除全部断点

CTRL + M,M  折叠/展开代码

CTRL + Left/Right 快速移动光标

CTRL + Alt + Up/Down 选择多行

CTRL +   Shift + Up/Down  选择下一个

Ctrl + D  复制行

Home/End

Ctrl + Home/End

Ctrl + Shift + Home/End 选择到头/尾 

Shift + 方向键，快速选择代码


Shift + Alt + ；多光标输入选择
Shift + Alt + . 多光标输入选择

Shift + Alt + Up/Down 垂直选择
