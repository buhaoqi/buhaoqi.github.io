任务三 字符编码

字符编码是什么​

字符编码，就是给每个字符（字母、数字、汉字、符号）分配一个唯一的数字编号，让计算机能识别、存储和传输字符。

通俗理解

计算机只认识 0 和 1，不认识 “A”“中”“#” 这些符号。 所以人们规定： 把 A 对应成数字 65 把 0 对应成数字 48 把 “中” 对应成某个固定数字 这个 **“字符 ↔ 数字” 的对应规则 **，就叫字符编码。

字符编码的分类：西文字符编码和汉字编码

汉字编码是什么​

汉字编码，就是专门给汉字设计的一套“数字编号规则”，让计算机能识别、存储、处理汉字。

简单说：计算机只懂 0 和 1，不认汉字，所以要给每个汉字编一个唯一的数字，这就是汉字编码。

一句话总结： 汉字编码 = 给汉字编的数字代号，方便计算机处理汉字。

汉字编码的四个阶段​

汉字在计算机里输入、存储、处理、显示、打印各阶段用的编码不一样，

汉字编码主要分 4 种，对应不同阶段：

  1. 输入码（外码） 用来把汉字输进电脑：拼音、五笔、区位码

  2. 汉字交换码 不同电脑之间传汉字用：国标 GB2312

  3. 机内码（内码） 汉字在电脑内部存储、处理时用

  4. 字形码（字模码） 汉字显示、打印时用：点阵字库、矢量字库

  5. 输入阶段 → 输入码（外码） 用来把汉字输进电脑的编码，比如：

    • 拼音码
    • 五笔字型
    • 区位码

  6. 存储 & 处理阶段 → 机内码（内码） 汉字在计算机内存、硬盘里保存和运算时用的编码。 常见：

    • GB2312
    • GBK
    • Unicode（UTF-8、UTF-16）

  7. 显示 & 打印阶段 → 字形码（字模码） 用来在屏幕显示、打印机打印汉字的点阵或矢量数据。 比如：16×16点阵、24×24点阵、TrueType 字库。

  8. 交换数据阶段 → 交换码 不同系统之间传输汉字用的标准编码。 我国标准：GB2312-80（汉字交换码）

一句话速记

  • 输入汉字用外码，
  • 存储汉字用内码，
  • 传输汉字用交换码，
  • 显示打印用字形码。

任务描述​

在计算机中，所有的数据在存储和运算时都要使用二进制数表示。电子计算机诞生于美国，美国的通用语言为英语，这些英文字符在计算机中二进制表示方法是ASCII码。我国的通用语言是汉字，我们的汉字也要在计算机中进行二进制编码，才能被计算机处理。这就是计算机的字符编码知识。

预备知识​

一、西文字符编码​

计算机中常用的西文字符编码是美国信息交换标准代码：ASCII码。它是最通用的信息交换标准，并等同于国际标准ISO/IEC 646。分为7位码和8位码两个版本。

  1. 基本ASCII码 7位码是国际通用码，一个字符占用一个字节的二进制位，是半角字符，有效数字7位，左侧最高位置0，共8位二进制位，也叫基本ASCII码。编码范围从00000000B ~ 01111111B（00H ~ 7FH）；可以表示128个不同的字符。其中通用控制字符33个，空格1个，阿拉伯数字10个，大、小写英文字母52个，各种标点符号和运算符号32个。

0 ~ 31及127（共33个）是控制字符或通信专用字符（其余为可显示字符），如控制符：LF（换行）、CR（回车）、FF（换页）、DEL（删除）、BS（退格）、BEL（响铃）等；通信专用字符：SOH（文头）、EOT（文尾）、ACK（确认）等；ASCII值为8、9、10和13分别转换为退格、制表、换行和回车字符。它们并没有特定的图形显示，但会依不同的应用程序，而对文本显示有不同的影响。

32 ~ 126（共95个）是可显示字符：其中32是空格；48 ~ 57为0到9十个阿拉伯数字；65 ~ 90为26个大写英文字母；97 ~ 122号为26个小写英文字母；其余为一些标点符号、运算符号等。表3-3-1：基本ASCII码。

二、汉字代码​

概念：计算机处理汉字信息的前提条件是对每个汉字进行二进制编码，这些编码统称为汉字代码。在汉字信息处理系统中，对于不同部位，存在着多种不同的编码方式。例如，从键盘输入汉字使用的汉字代码（外码）就与计算机内部对汉字信息进行存储、传送、加工所使用的代码（内码）不同，但它们都是为系统各相关部分标识汉字使用的。计算机系统工作时，汉字信息在系统的各部分之间传送，它到达某个部分就要用该部分所规定的汉字代码表示汉字。因此，汉字信息在系统内传送的过程就是汉字代码转换的过程。这些代码构成该系统的代码体系。

根据应用目的的不同，汉字编码主要有输入码、交换码、机内码、字形码四种。

1.汉字输入码（外码）​

是为用户由计算机外部输入汉字而编制的汉字编码，又称为汉字外部码，简称外码。汉字输入码位于人机界面上，面向用户，所以它的编码原则就是简单易记、操作方便、有利于提高输入速度。主要有以下四类： (1)顺序码：将汉字按一定顺序排好，然后逐个赋予一个号码作为该汉字的编码。例区位码、电报码等。最典型的就是汉字区位码。 区位码把全部汉字和图形符号排列在一个94行*94列的矩阵中，构成一个二维表格。 “区”：二维表格中的每一行，用区号表示，范围是0194； “位”：二维表格中的每一列，用位号表示，范围是0194； 区位码：每个汉字的区号和位号的组合而成的四位十进制数，（高两位是区号，低两位是位号） 比如：汉字“中”在区位码二维表格中的第54行48列，那么它的区号为54，位号为48，完整的“中”字区位码为5448。 汉字“国”的区位码为2590，表示它的区号是25，位号是90。 区位码最大优点是一字一码，即无重码，最大缺点是难以记忆。 (2)音码：根据汉字的读音进行编码。优点是只要具有汉语拼音的基础，就可掌握。最大弱点是对于那些不知道读音的字无法输入，另外重码率非常高。例如全拼输入法、智能ABC输入法、搜狗拼音输入法、QQ拼音输入法等各种拼音输入法。 (3)形码：根据汉字的字型进行编码。如五笔字型、郑码输入法、表形码输入法、大众码等。较常用的是五笔字型输入法。五笔相对于拼音输入法具有重码率低的特点，熟练后可快速输入汉字。但随着智能拼音流行，使用五笔的人数急剧下降。 (4)音形码：根据汉字的读音和字型进行编码。如双拼码、五十字元等。

2.汉字交换码——国标码​

(1)国标码的含义： 汉字信息处理系统之间或通信系统之间传输信息时，对每一个汉字所规定的统一编码。我国已指定了国家标准信息交换汉字编码《信息交换用汉字编码字符集——基本集》即GB 2312-80，又称国标码。 国标码收录的就是区位码中的所有字符，可以说区位码是国标码的另一种表现形式，共7 445个字符。其中有6 763个汉字和682个非汉字图形符号。汉字中有一级常用汉字3 755个，按拼音排序在区位码的16到55区；二级汉字3 008个，按部首排序在区位码的56到87区。

因为汉字符号非常多，相对于英文只需要8位二进制数就可以，汉字需要用16位二进制数来表示一个汉字符号，即使用两个字节存储一个国标码，每一个汉字符号都是全角字符。国家标准规定国标码的编码范围是2121H—7E7EH（两个字节的最高位都是0）。如表3-3-2为汉字“国”的二进制国标码。

表3-3-2 汉字“国”的二进制国标码

0011100101111010
高位字节39H低位字节7AH

“国”的国标码为397AH

(2)区位码和国标码的关系： 国标码中的汉字就是区位码中的汉字，只是它们的编码方式不同，区位码作为一种汉字输入码，是面向用户的，用十进制表示，例“东”的区位码是2565，是25区65位。而国标码是计算机中使用的，用16位二进制表示，通常简化写成十六进制编码，例“东”的国标码是3961H，其中高位字节39H对应区位码中的区号25，低位字节61H对应区位码的位号65。

区位码的区号0194，对应国标码的高位字节：21H7EH 区位码的位号0194，对应国标码的低位字节：21H7EH 国标码与区位码是一一对应的，它们之间的转换关系为：区位码的区号和位号分别转换为十六进制，再分别加20H就得到该字的国标码中的高位字节和低位字节，合起来就是它们的国标码，即十六进制的区位码+2020H=国标码。

例如： “中”字区位码为5448，区号54转换为十六进制为36H，加20H后为56H； 位号48转换为十六进制为30H，加20H后为50H； “中”的国标码为5650H。

例如： “国”字区位码为2590，转换为十六进制为195AH，195AH+2020H=397AH，“国”字的国标码为397AH。

3. 汉字机内码（内码）​

西方字符的标准交换码ASCII码是用一个字节来表示的，编码范围是00H ~ 7FH；汉字国标码是用两个字节来表示，编码范围是2121H ~ 7E7EH，很明显它们相关字节的最高位都是0，有相同编码部分。为了在计算机中把汉字和西文字符区别开并相互兼容，特规定：把国标码每个字节的最高位由0置为1，作为汉字内码（即内码的两个字节的最高位都是1，编码范围是A1A1H ~ FEFEH），在计算机内部使用。如表3-3-3为汉字“国”的机内码。这样计算机就可以用字节的最高位来区分西文和汉字，字节最高位为0时表示ASCII码，为1时表示汉字。所以机内码就是汉字信息处理系统内部存储、处理汉字而使用的编码，简称内码。

表3-3-3 汉字“国”的机内码

高位字节B9H低位字节FAH

国标码和机内码的转换规则为：国标码的两个字节分别加80H，得到汉字的内码，即机内码=国标码+8080H。

例如： 汉字“中”的国标码为5650H，5650H+8080H=D6D0H，“中”字的机内码为D6D0H。

汉字“国”的国标码为397AH，397AH+8080H=B9FAH，“国”的机内码为B9FAH。

4. 汉字字形码（汉字字库、汉字字模、汉字输出码、点阵码）​

存储在计算机内的汉字需要在屏幕上显示或在打印机上输出时，需要知道汉字的字形信息。汉字内码并不能直接反映汉字的字形，而要采用专门的字形码。

目前的汉字处理系统中，字形信息的表示大体上有两类形式：一类是用活字或文字版的母体字形形式；另一类是点阵表示法、矢量表示法等形式，其中最基本的，也是大多数字形库采用的是以点阵的形式存储汉字字形编码的方法。

点阵字形是将字符的字形分解成若干“点”组成的点阵，将此点阵置于网格上，每一小方格是点阵中的一个“点”，点阵中的每一个点可以有黑白两种颜色，有字形笔画的点用黑色；反之用白色，这样就能描写出汉字字形了。

图3-3-1是汉字“学”的点阵，如果用十进制的“1”表示黑色点，用“0”表示没有笔画的白色点，每一行16个点用两字节表示，则需32字节描述一个汉字的字形，即一个字形码占32字节。根据汉字输出的要求不同，点阵的多少也不同。简易型汉字为1616点阵，提高型汉字为2424

点阵、3232点阵，甚至更高。点阵越大，描述的字形越细致美观，质量越高，所占存储空间也越大。汉字点阵的信息量是很大的，以1616点阵为例，每个汉字要占用32个字节，国标码两级6763个汉字要占用约212KB存储空间。

一个计算机汉字处理系统常配有宋体、仿宋、黑体、楷体等多种字体。同一个汉字不同字体的字形编码是不相同的。

通常，计算机中所有汉字的字形码集合起来组成汉字库（或称为字模库）存放在计算机里，当汉字输出时由专门的字形检索程序根据这个汉字的内码从汉字库里检索出对应的字形码，由字形码再控制输出设备输出汉字。

声音、图像、视频的编码​

  1. 声音编码：计算机要处理自然界的声音，需要将波形数据转换成0、1数码组成的数字数据，即音频数字化。波形声音转换成数字化音频需要经过抽样、量化、编码三个步骤。目前，计算机音频的编码格式有多种标准，常见音频文件格式有MIDI格式、WAV格式、MP3格式和RealAudio格式。

  2. MIDI（Musical Instrument Digital Interface，乐器数字接口）是数字音乐/电子合成乐器国际标准。MIDI文件有几个变通的格式：CMF文件、RIFF文件等。

  3. WAV为微软公司开发的一种声音文件格式。

  4. WMA由微软开发，WMA针对的不是单机市场，是网络支持流技术，即一边读一边播放，因此WMA可以很轻松地实现在线广播。

  5. MP3是一种音频压缩技术，也称为动态影像专家压缩标准3。其重放的音质与最初的不压缩音频相比没有明显的下降。

  6. RealAudio（即时播音系统）是一种新型流式音频文件格式。主要适用于网络上的在线播放。主要有RA、RM、RMX等三种。

  7. AIFF是音频交换文件格式，是Apple苹果电脑上面的标准音频格式。

  8. 图像编码：图像编码与声音编码类似，也需要对模拟图形进行数字化转换，即模拟图像信号转换成数字化信号。图像编码技术也称为图像压缩技术。常见的图像文件格式有BMP格式、GIF格式和JPEG格式等。

  9. BMP是Windows操作系统中的标准图像文件格式。由于压缩比很小，图像质量比较高，但占用较大空间，常用于存储在单机中使用，少见于频繁的网络传输。

  10. GIF是“图像互换格式”，一种无损压缩格式，其压缩率一般在50%左右。目前几乎所有相关软件都支持GIF图像文件。

  11. JPEG是常见的一种静态图像有损压缩格式。文件后缀名为.jpg或.jpeg。

  12. TIFF标签图像文件格式，是一种灵活的位图格式，与JPEG和PNG一起成为流行的高位彩色图像格式。TIFF文件以.tif为扩展名。

  13. PNG，便携式网络图形，是一种采用无损压缩算法的位图格式。PNG文件的扩展名为.png。

  14. 视频编码 视频编码是指通过压缩技术，将原始视频格式的文件转换成另一种视频格式文件的方式。常见视频文件格式有AVI格式、MPEG格式、RealVideo格式、MOV格式、3GP格式和MKV格式等。

  15. AVI即音频视频交错格式，是将音频和视频同步组合在一起可以同步播放的文件格式。它对视频文件采用了一种有损压缩方式，可跨多平台使用。

  16. MPEG是国际通用的动态图像压缩算法标准。MPEG、MPG、DAT等格式同属MPEG。MP4，是一种使用MPEG -4的多媒体文档格式，文件扩展名为.mp4，以储存数码音讯及数码视讯为主，是目前运用非常广泛的一种视频格式。

  17. RealVideo格式文件包括扩展名为 .ra、.rm、.ram、.rmvb 的4种视频格式，是一种高压缩比的有损视频格式，是视频流技术的始创者。

  18. MOV即QuickTime影片格式，Apple公司开发流式视频封装标准。

  19. 3GP是一种3G流媒体的视频编码格式，多用于移动通信领域。3GP是MP4格式的一种简化版本，减少了储存空间和较低的频宽需求，让手机的有限储存空间可以使用。

  20. MKV不是一种压缩格式，而是Matroska的一种媒体文件。MKV最大的

知识点答案​

一、基础概念类​

  1. 为什么计算机中的数据都要用二进制表示？

答：计算机由逻辑电路组成，只能识别高、低电平两种状态，因此所有数据在存储和运算时都必须使用二进制数表示。

  1. 什么是字符编码？计算机中为什么要对字符进行编码？

答：字符编码是为字符分配的二进制表示规则。计算机只能处理二进制数据，字符必须转换为二进制编码才能被存储、传输和处理。

  1. 计算机中西文字符和汉字分别主要采用什么编码体系？

答：西文字符主要采用 ASCII 码；汉字主要采用 GB 2312 国标码、机内码、输入码、字形码等体系。

二、ASCII 编码相关​

  1. ASCII 码的全称是什么？对应的国际标准是什么？ 答：美国信息交换标准代码；等同于国际标准 ISO/IEC 646。

  2. ASCII 码分为哪两个版本？分别是几位码？ 答：基本 ASCII 码（7 位码）和扩展 ASCII 码（8 位码）。

  3. 基本 ASCII 码占用几个字节？最高位固定为多少？ 答：占用 1 个字节；最高位固定为 0。

  4. 基本 ASCII 码的二进制、十六进制范围分别是什么？ 答：二进制：00000000B～01111111B；十六进制：00H～7FH。

  5. 基本 ASCII 码一共可以表示多少个字符？ 答：128 个。

  6. 基本 ASCII 码中控制字符有多少个？范围是多少？ 答：33 个；范围是十进制 0～31 及 127。

  7. 基本 ASCII 码中可显示字符有多少个？范围是多少？ 答：95 个；范围是十进制 32～126。

  8. 空格字符的十进制、十六进制 ASCII 码分别是多少？ 答：十进制 32，十六进制 20H。

  9. 数字 0～9 的十进制、十六进制 ASCII 码范围分别是多少？ 答：十进制 48～57；十六进制 30H～39H。

  10. 大写字母 A～Z 的十进制、十六进制 ASCII 码范围分别是多少？ 答：十进制 65～90；十六进制 41H～5AH。

  11. 小写字母 a～z 的十进制、十六进制 ASCII 码范围分别是多少？ 答：十进制 97～122；十六进制 61H～7AH。

  12. 大写字母与小写字母的 ASCII 码差值是多少（十进制与十六进制）？ 答：十进制 32，十六进制 20H。

  13. LF、CR、FF、DEL、BS、BEL 分别代表什么控制字符？ 答：LF 换行，CR 回车，FF 换页，DEL 删除，BS 退格，BEL 响铃。

  14. 扩展 ASCII 码是几位？最高位有什么特点？ 答：8 位；最高位可以为 1。

  15. 标准 ASCII 码共有多少个字符？扩展 ASCII 码共有多少个字符？ 答：标准 128 个；扩展 256 个。

三、汉字编码总体类​

  1. 什么是汉字代码？ 答：计算机处理汉字信息时，为每个汉字分配的二进制编码统称为汉字代码。

  2. 汉字信息处理系统中为什么会存在多种编码？ 答：在输入、存储、传输、显示/打印等不同环节，对汉字的表示需求不同，因此需要不同编码。

  3. 汉字在系统中传输的本质是什么？ 答：汉字代码在不同编码形式之间转换的过程。

  4. 按应用目的划分，汉字编码主要分为哪四种？ 答：输入码（外码）、交换码（国标码）、机内码（内码）、字形码（输出码）。

  5. 完整的汉字信息处理流程是什么（从输入到输出）？ 答：输入 → 输入码 → 国标码 → 机内码 → 字形码 → 输出。

四、汉字输入码（外码）​

  1. 什么是汉字输入码？又被称为什么？ 答：为用户从外部输入汉字而编制的编码；又称外码。

  2. 汉字输入码的设计原则是什么？ 答：简单易记、操作方便、利于提高输入速度。

  3. 汉字输入码主要分为哪四类？ 答：顺序码、音码、形码、音形码。

  4. 什么是顺序码？典型代表有哪些？ 答：按顺序给汉字编号的编码；代表：区位码、电报码。

  5. 区位码的编码矩阵是几行几列？区号、位号范围分别是多少？ 答：94 行×94 列；区号 01～94，位号 01～94。

  6. 区位码由几部分组成？分别是什么？ 答：四位十进制数，高两位是区号，低两位是位号。

  7. 举例说明“中”“国”的区位码及其区、位号。 答：“中”：54 区 48 位，区位码 5448；“国”：25 区 90 位，区位码 2590。

  8. 区位码的优点和缺点分别是什么？ 答：优点：无重码；缺点：难记忆。

  9. 什么是音码？优点和缺点分别是什么？举例常见输入法。 答：按读音编码；优点：易上手；缺点：重码率高，不会读音无法输入。例：全拼、搜狗拼音、智能 ABC。

  10. 什么是形码？典型代表是什么？优缺点是什么？ 答：按字形结构编码；代表：五笔字型。优点：重码少、输入快；缺点：学习成本高。

  11. 什么是音形码？举例说明。 答：结合读音和字形编码；例：双拼码、五十字元。

五、汉字交换码（国标码）​

  1. 什么是汉字交换码？我国的标准全称是什么？标准号是多少？ 答：系统间交换汉字信息的统一编码；《信息交换用汉字编码字符集——基本集》；GB 2312-80。

  2. GB 2312-80 共收录多少个字符？其中汉字、符号各多少？ 答：共 7445 个；汉字 6763 个，非汉字图形符号 682 个。

  3. GB 2312-80 中一级汉字、二级汉字各多少个？分别按什么排序？分布在哪些区？ 答：一级 3755 个，按拼音排序，16～55 区；二级 3008 个，按部首排序，56～87 区。

  4. 一个国标码占用几个字节？属于全角还是半角字符？ 答：2 个字节；全角字符。

  5. 国标码的十六进制编码范围是什么？每个字节最高位是多少？ 答：2121H～7E7EH；每个字节最高位为 0。

  6. 区位码与国标码的对应关系是什么？ 答：一一对应；区号对应国标码高位字节，位号对应低位字节。

  7. 区位码转换为国标码的公式是什么？ 答：十六进制区位码 + 2020H = 国标码。

  8. 已知某汉字区位码，如何分步计算国标码？ 答：①区号、位号分别转十六进制；②分别加 20H；③组合为两字节国标码。

  9. 分别写出“中”“国”的区位码、国标码及计算过程。 答：

  • 中：区位码 5448 → 十六进制 3630H → +2020H → 国标码 5650H
  • 国：区位码 2590 → 十六进制 195AH → +2020H → 国标码 397AH

六、汉字机内码（内码）​

  1. 什么是汉字机内码？作用是什么？ 答：计算机内部存储、处理、传输汉字使用的编码；简称内码。

  2. 为什么要在国标码基础上设置机内码？ 答：避免国标码与 ASCII 码冲突，便于区分汉字与西文字符。

  3. 机内码如何由国标码转换而来？每个字节最高位变成多少？ 答：国标码每个字节最高位由 0 改为 1；最高位为 1。

  4. 机内码的十六进制范围是什么？ 答：A1A1H～FEFEH。

  5. 计算机如何通过最高位区分 ASCII 码和汉字？ 答：最高位为 0 是 ASCII 码；最高位为 1 是汉字内码。

  6. 国标码转机内码的计算公式是什么？ 答：国标码 + 8080H = 机内码。

  7. 分别写出“中”“国”的国标码、机内码及计算过程。 答：

  • 中：国标码 5650H + 8080H = 机内码 D6D0H
  • 国：国标码 397AH + 8080H = 机内码 B9FAH

  1. 机内码转换为国标码的公式是什么？ 答：机内码 − 8080H = 国标码。

  2. 国标码转换为区位码的公式是什么？ 答：国标码 − 2020H = 十六进制区位码。

七、汉字字形码（字模/输出码）​

  1. 什么是汉字字形码？作用是什么？ 答：描述汉字字形的编码；用于屏幕显示和打印输出。

  2. 字形信息主要有哪两类表示方式？最常用的是哪种？ 答：活字形式、点阵/矢量形式；最常用点阵表示法。

  3. 什么是点阵字形？如何用 0 和 1 表示字形？ 答：将汉字拆分为网格点阵；有笔画为 1，无笔画为 0。

  4. 常见汉字点阵有哪些规格？ 答：16×16、24×24、32×32 等。

  5. 点阵大小与字形质量、存储空间的关系是什么？ 答：点阵越大，字形越清晰美观，占用存储空间越大。

  6. 16×16 点阵的一个汉字占用多少字节？如何计算？ 答：32 字节；16×16÷8=32。

  7. 24×24 点阵的一个汉字占用多少字节？ 答：72 字节；24×24÷8=72。

  8. 什么是汉字库（字模库）？ 答：所有汉字字形码的集合。

  9. 不同字体（宋体、黑体等）的字形码是否相同？ 答：不相同。

  10. 计算机如何根据内码实现汉字显示/打印？ 答：根据内码从汉字库中查找对应字形码，再控制输出设备显示。

八、声音、图像、视频编码​

  1. 音频数字化需要经过哪三个步骤？ 答：抽样、量化、编码。

  2. 常见音频编码格式有哪些？分别简述特点。 答：

  • MIDI：电子音乐标准
  • WAV：微软原生无压缩格式
  • MP3：高压缩，音质好
  • WMA：支持流媒体
  • RealAudio：适合网络在线播放
  • AIFF：苹果标准音频格式

  1. 什么是图像编码？又被称为什么？ 答：将模拟图像转为数字信号；又称图像压缩技术。

  2. 常见图像文件格式有哪些？各自特点是什么？ 答：

  • BMP：Windows 标准，几乎无压缩
  • GIF：无损压缩，支持动图
  • JPEG：有损压缩，体积小
  • TIFF：灵活位图格式
  • PNG：无损压缩，透明背景

  1. 什么是视频编码？ 答：通过压缩将原始视频转换为其他格式的技术。

  2. 常见视频文件格式有哪些？各自特点是什么？ 答：

  • AVI：音视频交错
  • MPEG/MP4：通用压缩标准
  • RM/RMVB：高压缩流媒体
  • MOV：苹果 QuickTime 格式
  • 3GP：手机专用，体积小
  • MKV：封装多音视频字幕
  • WMV/ASF：微软流媒体格式

  1. MKV 格式与普通压缩格式最大区别是什么？ 答：MKV 是封装格式，可容纳多种编码的视频、音频、字幕。

  2. 3GP 格式与 MP4 格式的关系是什么？主要应用场景是什么？ 答：3GP 是 MP4 的简化版；主要用于移动通信设备。

补充简答

  1. 24×16 点阵汉字字模占用多少字节？ 答：48 字节。

  2. 24×24 点阵汉字字模占用多少字节？ 答：72 字节。

  3. 1KB 内存可存放多少个 ASCII 字符？多少个汉字内码？ 答：1024 个 ASCII 字符；512 个汉字内码。

  4. 已知某汉字国标码，如何计算机内码？ 答：国标码 + 8080H。

  5. 已知某汉字机内码，如何求国标码？ 答：机内码 − 8080H。

  6. 汉字内码每个字节的最高位固定为多少？ 答：1。

  7. 汉字字库中存储的是汉字的哪种编码？ 答：字形码（字模）。

  8. 显示/打印汉字时使用的是哪种编码？ 答：字形码。

  9. 一个汉字与一个英文字符在存储时所占字节数之比是多少？ 答：2:1。
