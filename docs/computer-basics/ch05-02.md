## 知识点
# 中英文录入·问答题（含答案）
1. 打字前为什么要端正坐姿？
答：坐姿不正确会影响打字速度的提高，还容易使人疲劳、出错。

2. 指法练习的核心技巧有哪些？
答：左右手指放在基本键上；击完其他键迅速返回原位；食指击键注意键位角度；小指击键力量保持均匀；数字键采用跳跃式击键；手指按分工放在正确键位；记忆键盘字符位置，养成不看键盘的输入习惯；练习时集中注意力，做到手、脑、眼协调一致，保证输入准确性。

3. 成为打字高手的公式是什么？
答：正确指法 + 键盘记忆 + 集中精力 + 准确输入 = 打字高手。

4. 英文输入的练习步骤是什么？
答：保持正确坐姿和指法，使用专门打字软件，从键盘指法练习开始，逐步过渡到单词、文章练习，实现快速准确输入英文。

5. 常用的打字软件有哪些？
答：金山打字通、运指如飞等。

6. 输入法切换的快捷键是什么？
答：`<Ctrl>+<Shift>` 键，可在已安装的输入法之间切换。

7. 打开/关闭输入法的快捷键是什么？
答：`<Ctrl>+<Space>` 键，可实现英文输入和中文输入法的切换。

8. 全角/半角切换的快捷键是什么？
答：`<Shift>+<Space>` 键，可进行全角和半角的切换。

9. 常见的中文输入法有哪两类？
答：五笔输入法和各种拼音输入法。

10. 五笔输入法的主要特点是什么？
答：重码率低，掌握字根拆分规则后打字速度快，曾在专业学生中流行。

11. 现在主流的中文输入法是什么？为什么？
答：拼音输入法是主流，因为移动终端广泛使用，且拼音输入法智能化程度高。

12. 中文输入的练习步骤是什么？
答：应用打字软件，依次进行拼音输入练习、音节练习、词汇练习和文章练习等。

## 预备知识
打字之前一定要端正坐姿。如果坐姿不正确，不但会影响打字速度的提高，而且还会很容易疲劳、出错。

掌握指法练习技巧：左右手指放在基本键上；击完它键迅速返回原位；食指击键注意键位角度；小指击键力量保持均匀；数字键采用跳跃式击键。一定把手指按照分工放在正确的键位上；有意识尽快地记忆键盘各个字符的位置，体会不同键位上的字键被敲击时手指的感觉，逐步养成不看键盘的输入习惯。进行打字练习时必须集中注意力，做到手、脑、眼协调一致，尽量避免边看原稿边看键盘，这样容易分散记忆力。初级阶段的练习即使速度慢，也一定要保证输入的准确性。

总之：正确指法+键盘记忆+集中精力+准确输入=打字高手。

### 3. 英文输入
保持正确的坐姿，使用正确的指法，再使用专门的打字软件，比如金山打字通、运指如飞等，以键盘指法练习为起点，逐步到单词、文章的练习，逐渐就可以快速准确输入英文。

### 4. 中文输入
首先要掌握汉字输入热键，从而在中英文输入法之间快速切换。
1.  输入法的切换：`<Ctrl>+<Shift>`键，通过它可在已安装的输入法之间进行切换。
2.  打开/关闭输入法：`<Ctrl>+<Space>`键，通过它可以实现英文输入和中文输入法的切换。
3.  全角/半角切换：`<Shift>+<Space>`键，通过它可以进行全角和半角的切换。

想要达到中文打字能手，需要熟练掌握至少一种中文输入法。常见的中文输入法有五笔输入法和各种拼音输入法。五笔输入法因为重码率低掌握字根拆分规则后打字速度快而曾经在专业学生中非常流行，但近几年随着移动终端的广泛使用和拼音输入法的智能化，现在拼音输入法成为主流。应用打字软件，依次进行拼音输入练习、音节练习、词汇练习和文章练习等。
