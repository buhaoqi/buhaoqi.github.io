题目1：求十进制数 -25 的8位原码、反码、补码

求解过程：

1. 先求 +25 的二进制：25 = 16+8+1 = 00011001（8位）
2. 原码：负号最高位变1 → 10011001
3. 反码：原码除符号位外取反 → 11100110
4. 补码：反码+1 → 11100111

答案：
原码：10011001
反码：11100110
补码：11100111

---

题目2：已知补码 10110100，求真值（十进制）

求解过程：

1. 符号位为1，是负数，需逆向求原码
2. 补码减1得反码：10110100 - 1 = 10110011
3. 反码取反得原码：11001100（符号位不动，其余取反）
4. 原码数值部分 1001100₂ = 64+8+4 = 76，加上负号 → -76

答案： 真值 = -76

---

题目3：求十进制数 +37 的8位原码、反码、补码

求解过程：

1. 37 = 32+4+1 = 00100101
2. 正数的原码、反码、补码相同
3. 原码 = 反码 = 补码 = 00100101

答案： 均为 00100101

题目4：用8位补码计算 -15 - 20，并判断是否溢出

求解过程：

1. 求 -15 补码：
   +15 = 00001111 → 原码10001111 → 反码11110000 → 补码11110001
2. 求 -20 补码：
   +20 = 00010100 → 原码10010100 → 反码11101011 → 补码11101100
3. 相加：
   11110001 + 11101100 = 1 11011101（进位1丢弃，8位结果为11011101）
4. 结果符号位为1，转为原码：减1得11011100 → 取反得10100011 → 数值 35 → 真值 -35
5. 检查溢出：两个负数相加，结果符号位为1（正常），未超出-128~127，无溢出

答案： 补码计算结果为 11011101，真值 -35，无溢出

---

题目5：将原码 10101101 转换为补码

求解过程：

1. 原码为 10101101，符号位1表示负数
2. 数值部分 0101101₂ = 32+8+4+1=45
3. 反码 = 符号位不变，数值位取反 → 11010010
4. 补码 = 反码+1 → 11010011

答案： 补码 = 11010011
