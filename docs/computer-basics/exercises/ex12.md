## 简答题

补充简答
1. 24×16 点阵汉字字模占用多少字节？
答：48 字节。

2. 24×24 点阵汉字字模占用多少字节？
答：72 字节。

3. 1KB 内存可存放多少个 ASCII 字符？多少个汉字内码？
答：1024 个 ASCII 字符；512 个汉字内码。

4. 已知某汉字国标码，如何计算机内码？
答：国标码 + 8080H。

5. 已知某汉字机内码，如何求国标码？
答：机内码 − 8080H。

6. 汉字内码每个字节的最高位固定为多少？
答：1。

7. 汉字字库中存储的是汉字的哪种编码？
答：字形码（字模）。

8. 显示/打印汉字时使用的是哪种编码？
答：字形码。

9. 一个汉字与一个英文字符在存储时所占字节数之比是多少？
答：2:1。
