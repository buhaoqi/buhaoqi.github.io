任务一 认识键盘

知识点​

  1. 键盘的主要作用是什么？ 答：键盘是最常用也是最主要的输入设备，可将英文字母、汉字、数字、标点符号等输入计算机，向计算机发出命令、输入数据。

  2. 按形式键盘可以分为哪几类？ 答：可分为无线键盘、USB键盘和多功能键盘。

  3. 无线键盘有什么特点？ 答：不需要电缆与计算机连接，通过无线方式操作，避免电缆长度限制，使用更方便。

  4. 常用键盘按功能分为哪几个区域？ 答：分为主键盘区、功能键区、控制键区（编辑键区）、数字键区（辅助键区）和状态指示区。

  5. 主键盘区的按键分为哪几类？ 答：分为字母键、数字（符号）键和功能键三类。

  6. 字母键共有多少个？ 答：A~Z共26个字母键。

  7. 什么是双字符键？ 答：每个键面上有上下两种符号的键，可显示符号和数字，直接按为下挡字符，配合Shift键为上挡字符。

  8. 主键盘区的功能键主要有哪些？ 答：主要有Tab、CapsLock、Shift、Ctrl、Alt、Windows键等。

  9. Tab键的作用是什么？ 答：通常用以将鼠标指针推进到下一个制表位。

  10. CapsLock键的功能是什么？ 答：用于切换英文大小写锁定状态，指示灯亮为大写，灯灭为小写。

  11. Shift键的主要作用有哪些？ 答：输入双字符键的上挡字符；切换字母的大小写。

  12. Ctrl键和Alt键有什么使用特点？ 答：不能单独使用，需与其他键组合使用，以完成特定控制或特殊功能。

  13. 功能键区主要包含哪些按键？ 答：包括Esc键、F1~F12键、PrintScreen、ScrollLock、Pause/Break键。

  14. Esc键的作用是什么？ 答：叫作取消键/退出键，用于脱离当前操作或退出当前运行的软件。

  15. F1~F12键的作用是什么？ 答：作为软件的功能热键，如F1常用于打开帮助。

  16. PrintScreen键的功能是什么？ 答：可将屏幕内容复制到剪贴板；Alt+PrintScreen可复制当前窗口内容。

  17. Pause/Break键的作用是什么？ 答：暂停程序或命令执行，按任意键继续；Ctrl+Break可中断命令或程序运行。

  18. 控制键区（编辑键区）的主要作用是什么？ 答：用于文字处理时控制光标位置及进行页面操作。

  19. PageUp和PageDown键的作用分别是什么？ 答：PageUp向上翻页，PageDown向下翻页。

  20. Home键和End键的功能是什么？ 答：Home键使光标移到本行开头，End键使光标移到本行末尾。

  21. Delete键的作用是什么？ 答：删除光标所在位置上的字符。

  22. 数字键区（小键盘区）的主要作用是什么？ 答：主要用于方便输入数字和运算符号。

  23. NumLock键的功能是什么？ 答：控制数字键区状态，指示灯亮可输入数字，灯灭则为光标控制功能。

  24. 状态指示区有哪几个指示灯？ 答：Num Lock灯、Caps灯、Scroll Lock灯。

  25. 键盘状态指示灯分别表示什么功能？ 答：Num Lock表示数字键锁定状态，Caps表示大小写锁定状态，Scroll Lock表示屏幕滚动锁定状态。

任务检测​

一、选择题​

  1. 键盘中不包含的区域是（） A.主键盘区 B.功能键区 C.编辑键区 D.运算器区

  2. 下列设备中，属于计算机输入设备的是（） A.打印机 B.键盘 C.显示器 D.音箱

  3. 下列关于CapsLock键的说法，错误的是（） A.大写字母锁定键 B.主要用于中英文切换 C.指示灯亮为大写状态 D.指示灯灭为小写状态

  4. 要输入双字符键的上挡符号，需要配合使用的键是（） A.Ctrl B.Alt C.Shift D.Enter

  5. 要使光标快速移到本行开头，应使用的键是（） A.Home B.End C.PageUp D.PageDown

  6. 删除光标所在位置字符的键是（） A.Insert B.Delete C.Tab D.Esc

  7. 关于小键盘区，说法错误的是（） A.也叫数字键区 B.NumLock灯亮可输入数字 C.NumLock灯灭可控制光标 D.只能输入数字不能运算

  8. 下列属于退出键，用于取消或退出当前操作的是（） A.F1 B.Esc C.Enter D.Pause

  9. 键盘中用于上下左右移动光标的键区是（） A.主键盘区 B.功能键区 C.控制键区 D.数字键区

  10. 关于键盘使用，说法正确的是（） A.Shift键可以单独使用 B.Ctrl键可以单独使用 C.键盘只能输入英文不能输入汉字 D.PrintScreen可以复制屏幕内容

二、填空题​

  1. 键盘按形式可分为无线键盘、USB键盘和________。
  2. 标准键盘按功能分为主键盘区、功能键区、控制键区、数字键区和________。
  3. 主键盘区的按键分为字母键、数字（符号）键和________。
  4. ________键可以切换英文大小写锁定状态。
  5. ________键用于输入双字符键的上挡字符，也可临时切换字母大小写。
  6. ________键可以将屏幕内容复制到剪贴板。
  7. PageUp是________键，PageDown是向下翻页键。
  8. Home键可以使光标快速移动到________，End键移到本行末尾。
  9. ________键控制数字键区的数字输入与光标控制切换。
  10. 键盘状态指示灯包括Num Lock灯、Caps灯和________。

答案​

1.D 2.B 3.B 4.C 5.A 6.B 7.D 8.B 9.C 10.D 1.多功能键盘 2.状态指示区 3.功能键 4.CapsLock 5.Shift 6.PrintScreen 7.向上翻页 8.本行开头 9.NumLock 10.Scroll Lock

任务描述​

键盘是操作电脑的基本设备，相信很多新手在接触到电脑时，对电脑的操作也有些陌生，学习打字先要了解键盘，把键盘的每个按键的功能都了解清楚了，后面打字才会更得心应手。标准的键盘是由一百多个按键组成，每个按键都有自己的功能，快来认识这些按键吧！

预备知识​

键盘是最常用也是最主要的输入设备，通过键盘可以将英文字母、汉字、数字、标点符号等输入到计算机中，从而向计算机发出命令、输入数据等。

一、分类​

按键盘的形式可分为无线键盘、USB键盘和多功能键盘：

  1. 无线键盘：不需要电缆与计算机相连接，而是通过无线的方式来操作计算机。它避免了由于电缆长度的限制而使操作人员感到不方便的现象。
  2. USB接口键盘：就是使用USB接口的键盘，其他同标准的键盘一样。
  3. 多功能键盘：上面的功能键要比标准键盘多得多。通过键盘上的多功能键，可以上网，听音乐，看VCD。有的还有声卡和开关功能。

二、键盘的分区​

目前常用的是101、104键键盘，按照功能的不同，我们把这100多个键划分成主键盘区、功能键区、控制键区（编辑键区）、数字键区（辅助键区）和状态指示区。

1. 主键盘区​

键盘中最常用的区域是主键盘区，主键盘区中的键又分为三大类，即字母键、数字（符号）键和功能键。

  1. 字母键：AZ共26个字母键。在字母键的键面上标有大写英文字母AZ，每个键可打大小写两种字母。
  2. 数字（符号）键：共有21个键，包括数字、运算符号、标点符号和其他符号。每个键面上都有上下两种符号，也称双字符键，可以显示符号和数字。上面的一行称为上挡符号，下面的一行称为下挡符号。
  3. 功能键：功能键共有14个，Alt、Shift、Ctrl、Windows键各有两个，对称分布在左右两边，功能完全一样，只是为了操作方便。
    • Tab（制表键）：通常用以将鼠标指针推进到下一个制表位。
    • CapsLock（大写字母锁定键，也叫大小写换挡键）：位于主键盘区最左边的第三排。每按一次大写字母锁定键，英文大小写字母的状态就改变一次。状态指示区上标有CapsLock的那盏灯亮了就是大写字母状态，否则为小写字母状态。键盘的初始状态为英文小写字母状态，按一下该键，键盘右上角的CapsLock指示灯亮，表示已转换为大写状态并锁定，此时在键盘上按任何字母键均为大写字母。如果再按一次该键，CapsLock指示灯灭，又变为小写状态。
    • Shift（上挡键，也叫换挡键）：位于主键盘区的第四排，左右各有一个，用于键入双字符键中的上挡字符。主键盘区的数字（符号）键是双字符键，直接按下显示下挡字符，按住Shift键再按则显示上挡字符。Shift键也可切换字母大小写：小写状态下按Shift+字母键输出大写字母，大写状态下则输出小写字母。
    • Ctrl（控制键）：一共有两个，位于主键盘区的左下角和右下角。该键不能单独使用，需要和其他键组合使用，能完成某些特定的控制功能。
    • Alt（转换键）：一共有两个，位于空格键的两侧。Alt键与Ctrl键一样，也不能单独使用，需要和其他的键组合使用，可以完成一些特殊功能，在不同的工作环境下，转换键转换的状态也不同。

2. 功能键区​

位于键盘的最上方，包括Esc和F1~F12键，这些按键用于完成一些特定的功能。

  1. Esc键：叫作取消键，位于键盘的左上角。Esc是英文Escaps（取消）的缩写，在许多软件中它被定义为退出键，一般用作脱离当前操作或退出当前运行的软件。
  2. F1~F12功能键：一般软件都是利用这些键来充当软件中的功能热键，例如用F1键寻求帮助。
  3. Printscreen（屏幕硬拷贝键）：在打印机已经联机的情况下，按下该键可以将计算机屏幕的显示内容通过打印机输出。还可以将当前屏幕的内容复制到剪贴板。按Alt+Printscreen键可以将当前窗口的内容复制到剪贴板。
  4. ScrollLock（屏幕滚动显示锁定键）：目前该键已很少用到。
  5. Pause或Break（暂停键）：按该键，能使得计算机正在执行的命令或应用程序暂时停止工作，直到按键盘上任意一个键则继续。另外，按Ctrl+Break键能中断命令的执行或程序的运行。

3. 控制键区（编辑键区）​

控制键区也叫编辑键区共有10个键，位于主键盘区的右侧，包括所有对光标进行操作的按键以及一些页面操作功能键，这些按键用于在进行文字处理时控制光标的位置。

  1. PageUp（向上翻页键）：按一下这个键可以使屏幕向前翻一页。
  2. PageDown（向下翻页键）：按一下这个键可以使屏幕向后翻一页。
  3. Home键：按一下这个键可以使光标快速移动到本行的开始。
  4. End键：按一下这个键可以使光标快速移动到本行的末尾。
  5. Insert键：按一下这个键可以改变插入与改写状态。
  6. Delete（删除键）：删除光标所在位置上的字符。
  7. 方向键：移动这四个键，可以使光标在屏幕内上下左右移动。

4. 数字键区​

数字键区位于键盘的右侧，又称为“小键盘区”，主要是为了输入数据方便，一共有十七个键，其中大部分是双字符键，其中包括0~9的数字键和常用的加减乘除运算符号键，这些按键主要用于输入数字和运算符号。

  1. NumLock（数字锁定键）：位于小键盘区的左上角，相当于上挡键的作用。当NumLock指示灯亮时，表示数字键区的上位字符数字输入有效，可以直接输入数字；再按一下NumLock键，指示灯灭，其下位字符编辑键有效，用于控制光标的移动。数字锁定键的信号灯是键盘右上角最左边的那盏灯，只有当它被点亮的时候，才可以输入数字。而当它熄灭时，小键盘上的这组数字键变成了光标移动键，可以控制光标的移动。
  2. 数字键：实际上这是个双字符键，上面是数字，下面几个控制键。
  3. 运算符号键：包括加（+）、减（−）、乘（*）、除（/）运算符。
  4. Enter键：叫小回车键，与主键盘上的回车键功能完全相同。

5. 状态指示区​

状态指示区位于数字键区的上方，包括3个状态指示灯Num Lock灯、Caps灯和Scroll Lock灯，用于提示键盘的工作状态。
