## 如何使用MySQL客户端登录MySQL数据库？
- 第一步：打开MySQL客户端
- 第二步：在窗口内输入密码123456即可登录

提示：当你看到
```
Welcome to the MySQL monitor
```
的时候，就表示登录成功。

## 如何退出MySQL数据库？
```bash
exit
```
或
```bash
quit
```
