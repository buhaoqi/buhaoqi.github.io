## 数据库是什么

定义：

> 数据库(Database)是按照数据结构来组织、存储和管理数据的仓库。 

理解:

- 数据库是存储数据的仓库。
- 数据库必须是具有某种**管理功能**的数据的集合。(随便收集起来的数据不能称为数据库)
- 功能是指数据库具备”增删改查“等管理功能。
- 数据库为了实现“增删改查”等功能，必须按照特定的数据结构组织起来。
- 数据库的英文是：`database`
