## 一、主键（Primary Key）
### 1. 定义
- 主键是一个字段。
- 主键也可以是字段组合。
- 主键用于标识表中的每行数据。
- 作为主键的字段不可重复，具有唯一性。
- 主键是表中**唯一标识每一行数据**的字段。
 
它的核心特性如下：

- **唯一性**：主键的值在表中必须唯一，不可重复。
- **非空性**：主键字段的值不允许为 `NULL`。
- **唯一约束**：一个表只能有一个主键。

### 2. 作用
- **唯一标识**：通过主键可以快速定位到表中的某一行。
- **数据完整性**：确保表中没有重复或无效的数据。
- **外键引用**：其他表可以通过主键建立关联（外键关系）。

### 3. 示例
```sql
-- 创建主键
CREATE TABLE students (
    student_id INT PRIMARY KEY,  -- 主键
    name VARCHAR(50),
    email VARCHAR(100) UNIQUE
);
```
- `student_id` 是主键，每个学生的 ID 唯一且非空。
