第一节 教育基础知识

一、教育的概念​

(一)广义概念​

概念：一切有目的的活动。

  • 活动：增进人的知识技能、发展人的智力体力、影响人的思想观念。
  • 包括：家庭教育、学校教育、社会教育

(二)狭义概念​

概念：指教育者对受教育者的活动。

  • 活动：根据社会要求，有目的、有机会、有组织、有系统的，促使受教育者朝着期望的方向发展的活动
  • 包括：学校教育

二、教育一词的由来​

  1. 战国《孟子·尽心上》,中国典籍中最早将 “教” 与 “育” 二字连用的记载。

得天下英才而教育之，三乐也。

解读：得到天下优秀的人才，对他们加以教育(培养)，这是第三件快乐的事。

  1. 东汉 许慎《说文解字》，最早解释“教育"的词义

教，上所施，下所效也；育，养子使作善也。

解读：教：上位者施行的言行示范，下位者去效仿遵从。育：培育后辈，引导他们行善修德。

三、教育的形态​

(一)教育规范程度​

  • 制度化教育：由专门的教育人员、机构、制度构成的教育形态。
  • 非制度化教育：没有形成相对独立的教育形态。

(二)教育场所​

  • 家庭教育：以家庭为单位进行的教育。
    • 特点：终针先生权威感染
  • 学校教育：以学校为单位进行的教育。
    • 特点：目计专集可系组效
  • 社会教育：以学校为单位进行的教育。
    • 特点：自伸自巩持灵

(三)社会形态​

  • 农业社会教育
  • 工业社会教育
  • 信息社会教育

四、教育的基本要素​

(一)教育者​

定义：指在教育活动中承担教的责任，对受教育者的身心发展产生影响的人。

教育者包括：

  1. 教育管理人员
  2. 教师：教育主题，最直接的教育者，在教育活动中起主导作用
  3. 家长

(二)受教育者​

定义：指在教育活动中承担学习责任，接受教育影响的人。

受教育者是教育的对象，是学习和发展的主体。

(三)教育影响​

定义：在教育活动中，教育者作用于受教育者的全部信息。

教育影响主要包括：

  1. 教育内容
  2. 教育手段

五、教育的属性​

(一)教育的本质属性​

教育的本质：教育是一种有目的地培养人的社会活动。

理解：

  1. 培养人：教育的对象是人，动物界不存在教育。
  2. 有目的：只有有目的的培养人的活动才是教育，无目的的影响不是教育。
  3. 社会活动：人的先天本能不属于教育。比如：孩子他娘给孩子喂奶。

(二)教育的社会属性​

  1. 永恒性：教育是成长和发展的手段，只要人类社会存在，教育就存在。
  2. 历史性：体现在时代性和阶级性
  3. 相对独立性：教育有自身发展规律，继承历史，与社会发展不完全同步。
  4. 生产型：一定的投入，能产生一定的经济效益。
  5. 长期性：教育活动活教育个体，教育周期比较长。

六、教育的功能​

(一)个体功能与社会功能​

  1. 个体功能：教育可以对个体的发展产生影响和作用。（本体功能）
    • 促进个体社会化：社会价值观、自我观念、社会技能、社会角色
    • 促进个体个性化：唤醒生命自觉、尊重个体的差异性、独特性、促进个性和创造性发展
  2. 社会功能：教育对社会稳定和发展的影响和作用（派生功能）
    • 社会变迁功能：通过教育推动社会的改革和发展。
    • 社会流动功能：通过教育在社会环境和社会阶层之间转换和调动。

(二)正向功能与负向功能​

  1. 正向功能：教育有助于社会进步和个体发展。

  2. 负向功能：教育阻碍社会进步和个体发展。

(三)显性功能与隐性功能​

  1. 显性功能：教育目的和教育结果吻合。

  2. 隐性功能：出现非预期的教育结果。

七、教育的起源​

(一)神话起源​

  • 主要观点：教育起源自神的创造
  • 代表人物：朱熹
  • 评价：教育起源的最古老的观点，缺乏科学依据。

(二)生物起源​

  • 主要观点：教育起源于动物生存的本能活动
  • 代表人物：利托尔诺(法)、沛西·能(英)
  • 评价：教育史上第一个正式提出教育起源的学说。标志从神话解释转向科学解释。错误：没把握人类教育的目的性和社会性。
  • 记忆口诀：“本能生利息“(动物的生存本能、生物起源论、利托尔诺、沛西·能)

"动物尤其是略为高等的动物，完全同人一样，生来就有一种由遗传而得到的潜在的教育，其效果见诸个体的发展过程。"

利托尔诺

“我之所以把教育称之为生物学的过程，意思就是说，教育是与种族需要、种族生活相应的、天生的，而不是获得的表现形式，教育既无须周密的考虑使它产生，也无需科学予以指导，它是扎根于本能的不可避免的行为。”

—— 沛西・能（1923 年报告）

(三)心理起源​

  • 主要观点：教育起源于儿童对成人的无意识模仿
  • 代表人物：孟禄(美)
  • 评价：正确性：把教育限定在人类的社会活动范围内。错误性：抹杀了教育的有意识性，否认教育的社会性。
  • 记忆口诀：心理仿孟禄（心理起源论、儿童对成人无意识的模仿、孟禄）

(四)劳动起源​

  • 主要观点：教育起源于劳动
  • 代表人物：米丁斯基(苏联)、凯洛夫(苏联)
  • 评价：劳动起源说是在马克思唯物主义理论指导下形成的。
  • 记忆口诀：米凯爱劳动（米丁斯基、凯洛夫、教育起源于劳动）

八、教育的历史发展​

(一)原始社会的教育特征​

  1. 没有阶级性
  2. 为生产劳动服务
  3. 在社会生产和生活中进行
  4. 教育手段原始

(二)古代社会的教育特征​

  1. 出现专门的教育机构和专职的教育人员
  2. 具有鲜明的阶级性和严格的等级性
  3. 教育内容逐渐丰富
  4. 教育与生产劳动相脱离
  5. 教学组织形式：个别施教活集体个别施教
  6. 教育适应古代思想文化的发展，教育表现出：
    • 道统性
    • 专制性
    • 刻板性
    • 象征性

(三)近代社会的教育特征​

  1. 公立教育崛起、国家重视和干预教育
  2. 初等义务教育普遍实施（德国是最早实行义务教育的国家）
  3. 教育世俗化（从宗教中分离出来）
  4. 重视教育立法、依法治教

(四)20世纪后的教育特征​

  1. 教育终身化
  2. 教育全民化
  3. 教育民主化
  4. 教育多元化
  5. 教育现代化

九、中国古代教育的发展​

(一)夏朝​

  1. 虞舜时代教育机构：庠（xiáng,本义为 "礼官养老" 的地方）
  2. 夏朝教育机构：庠、序(东序、西序)、校

(二)商朝​

  1. 教育特点：根据年龄划分教育阶段。
  2. 教育机构：大学、小学、瞽宗、庠、序
    • 瞽宗(gǔ zōng)：商朝大学,奴隶主贵族子弟学习礼乐的学校。

(三)西周​

  1. 教育特点
  2. 教育内容

(四)春秋战国​

  1. 教育特点：我国教育史上的重要里程碑。促成百家争鸣。
  2. 教育机构：稷下（齐国著名学府)

(五)汉朝​

  1. 教育机构：太学、鸿都门学、宫邸学、郡国学、书馆、经馆

(六)魏晋南北朝​

  1. 教育机构：
    • 西晋：国子学与太学并列
    • 南朝设立四馆：儒学馆、玄学馆、史学馆、文学馆
  2. 教育特点：我国最早的分专业的综合学校

(七)隋唐​

  1. 教育机构：分为六学二馆
    • 六学大学性质：国子学、太学、四门学
    • 六学专科性质：书学、律学、算学
    • 二馆：崇文馆、弘文馆
  2. 教育特点：
    • 等级性：六学二馆体现了我国古代教育的等级性
    • 科举考试：隋朝创立科举考试制度。隋炀帝建进士科，标志科举考试制度确立。

(八)宋元​

  1. 教育机构：国子学、太学、辟雍、小学
  2. 教育内容：四书五经
    • 四书：《大学》《论语》《中庸》《孟子》
    • 五经：诗、书、礼、易、春秋
  3. 教育特点：四书作为教学的教材及科举考试的依据

(九)明清​

  1. 教育内容：四书五经
  2. 教育特点：
    • 明朝：八股文作为科举考试的固定格式
    • 清朝：科举考试走线衰落，1905 年正式废除。

十、外国古代教育的发展​

(一)古印度​

  1. 教育特点：教育控制在婆罗门教和佛教手中。
  2. 教育内容：《吠陀》

(二)古埃及​

  1. 教育机构：宫廷学校、职官学校、文士学校
  2. 教育特点：以僧为师，以吏为师

(三)古雅典​

  1. 教育特点：西方最早形成体育、的语、智育、美育和谐发展
  2. 教育目的：
    • 培养政治家和商人（崇文教育）
    • 培养军人和武士（尚武教育）

(四)中世纪欧洲​

  1. 教育特点：教会教育和骑士教育
  2. 教育目的：培养封建骑士
  3. 教育内容：
    • 教会教育三科：文法、修辞、辩证法
    • 教会教育四学：算术、几何、天文、音乐
    • 骑士教育七技：骑马、击剑、打猎、投枪、游泳、下棋、吟诗
