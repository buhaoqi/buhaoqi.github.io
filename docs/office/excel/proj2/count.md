## **一、COUNT() 函数：基础数字计数**

### **1. 基本语法**
```excel
=COUNT(value1, [value2], ...)
```

### **2. 参数特性**
- **只统计数字**：忽略文本、逻辑值、空单元格
- **统计范围**：包含数字的单元格、数字本身、日期、时间
- **最多255个参数**

### **3. 使用示例**

#### **示例1：直接数值计数**
```excel
=COUNT(10, 20, 30)                // 结果：3
=COUNT(10, "文本", 30)            // 结果：2（忽略文本）
=COUNT(10, TRUE, 30)              // 结果：2（忽略TRUE）
```

#### **示例2：单元格区域计数**
```excel
A1=10, A2="文本", A3=20, A4=空, A5=TRUE
=COUNT(A1:A5)                     // 结果：2（只计算10和20）
```

#### **示例3：混合区域计数**
```excel
=COUNT(A1:A10, B1:B5, 100)       // 多个区域和单个数值
```

#### **示例4：日期时间计数**
```excel
A1=2024-01-01, A2=12:30, A3="日期", A4=100
=COUNT(A1:A4)                     // 结果：3（日期、时间、数字都统计）
```

### **4. 相关计数函数对比**
```excel
=COUNT(A1:A10)      // 统计数字个数
=COUNTA(A1:A10)     // 统计非空单元格个数（含文本、逻辑值）
=COUNTBLANK(A1:A10) // 统计空单元格个数
=COUNTIF(A1:A10,">0") // 统计大于0的数字个数
```

## **二、COUNTIF() 函数：单条件计数**

### **1. 基本语法**
```excel
=COUNTIF(range, criteria)
```

### **2. 参数详解**
| 参数 | 说明 |
|------|------|
| **range** | 要统计的单元格区域 |
| **criteria** | 统计条件（数字、文本、表达式、单元格引用） |

### **3. 条件写法大全**

#### **文本条件**
```excel
=COUNTIF(A:A,"苹果")        // 等于"苹果"
=COUNTIF(A:A,"*果")         // 以"果"结尾
=COUNTIF(A:A,"苹*")         // 以"苹"开头
=COUNTIF(A:A,"??果")        // 三个字且以"果"结尾
=COUNTIF(A:A,"<>苹果")      // 不等于"苹果"
=COUNTIF(A:A,"苹果?")       // "苹果"开头且后面还有一个字
```

#### **数字条件**
```excel
=COUNTIF(B:B,">60")         // 大于60
=COUNTIF(B:B,">=80")        // 大于等于80
=COUNTIF(B:B,"<>0")         // 不等于0
=COUNTIF(B:B,100)           // 等于100
=COUNTIF(B:B,">=60")-COUNTIF(B:B,">80") // 60-80之间
```

#### **日期条件**
```excel
=COUNTIF(C:C,">=2024-01-01")
=COUNTIF(C:C,">"&DATE(2024,6,1))
=COUNTIF(C:C,">="&TODAY())
```

#### **特殊条件**
```excel
=COUNTIF(A:A,"")            // 统计空单元格
=COUNTIF(A:A,"<>")          // 统计非空单元格
=COUNTIF(A:A,"*")           // 统计包含文本的单元格
=COUNTIF(A:A,">=0")         // 统计所有数字（包括0）
```

#### **单元格引用作为条件**
```excel
=COUNTIF(A:A,D2)            // D2的内容作为条件
=COUNTIF(B:B,">"&E2)        // 大于E2的值
```

### **4. 实际应用示例**

#### **示例1：学生成绩统计**
```excel
A列：班级  B列：成绩

1班人数：=COUNTIF(A:A,"1班")
及格人数：=COUNTIF(B:B,">=60")
优秀人数(>=90)：=COUNTIF(B:B,">=90")
缺考人数(0分)：=COUNTIF(B:B,0)
非1班人数：=COUNTIF(A:A,"<>1班")
```

#### **示例2：销售数据分析**
```excel
A列：产品  B列：销售额

手机订单数：=COUNTIF(A:A,"手机")
高销售额订单(>10000)：=COUNTIF(B:B,">10000")
非手机订单：=COUNTIF(A:A,"<>手机")
包含"Pro"的产品数：=COUNTIF(A:A,"*Pro*")
```

#### **示例3：考勤统计**
```excel
A列：日期  B列：状态

出勤天数：=COUNTIF(B:B,"出勤")
请假天数：=COUNTIF(B:B,"请假")
迟到次数：=COUNTIF(B:B,"迟到")
12月份出勤：=COUNTIFS(B:B,"出勤", A:A,">=2024-12-01", A:A,"<=2024-12-31")
```

### **5. 常见错误示例**
```excel
错误：=COUNTIF(A:A,>60)           // 条件缺少引号
正确：=COUNTIF(A:A,">60")

错误：=COUNTIF(A:A,">D2")         // D2被当作文本
正确：=COUNTIF(A:A,">"&D2)        // 使用&连接

错误：=COUNTIF(A:A,苹果)          // 文本缺少引号
正确：=COUNTIF(A:A,"苹果")        // 文本加引号
```

### **6. 高级技巧**

#### **技巧1：统计唯一值个数**
```excel
=SUMPRODUCT(1/COUNTIF(A1:A10, A1:A10))  // 统计A1:A10中唯一值个数
```

#### **技巧2：统计包含特定字符的单元格**
```excel
=COUNTIF(A:A,"*"&"特定字符"&"*")    // 包含"特定字符"
=COUNTIF(A:A,"???"&"特定字符")      // 以三个任意字符+"特定字符"结尾
```

#### **技巧3：多条件计数（简化版）**
```excel
=COUNTIF(A:A,"条件1")+COUNTIF(A:A,"条件2")  // 满足条件1或条件2
```

## **三、COUNTIFS() 函数：多条件计数**

### **1. 基本语法**
```excel
=COUNTIFS(criteria_range1, criteria1, 
          [criteria_range2, criteria2], ...)
```

### **2. 参数说明**
| 参数 | 说明 |
|------|------|
| **criteria_range1** | 第一个条件的区域 |
| **criteria1** | 第一个条件 |
| **criteria_range2, criteria2, ...** | 可选，最多127个条件对 |

### **3. 使用示例**

#### **示例1：双条件计数**
```excel
A列：部门  B列：职级

统计"技术部"且"高级"的人数：
=COUNTIFS(A:A,"技术部", B:B,"高级")
```

#### **示例2：多条件组合**
```excel
统计"销售部"在"北京"且工龄>5年的人数：
=COUNTIFS(部门列,"销售部", 地区列,"北京", 工龄列,">5")
```

#### **示例3：日期范围计数**
```excel
A列：日期  B列：产品

统计2024年1月"手机"订单数：
=COUNTIFS(B:B,"手机", A:A,">=2024-01-01", A:A,"<=2024-01-31")
```

### **4. 实际工作场景**

#### **场景1：员工统计分析**
```excel
数据表：部门 | 职级 | 入职年份 | 绩效等级 | 性别

1. 技术部P7级人数：
   =COUNTIFS(部门列,"技术部", 职级列,"P7")

2. 2019年及以后入职的女性员工数：
   =COUNTIFS(入职年份列,">=2019", 性别列,"女")

3. 绩效A且工龄>3年的人数：
   =COUNTIFS(绩效列,"A", 工龄列,">3")

4. 技术部或研发部且职级>=P6：
   =COUNTIFS(部门列,"技术部", 职级列,">=P6") + 
     COUNTIFS(部门列,"研发部", 职级列,">=P6")
```

#### **场景2：销售订单分析**
```excel
数据表：订单日期 | 客户类型 | 产品类别 | 金额 | 状态

1. Q2大客户(金额>10000)订单数：
   =COUNTIFS(订单日期列,">=2024-04-01", 订单日期列,"<=2024-06-30",
             客户类型列,"大客户", 金额列,">10000")

2. 已完成的手机订单数：
   =COUNTIFS(产品类别列,"手机", 状态列,"已完成")

3. 1月份非退货订单数：
   =COUNTIFS(订单日期列,">=2024-01-01", 订单日期列,"<=2024-01-31",
             状态列,"<>退货")

4. 老客户(购买次数>3)且金额>5000：
   =COUNTIFS(客户类型列,"老客户", 金额列,">5000")
```

#### **场景3：库存管理统计**
```excel
数据表：商品编码 | 分类 | 仓库 | 库存量 | 预警值

1. 食品类且库存低于预警值的商品数：
   =COUNTIFS(分类列,"食品", 库存量列,"<"&预警值列)

2. A仓库非零库存商品数：
   =COUNTIFS(仓库列,"A仓", 库存量列,"<>0")

3. 库存为0且30天内无进货的商品：
   =COUNTIFS(库存量列,0, 最后进货日期列,"<"&TODAY()-30)
```

### **5. 高级技巧**

#### **技巧1：动态条件范围**
```excel
=COUNTIFS(A:A,">="&开始日期, A:A,"<="&结束日期, B:B,">="&最小值)
```

#### **技巧2：使用通配符组合**
```excel
=COUNTIFS(班级列,"高?班", 科目列,"*数学*")  // 高X班且科目包含"数学"
```

#### **技巧3：统计某列非空且满足条件**
```excel
=COUNTIFS(A:A,"<>", B:B,">60")  // A列非空且B列>60
```

#### **技巧4：数组条件（Excel 365+）**
```excel
=COUNTIFS(B:B,{"A","B","C"})  // B列为A或B或C的数量
```

#### **技巧5：统计两列同时满足条件**
```excel
=COUNTIFS(A:A,"北京", B:B,">10000", C:C,"是")  // A列=北京、B列>10000、C列=是
```

## **四、三函数对比总结**

| 特性 | COUNT() | COUNTIF() | COUNTIFS() |
|------|---------|-----------|------------|
| **功能** | 统计数字个数 | 单条件计数 | 多条件计数 |
| **条件数量** | 无条件 | 单条件 | 多条件（最多127个） |
| **语法结构** | COUNT(区域) | COUNTIF(区域,条件) | COUNTIFS(区域1,条件1,区域2,条件2...) |
| **统计内容** | 只统计数字 | 根据条件统计 | 根据多个条件统计 |
| **通配符支持** | 不支持 | 支持 | 支持 |
| **性能** | 最快 | 中等 | 较慢（条件多时） |
| **版本要求** | 所有版本 | Excel 2000+ | Excel 2007+ |

## **六、常见问题与解决方案**

### **Q1：为什么COUNT统计结果为0？**
```excel
可能原因：区域内没有数字
检查：=COUNT(A1:A10)  // 如果都是文本，结果为0
解决：用COUNTA统计非空单元格：=COUNTA(A1:A10)
```

### **Q2：如何统计包含特定文本的单元格？**
```excel
包含"北京"：=COUNTIF(A:A,"*北京*")
以"公司"结尾：=COUNTIF(A:A,"*公司")
以"中国"开头：=COUNTIF(A:A,"中国*")
等于"北京"：=COUNTIF(A:A,"北京")
```

### **Q3：如何统计多个条件的"或"关系？**
```excel
// 方法1：多个COUNTIF相加
=COUNTIF(A:A,"北京")+COUNTIF(A:A,"上海")+COUNTIF(A:A,"广州")

// 方法2：使用SUM+COUNTIFS
=SUM(COUNTIFS(A:A,{"北京","上海","广州"}))

// 方法3：使用SUMPRODUCT
=SUMPRODUCT((A1:A100={"北京","上海","广州"})*1)
```

### **Q4：如何忽略错误值进行统计？**
```excel
统计非错误值：=COUNTIF(A:A,"<>#N/A")+COUNTIF(A:A,"<>#VALUE!")+...
或使用：=SUMPRODUCT(--NOT(ISERROR(A1:A10)))
```

### **Q5：COUNTIF与COUNTIFS参数顺序不同？**
```excel
COUNTIF：条件区域 → 条件
COUNTIFS：条件区域1 → 条件1 → 条件区域2 → 条件2 → ...

记忆：COUNTIF是单数，COUNTIFS是复数，复数需要更多参数
```

### **Q6：如何统计两个数字之间的数量？**
```excel
60-80之间：=COUNTIF(B:B,">=60")-COUNTIF(B:B,">80")
或：=COUNTIFS(B:B,">=60", B:B,"<=80")
```

### **Q7：如何统计不重复值的个数？**
```excel
=SUMPRODUCT(1/COUNTIF(A1:A10, A1:A10))
注意：如果有空单元格会出错，改进：
=SUMPRODUCT((A1:A10<>"")/COUNTIF(A1:A10, A1:A10&""))
```

## **七、性能优化建议**

1. **避免整列引用**：使用具体范围`A1:A1000`而非`A:A`
2. **减少条件数量**：COUNTIFS条件越多越慢
3. **先筛选后统计**：大数据集可先筛选再使用COUNT
4. **使用辅助列**：复杂条件可先计算辅助列，再简单统计
5. **数组公式慎用**：大量数据时避免使用数组公式

## **八、实际应用模板**

### **模板1：学生成绩统计系统**
```excel
A列：学号  B列：姓名  C列：班级  D列：语文  E列：数学  F列：英语

1. 各班人数：
   1班：=COUNTIF(C:C,"1班")
   2班：=COUNTIF(C:C,"2班")
   3班：=COUNTIF(C:C,"3班")

2. 各分数段人数：
   优秀(>=90)：=COUNTIF(D:F,">=90")
   良好(80-89)：=COUNTIFS(D:F,">=80", D:F,"<90")
   及格(60-79)：=COUNTIFS(D:F,">=60", D:F,"<80")
   不及格(<60)：=COUNTIF(D:F,"<60")

3. 三科都及格人数：
   =COUNTIFS(D:D,">=60", E:E,">=60", F:F,">=60")

4. 至少一科优秀人数：
   =COUNT(IF((D:D>=90)+(E:E>=90)+(F:F>=90)>0,1))  // 数组公式
```

### **模板2：销售订单分析系统**
```excel
A列：订单号  B列：销售员  C列：产品  D列：金额  E列：日期  F列：状态

1. 各销售员订单数：
   =COUNTIF(B:B,销售员姓名)

2. 各产品销量：
   =COUNTIF(C:C,产品名称)

3. 本月已完成订单：
   =COUNTIFS(E:E,">="&EOMONTH(TODAY(),-1)+1, 
              E:E,"<="&EOMONTH(TODAY(),0), 
              F:F,"已完成")

4. 高价值订单(金额>平均)数量：
   =COUNTIFS(D:D,">"&AVERAGE(D:D))

5. 统计各金额区间订单：
   <1000：=COUNTIFS(D:D,"<1000")
   1000-5000：=COUNTIFS(D:D,">=1000", D:D,"<5000")
   >=5000：=COUNTIFS(D:D,">=5000")
```

### **模板3：员工考勤统计**
```excel
A列：日期  B列：姓名  C列：上班时间  D列：下班时间  E列：状态

1. 各员工出勤天数：
   =COUNTIFS(B:B,员工姓名, E:E,"出勤")

2. 迟到早退统计：
   迟到：=COUNTIFS(B:B,员工姓名, E:E,"迟到")
   早退：=COUNTIFS(B:B,员工姓名, E:E,"早退")
   请假：=COUNTIFS(B:B,员工姓名, E:E,"请假")

3. 月度全勤人数：
   =COUNTIFS(E:E,"出勤", A:A,">="&月初, A:A,"<="&月末)
   // 然后统计出现次数=工作日数的员工
```

## **九、记忆技巧**

1. **COUNT**：数数字（Count Numbers）
2. **COUNTIF**：有条件的数（Count + IF）
3. **COUNTIFS**：多个条件数（Count + IFS）

**实用口诀**：
> 数数字用COUNT  
> 有条件用COUNTIF  
> 多条件COUNTIFS  
> 条件顺序别搞错

**特别提醒**：
- `COUNT`只认数字，文本逻辑都忽略
- `COUNTIF`条件要加引号（除非是单元格引用）
- `COUNTIFS`参数成对出现（区域1,条件1,区域2,条件2...）
