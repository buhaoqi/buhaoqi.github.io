## **一、MAX() 函数：最大值**

### **1. 基本语法**
```excel
=MAX(number1, [number2], ...)
```

### **2. 参数特性**
- 忽略文本、逻辑值、空单元格（除非直接作为参数）
- 支持最多255个参数
- 日期和时间作为数值处理
- 错误值会导致函数返回错误

---

## **二、MIN() 函数：最小值**

### **1. 基本语法**
```excel
=MIN(number1, [number2], ...)
```

### **3. 使用示例**

#### **示例1：直接数值运算**
```excel
=MAX(10, 20, 30, 5, 25)     // 结果：30
=MIN(10, 20, 30, 5, 25)     // 结果：5

=MAX(-10, -5, 0, 5, 10)     // 结果：10
=MIN(-10, -5, 0, 5, 10)     // 结果：-10
```

#### **示例2：单元格区域运算**
```excel
A1=10, A2=20, A3=30, A4="文本", A5=空
=MAX(A1:A5)                 // 结果：30（忽略文本和空值）
=MIN(A1:A5)                 // 结果：10
```

#### **示例3：混合参数运算**
```excel
=MAX(A1:A3, 100, B1:B2)     // 区域+数值+区域
=MIN(A1:A3, 0, B1:B2)       // 区域+数值+区域
```

#### **示例4：日期时间运算**
```excel
A1=2024-01-01, A2=2024-06-01, A3=2024-12-31
=MAX(A1:A3)                 // 结果：2024-12-31
=MIN(A1:A3)                 // 结果：2024-01-01
```

#### **示例5：特殊值处理**
```excel
A1=TRUE, A2=10, A3=FALSE
=MAX(A1:A3)                 // 结果：10（TRUE和FALSE被忽略）
=MIN(A1:A3)                 // 结果：10

=MAX(TRUE, 10, FALSE)       // 结果：10（TRUE=1，FALSE=0）
=MIN(TRUE, 10, FALSE)       // 结果：0（FALSE）
```

## **三、MAXA() 和 MINA()：包含文本和逻辑值**

### **1. MAXA() 函数**
```excel
=MAXA(value1, [value2], ...)
```
- 文本视为0
- TRUE视为1，FALSE视为0
- 空单元格被忽略

### **2. MINA() 函数**
```excel
=MINA(value1, [value2], ...)
```

### **3. 对比示例**
```excel
数据：A1=10, A2="文本", A3=TRUE, A4=FALSE, A5=空

MAX(A1:A5)    // 结果：10
MIN(A1:A5)    // 结果：10
MAXA(A1:A5)   // 结果：10
MINA(A1:A5)   // 结果：0（FALSE=0）

MAX(10, "文本", TRUE, FALSE)   // 结果：10
MIN(10, "文本", TRUE, FALSE)   // 结果：10
MAXA(10, "文本", TRUE, FALSE)  // 结果：10
MINA(10, "文本", TRUE, FALSE)  // 结果：0
```

## **四、条件极值函数：MAXIFS() 和 MINIFS()**

### **1. MAXIFS() 函数（Excel 2016+）**
```excel
=MAXIFS(max_range, criteria_range1, criteria1, 
        [criteria_range2, criteria2], ...)
```

### **2. MINIFS() 函数（Excel 2016+）**
```excel
=MINIFS(min_range, criteria_range1, criteria1,
        [criteria_range2, criteria2], ...)
```

### **3. 使用示例**

#### **示例1：单条件最大值/最小值**
```excel
A列：部门  B列：销售额

求销售部的最高销售额：
=MAXIFS(B:B, A:A,"销售部")

求销售部的最低销售额：
=MINIFS(B:B, A:A,"销售部")
```

#### **示例2：多条件极值**
```excel
求销售部在Q2的最高销售额：
=MAXIFS(销售额列, 部门列,"销售部", 季度列,"Q2")

求技术部工龄>3年的最低工资：
=MINIFS(工资列, 部门列,"技术部", 工龄列,">3")
```

#### **示例3：日期条件极值**
```excel
求2024年1月的最高温度：
=MAXIFS(温度列, 日期列,">=2024-01-01", 日期列,"<=2024-01-31")
```

## **五、实际应用场景**

### **场景1：销售数据分析**
```excel
数据表：销售员 | 产品 | 日期 | 销售额 | 利润

1. 最高单笔销售额：
   =MAX(D:D)

2. 最低单笔销售额（排除0）：
   =MINIFS(D:D, D:D,">0")

3. 手机产品的最高销售额：
   =MAXIFS(D:D, B:B,"手机")

4. 张三在Q2的最高销售额：
   =MAXIFS(D:D, A:A,"张三", C:C,">=2024-04-01", C:C,"<=2024-06-30")

5. 各产品线最高和最低：
   手机最高：=MAXIFS(D:D, B:B,"手机")
   手机最低：=MINIFS(D:D, B:B,"手机")
```

### **场景2：学生成绩分析**
```excel
数据表：姓名 | 班级 | 语文 | 数学 | 英语 | 总分

1. 年级最高分和最低分：
   最高：=MAX(F:F)
   最低：=MIN(F:F)

2. 1班数学最高分：
   =MAXIFS(D:D, B:B,"1班")

3. 及格分数中的最低分：
   =MINIFS(F:F, F:F,">=60")

4. 各科最高最低分：
   语文最高：=MAX(C:C)
   语文最低：=MIN(C:C)
   数学最高：=MAX(D:D)
   数学最低：=MIN(D:D)
```

### **场景3：库存价格管理**
```excel
数据表：商品 | 分类 | 进货价 | 零售价 | 库存

1. 最高进货价：
   =MAX(C:C)

2. 最低零售价：
   =MIN(D:D)

3. 食品类最高零售价：
   =MAXIFS(D:D, B:B,"食品")

4. 库存>0的最低零售价：
   =MINIFS(D:D, E:E,">0")

5. 非促销商品最高利润（零售价-进货价）：
   =MAXIFS(D:D-C:C, F:F,"<>促销")
```

### **场景4：温度监控**
```excel
数据表：时间 | 传感器1 | 传感器2 | 传感器3

1. 今日最高温度：
   =MAX(B:D)

2. 传感器1今日最低温度：
   =MIN(B:B)

3. 工作时间（9-18点）最高温度：
   =MAXIFS(B:D, A:A,">=9:00", A:A,"<=18:00")

4. 各传感器极端值：
   传感器1最高：=MAX(B:B)
   传感器1最低：=MIN(B:B)
   传感器2最高：=MAX(C:C)
   传感器2最低：=MIN(C:C)
```

## **六、高级技巧与应用**

### **技巧1：求第二/第三大/小值**
```excel
第一大：=MAX(A:A)
第二大：=LARGE(A:A, 2)
第三大：=LARGE(A:A, 3)

第一小：=MIN(A:A)
第二小：=SMALL(A:A, 2)
第三小：=SMALL(A:A, 3)
```

### **技巧2：排除0值求最小值**
```excel
=MINIFS(A:A, A:A,"<>0")          // Excel 2016+
=MIN(IF(A:A>0, A:A))            // 数组公式（Ctrl+Shift+Enter）
```

### **技巧3：求最大/最小值的位置**
```excel
最大值的行号：=MATCH(MAX(A:A), A:A, 0)
最小值的行号：=MATCH(MIN(A:A), A:A, 0)

最大值的内容：=INDEX(B:B, MATCH(MAX(A:A), A:A, 0))
```

### **技巧4：动态范围极值**
```excel
=MAX(OFFSET(A1,0,0,COUNTA(A:A),1))   // 动态范围的最大值
=MIN(OFFSET(A1,0,0,COUNTA(A:A),1))   // 动态范围的最小值
```

### **技巧5：求多列中的最大值**
```excel
=MAX(A:C)                           // 求A列到C列所有数据中的最大值
=MAX(A1:C100)                       // 指定区域的最大值
```

### **技巧6：条件格式结合**
```excel
条件格式标记最大值：=A1=MAX($A$1:$A$100)
条件格式标记最小值：=A1=MIN($A$1:$A$100)
```

### **技巧7：求最大/最小值的平均值**
```excel
去掉最高最低求平均：=(SUM(A:A)-MAX(A:A)-MIN(A:A))/(COUNT(A:A)-2)
```

## **七、函数对比总结**

| 函数 | 功能 | 忽略内容 | 包含内容 | 版本要求 |
|------|------|----------|----------|----------|
| **MAX()** | 最大值 | 文本、逻辑值、空单元格 | 数字、日期、时间 | 所有版本 |
| **MIN()** | 最小值 | 文本、逻辑值、空单元格 | 数字、日期、时间 | 所有版本 |
| **MAXA()** | 最大值 | 空单元格 | 数字、日期、时间、文本(0)、逻辑值(TRUE=1,FALSE=0) | 所有版本 |
| **MINA()** | 最小值 | 空单元格 | 数字、日期、时间、文本(0)、逻辑值(TRUE=1,FALSE=0) | 所有版本 |
| **MAXIFS()** | 多条件最大值 | 同MAX | 同MAX | Excel 2016+ |
| **MINIFS()** | 多条件最小值 | 同MIN | 同MIN | Excel 2016+ |
| **LARGE()** | 第N大值 | 同MAX | 同MAX | 所有版本 |
| **SMALL()** | 第N小值 | 同MIN | 同MIN | 所有版本 |

## **九、常见问题与解决方案**

### **Q1：为什么MAX/MIN返回0而不是预期值？**
```excel
可能原因：区域内没有数字，只有文本或空单元格
检查：=COUNT(A1:A10)  // 如果为0，则区域无数字
解决：检查数据是否为文本格式，或使用MAXA/MINA
```

### **Q2：如何忽略0值求最小值？**
```excel
Excel 2016+：=MINIFS(A:A, A:A,"<>0")
旧版本：=MIN(IF(A1:A10>0, A1:A10))  // 数组公式
```

### **Q3：如何求最大/最小值的出现次数？**
```excel
最大值出现次数：=COUNTIF(A:A, MAX(A:A))
最小值出现次数：=COUNTIF(A:A, MIN(A:A))
```

### **Q4：如何求绝对值最大/最小值？**
```excel
绝对值最大值：=MAX(ABS(A1:A10))  // 数组公式
绝对值最小值：=MIN(ABS(A1:A10))  // 数组公式
```

### **Q5：MAXIFS/MINIFS与MAX/MIN的参数顺序不同？**
```excel
MAX/MIN：求值区域作为参数
MAXIFS/MINIFS：求值区域作为第一个参数，然后是条件区域对

记忆：IFS函数都是"求值区域"在前
```

### **Q6：如何处理错误值？**
```excel
排除错误值求最大值：=MAX(IF(NOT(ISERROR(A1:A10)), A1:A10))  // 数组公式
排除错误值求最小值：=MIN(IF(NOT(ISERROR(A1:A10)), A1:A10))  // 数组公式
```

### **Q7：如何求多行多列区域的最大/最小值？**
```excel
整个区域最大值：=MAX(A1:Z100)
每行最大值：=MAX(A1:Z1)  // 向右填充
每列最大值：=MAX(A1:A100) // 向下填充
```

## **十、性能优化建议**

1. **避免整列引用**：使用具体范围`A1:A1000`而非`A:A`
2. **减少条件数量**：MAXIFS/MINIFS条件越多越慢
3. **先筛选后计算**：大数据集可先筛选再使用MAX/MIN
4. **使用辅助列**：复杂计算可先存储中间结果
5. **数组公式慎用**：大量数据时避免使用数组公式

## **十二、记忆技巧**

1. **MAX**：最大（Maximum）
2. **MIN**：最小（Minimum）
3. **MAXA/MINA**：包含A(ll)类型
4. **MAXIFS/MINIFS**：有IF条件的极值

**实用口诀**：
> 求最大用MAX  
> 求最小用MIN  
> 要条件加IFS  
> 含文本用A版

**特别提醒**：
- 条件极值函数从Excel 2016开始支持
- MAXA/MINA将文本视为0，可能影响结果
- 条件格式配合极值函数可实现视觉突出效果
