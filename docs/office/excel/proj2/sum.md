## **一、SUM() 函数：基础求和**

### **1. 基本语法**
```excel
=SUM(number1, [number2], ...)
```

### **2. 参数说明**
- **number1**：必需，第一个数字、单元格引用或区域
- **number2, ...**：可选，最多255个参数

### **3. 使用示例**

#### **示例1：直接数值求和**
```excel
=SUM(10, 20, 30)          // 结果：60
```

#### **示例2：单元格引用求和**
```excel
A1=10, A2=20, A3=30
=SUM(A1:A3)               // 结果：60
=SUM(A1, A2, A3)          // 结果：60
```

#### **示例3：混合参数**
```excel
=SUM(A1:A3, 40, B1:B2)    // 区域+数值+区域
```

#### **示例4：跳过非数值**
```excel
A1=10, A2="文本", A3=20, A4=TRUE
=SUM(A1:A4)               // 结果：31（10+20+1，TRUE转为1，文本被忽略）
```

### **4. 特殊功能**
```excel
=SUM(A1:A10*B1:B10)       // 数组公式（按Ctrl+Shift+Enter）
=SUM(--(A1:A10>50))       // 统计大于50的个数
```

## **二、SUMIF() 函数：单条件求和**

### **1. 基本语法**
```excel
=SUMIF(range, criteria, [sum_range])
```

### **2. 参数详解**
| 参数 | 说明 | 必需 |
|------|------|------|
| **range** | 要应用条件的单元格区域 | 是 |
| **criteria** | 条件，可以是数字、表达式、文本或单元格引用 | 是 |
| **sum_range** | 要求和的实际单元格区域（可选，省略时对range求和） | 否 |

### **3. 条件写法大全**

#### **文本条件**
```excel
=SUMIF(A:A,"苹果",B:B)      // 等于"苹果"
=SUMIF(A:A,"*果",B:B)       // 以"果"结尾
=SUMIF(A:A,"苹*",B:B)       // 以"苹"开头
=SUMIF(A:A,"?果",B:B)       // 两个字且以"果"结尾
=SUMIF(A:A,"<>苹果",B:B)    // 不等于"苹果"
```

#### **数字条件**
```excel
=SUMIF(B:B,">100")          // 大于100
=SUMIF(B:B,">=500")         // 大于等于500
=SUMIF(B:B,"<>0")           // 不等于0
=SUMIF(B:B,"<1000")         // 小于1000
=SUMIF(B:B,100)             // 等于100
```

#### **日期条件**
```excel
=SUMIF(C:C,">=2024-01-01",D:D)
=SUMIF(C:C,">"&DATE(2024,1,1),D:D)
=SUMIF(C:C,">="&TODAY(),D:D)
```

#### **单元格引用作为条件**
```excel
=SUMIF(A:A,D2,B:B)          // D2的内容作为条件
=SUMIF(B:B,">"&E2)          // 大于E2的值
```

### **4. 实际应用示例**

#### **销售数据统计**
```excel
A列：销售员  B列：销售额

求"张三"的销售额：=SUMIF(A:A,"张三",B:B)
求销售额>5000的总和：=SUMIF(B:B,">5000")
求非"张三"的销售额：=SUMIF(A:A,"<>张三",B:B)
```

#### **学生成绩统计**
```excel
A列：班级  B列：成绩

求1班总成绩：=SUMIF(A:A,"1班",B:B)
求及格(>=60)总成绩：=SUMIF(B:B,">=60")
求姓"李"的学生成绩：=SUMIF(A:A,"李*",B:B)
```

### **5. 常见错误及处理**
```excel
错误写法：=SUMIF(B:B,">D2")       // D2会被当作文本
正确写法：=SUMIF(B:B,">"&D2)     // 用&连接

错误：=SUMIF(A:A,条件,B1:B10)     // 区域大小不一致
正确：=SUMIF(A1:A10,条件,B1:B10)  // 区域大小相同
```

## **三、SUMIFS() 函数：多条件求和**

### **1. 基本语法**
```excel
=SUMIFS(sum_range, criteria_range1, criteria1, 
        [criteria_range2, criteria2], ...)
```

### **2. 参数说明**
| 参数 | 说明 |
|------|------|
| **sum_range** | 要求和的实际单元格区域 |
| **criteria_range1** | 第一个条件的区域 |
| **criteria1** | 第一个条件 |
| **criteria_range2, criteria2, ...** | 可选，最多127个条件对 |

### **3. 使用示例**

#### **示例1：双条件求和**
```excel
A列：部门  B列：月份  C列：销售额

求"销售部"在"1月"的销售额：
=SUMIFS(C:C, A:A,"销售部", B:B,"1月")
```

#### **示例2：多条件组合**
```excel
求"销售部"在"1月"且销售额>10000的记录：
=SUMIFS(C:C, A:A,"销售部", B:B,"1月", C:C,">10000")
```

#### **示例3：日期范围条件**
```excel
A列：日期  B列：销售额

求2024年1月的销售额：
=SUMIFS(B:B, A:A,">=2024-01-01", A:A,"<=2024-01-31")
```

### **4. 实际工作场景**

#### **场景1：销售数据分析**
```excel
数据表：销售员 | 产品 | 地区 | 销售额 | 日期

1. 张三在华东区销售手机的总金额：
   =SUMIFS(销售额列, 销售员列,"张三", 产品列,"手机", 地区列,"华东")

2. 2024年Q1（1-3月）电脑销售额：
   =SUMIFS(销售额列, 产品列,"电脑", 日期列,">=2024-01-01", 日期列,"<=2024-03-31")

3. 销售额>10000且利润>20%的记录：
   =SUMIFS(销售额列, 销售额列,">10000", 利润列,">0.2")
```

#### **场景2：库存管理**
```excel
数据表：商品ID | 类别 | 仓库 | 库存量 | 预警值

1. "食品"类在"北京仓"的库存总量：
   =SUMIFS(库存量列, 类别列,"食品", 仓库列,"北京仓")

2. 库存低于预警值的"电子产品"总量：
   =SUMIFS(库存量列, 类别列,"电子产品", 库存量列,"<"&预警值列)

3. 排除"已停用"商品的库存：
   =SUMIFS(库存量列, 状态列,"<>已停用")
```

### **5. 高级技巧**

#### **技巧1：使用通配符**
```excel
=SUMIFS(C:C, A:A,"*分公司", B:B,"?月")  // 以"分公司"结尾且月份是两个字
```

#### **技巧2：条件为公式结果**
```excel
=SUMIFS(C:C, B:B,">"&AVERAGE(B:B))  // 大于平均值
```

#### **技巧3：处理文本型数字**
```excel
=SUMIFS(C:C, A:A,TEXT(B2,"0"))      // B2是文本型数字
```

## **四、三函数对比总结**

| 特性 | SUM() | SUMIF() | SUMIFS() |
|------|-------|---------|----------|
| **条件数量** | 无条件 | 单条件 | 多条件（最多127个） |
| **语法顺序** | SUM(求和区域) | SUMIF(条件区域,条件,求和区域) | SUMIFS(求和区域,条件区域1,条件1,...) |
| **求和区域** | 直接是参数 | 第三个参数 | 第一个参数 |
| **性能** | 最快 | 中等 | 较慢（条件多时） |
| **适用场景** | 简单求和 | 按单个条件筛选求和 | 复杂多条件筛选求和 |
| **通配符** | 不支持 | 支持 | 支持 |
| **数组公式** | 支持 | 有限支持 | 有限支持 |

## **六、常见问题解答**

### **Q1：为什么SUMIFS的第一个参数是求和区域？**
这是为了与SUMIF区分，避免混淆。记住口诀：
- SUMIF：条件在前，求和在后
- SUMIFS：求和在前，条件在后

### **Q2：如何处理条件区域和求和区域大小不一致？**
```excel
错误：=SUMIF(A:A,条件,B1:B100)     // A列和B列行数不同
正确：=SUMIF(A1:A100,条件,B1:B100) // 指定相同范围
```

### **Q3：如何求和包含特定文本的单元格？**
```excel
=SUMIF(A:A,"*包含的文本*",B:B)
```

### **Q4：如何忽略错误值？**
```excel
=SUMIF(B:B,"<>#N/A")  // 忽略#N/A错误
=SUMIF(B:B,"<>#DIV/0!")  // 忽略除零错误
```

### **Q5：如何求和可见单元格（筛选后）？**
使用SUBTOTAL函数：
```excel
=SUBTOTAL(109,求和区域)  // 109表示SUM且忽略隐藏行
```

## **七、性能优化建议**

1. **避免整列引用**：尽量使用具体范围`A1:A1000`而非`A:A`
2. **减少条件数量**：SUMIFS的条件越多，计算越慢
3. **先筛选后计算**：大数据集可先筛选再使用SUM
4. **使用表格结构化引用**：`=SUM(Table1[销售额])`

## **八、记忆技巧**

1. **SUM**：最简单，就是"加"
2. **SUMIF**：SUM + IF（如果）→ 如果满足条件就加
3. **SUMIFS**：SUM + IFS（多个如果）→ 如果满足多个条件就加

**实用口诀**：
> SUM无条件，全部加起来  
> SUMIF单条件，符合条件的加  
> SUMIFS多条件，全部符合才加

掌握了这三个函数，你就能解决Excel中90%的求和需求！
