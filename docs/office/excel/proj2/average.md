Excel平均值函数详解：AVERAGE()、AVERAGEIF()、AVERAGEIFS()

## **一、函数家族概述**

**1. 核心函数对比表**


| 特性 | AVERAGE() | AVERAGEIF() | AVERAGEIFS() |
|------|-----------|-------------|--------------|
| **用途** | 求行求列求区域平均值 | 单条件求单列平均值 | 多条件 |
| **语法顺序** | AVERAGE(求值区域) | AVERAGEIF(条件区域,条件,求值区域) | AVERAGEIFS(求值区域,条件区域1,条件1,...) |
| **求值区域位置** | 直接参数 | 第三个参数 | 第一个参数 |
| **忽略的值** | 文本、空单元格、逻辑值（在区域中） | 同AVERAGE | 同AVERAGE |
| **通配符支持** | 不支持 | 支持 | 支持 |


**2. 重要特性**

- 忽略**文本**、**逻辑值**、**空单元格**
- 最多255个参数（AVERAGE）
- 最多127个条件对（AVERAGEIFS）
- 支持通配符（* ? ~）
- 返回**数字**类型

---

## **二、AVERAGE() 函数**

用途：求行、求列、求区域的平均值

基本语法

```excel
=AVERAGE(number1, [number2], ...)
```

参数特性

- 忽略文本、逻辑值、空单元格（除非直接作为参数）
- TRUE视为1，FALSE视为0（仅当直接作为参数时）
- 支持最多255个参数

使用示例

示例1：直接数值计算

```excel
=AVERAGE(10, 20, 30)          // 结果：20
=AVERAGE(10, 20, 30, 0)       // 结果：15
```

示例2：单元格区域计算

```excel
A1=10, A2=20, A3=30, A4="文本", A5为空
=AVERAGE(A1:A5)               // 结果：20 (只计算10,20,30，忽略文本和空单元格)
```

示例3：混合参数计算

```excel
=AVERAGE(A1:A3, 40, B1:B2)    // 区域+数值+区域
```

示例4：特殊值处理

```excel
A1=TRUE, A2=10, A3=FALSE
=AVERAGE(A1:A3)               // 结果：10 (只计算10，TRUE和FALSE被忽略)
=AVERAGE(TRUE, 10, FALSE)     // 结果：3.6667 (1+10+0)/3
```

## **二、AVERAGEIF() 函数：**

用途：单条件求列求行平均值

它的工作方式是：在“条件区域”中找符合“条件”的单元格，然后返回对应位置的“求值区域”的值进行平均。

但求值区域必须是单列或单行，不能是多列区域，否则会出错或返回意想不到的结果。



基本语法

```excel
=AVERAGEIF(range, criteria, [average_range])
```

参数详解

| 参数 | 说明 | 必需 |
|------|------|------|
| **range** | 条件判断区域 | 是 |
| **criteria** | 条件表达式 | 是 |
| **average_range** | 实际求平均的区域（可选，省略时对range求平均） | 否 |

**文本条件**

```excel
=AVERAGEIF(A:A,"苹果",B:B)      // A列为"苹果"时，求B列平均值
=AVERAGEIF(A:A,"*果",B:B)       // 以"果"结尾
=AVERAGEIF(A:A,"苹*",B:B)       // 以"苹"开头
=AVERAGEIF(A:A,"<>苹果",B:B)    // 不等于"苹果"
```

**数字条件**

```excel
=AVERAGEIF(B:B,">60")           // 大于60的平均值
=AVERAGEIF(B:B,">=80")          // 大于等于80
=AVERAGEIF(B:B,"<>0")           // 不等于0
=AVERAGEIF(B:B,75)              // 等于75（通常无意义，但语法正确）
```

**日期条件**

```excel
=AVERAGEIF(C:C,">=2024-01-01",D:D)
=AVERAGEIF(C:C,">"&DATE(2024,6,1),D:D)
```

**单元格引用作为条件**

```excel
=AVERAGEIF(A:A,D2,B:B)          // D2的内容作为条件
=AVERAGEIF(B:B,">"&E2)          // 大于E2的值
```
### 单行求平均值

```
月份：  Jan  Feb  Mar  Apr  May  Jun
销售额：100  80   120  90   150  70
目标：  >90  >90  >90  >90  >90  >90
```

**示例1：学生成绩分析**

```excel
A列：班级  B列：成绩

1班平均分：=AVERAGEIF(A:A,"1班",B:B)
及格(>=60)平均分：=AVERAGEIF(B:B,">=60")
姓"王"的学生平均分：=AVERAGEIF(A:A,"王*",B:B)
排除缺考(0分)：=AVERAGEIF(B:B,"<>0")
```

**示例2：销售数据分析**

```excel
A列：产品  B列：销售额  C列：利润

手机平均销售额：=AVERAGEIF(A:A,"手机",B:B)
高销售额(>10000)平均利润：=AVERAGEIF(B:B,">10000",C:C)
非促销产品平均销售额：=AVERAGEIF(D:D,"<>促销",B:B)
```

**5. 常见错误示例**

```excel
错误：=AVERAGEIF(A:A,">D2",B:B)      // D2被当作文本
正确：=AVERAGEIF(A:A,">"&D2,B:B)    // 使用&连接

错误：=AVERAGEIF(A1:A10,"条件",B1:B9) // 区域大小不一致
正确：=AVERAGEIF(A1:A10,"条件",B1:B10)
```

## **三、AVERAGEIFS() 函数**

用途：多条件平均值

**基本语法**

```excel
=AVERAGEIFS(average_range, criteria_range1, criteria1, 
            [criteria_range2, criteria2], ...)
```

**参数说明**

| 参数 | 说明 |
|------|------|
| **average_range** | 要求平均的实际区域（第一个参数！） |
| **criteria_range1** | 第一个条件的区域 |
| **criteria1** | 第一个条件 |
| **criteria_range2, criteria2, ...** | 可选，最多127个条件对 |


**示例1：双条件平均值**

```excel
A列：部门  B列：职级  C列：工资

求"技术部""高级"员工的平均工资：
=AVERAGEIFS(C:C, A:A,"技术部", B:B,"高级")
```

**示例2：多条件组合**

```excel
求"销售部"在"北京"且工龄>5年的平均业绩：
=AVERAGEIFS(业绩列, 部门列,"销售部", 地区列,"北京", 工龄列,">5")
```

**示例3：日期范围条件**

```excel
A列：日期  B列：温度

求2024年夏季(6-8月)平均温度：
=AVERAGEIFS(B:B, A:A,">=2024-06-01", A:A,"<=2024-08-31")
```

**场景1：员工绩效分析**

```excel
数据表：部门 | 职级 | 工龄 | 绩效分数 | 奖金

1. 技术部P7级平均绩效：
   =AVERAGEIFS(绩效分数列, 部门列,"技术部", 职级列,"P7")

2. 工龄3-5年且绩效>85的平均奖金：
   =AVERAGEIFS(奖金列, 工龄列,">=3", 工龄列,"<=5", 绩效分数列,">85")

3. 排除试用期员工的平均绩效：
   =AVERAGEIFS(绩效分数列, 状态列,"<>试用期")
```

**场景2：产品质量分析**

```excel
数据表：生产线 | 班次 | 日期 | 产品型号 | 合格率

1. A生产线早班的平均合格率：
   =AVERAGEIFS(合格率列, 生产线列,"A线", 班次列,"早班")

2. 2024年Q2型号X100的平均合格率：
   =AVERAGEIFS(合格率列, 产品型号列,"X100", 
               日期列,">=2024-04-01", 日期列,"<=2024-06-30")

3. 合格率>95%且产量>1000的平均数据：
   =AVERAGEIFS(某指标列, 合格率列,">0.95", 产量列,">1000")
```

**场景3：销售数据分析**

```excel
数据表：销售员 | 产品类别 | 季度 | 销售额 | 客户评分

1. 张三在Q2销售电器的平均销售额：
   =AVERAGEIFS(销售额列, 销售员列,"张三", 
               产品类别列,"电器", 季度列,"Q2")

2. 客户评分>4且销售额>10000的平均利润率：
   =AVERAGEIFS(利润率列, 客户评分列,">4", 销售额列,">10000")

3. 排除退货订单的平均交易额：
   =AVERAGEIFS(交易额列, 订单状态列,"<>退货")
```

```excel
=AVERAGEIFS(H2:H13, C2:C13, "手机", J2:J13, "是")

结果：类别为"手机"且正在促销的产品平均售价
```

**示例17：复杂条件平均值**

```excel
计算库存充足（>安全库存）且销量>100的产品平均成本：
=AVERAGEIFS(I2:I13, D2:D13, ">"&E2:E13, F2:F13, ">100")

注意：这里E2:E13需要逐行比较，实际应该为：
=AVERAGEIFS(I2:I13, D2:D13, ">="&E2:E13, F2:F13, ">100")
但Excel不支持数组条件，正确做法：
使用SUMPRODUCT或辅助列
```

**示例18：排除特定条件计算**

```excel
计算非食品类且非促销产品的平均库存：
=AVERAGEIFS(D2:D13, C2:C13, "<>食品", J2:J13, "否")

结果：类别不是食品且非促销的产品平均库存量
```

**示例19：多条件日期范围**

```excel
假设表2有月份列，计算Q1销售部业绩：
=AVERAGEIFS(业绩列, 部门列, "销售部", 日期列, ">=2024-01-01", 日期列, "<=2024-03-31")
```

## 六、AVERAGE的特殊情况

**忽略0值计算平均值**

```excel
假设有0值表示缺考，要忽略0值：
=AVERAGEIF(D2:D20, "<>0")

或者数组公式：
=AVERAGE(IF(D2:D20>0, D2:D20))
```

**处理错误值**

```excel
使用IFERROR：
=AVERAGE(IFERROR(D2:D20, ""))

或者AGGREGATE：
=AGGREGATE(1, 6, D2:D20)  // 1=平均值，6=忽略错误
```

## **七、常见问题与解决方案**

### **Q1：为什么平均值显示为#DIV/0!？**
```excel
原因：没有满足条件的数值
解决：=IFERROR(AVERAGEIF(...), "无数据")
```

### **Q2：如何计算加权平均值？**
```excel
普通平均值：=AVERAGE(B2:B10)
加权平均值：=SUMPRODUCT(B2:B10, C2:C10)/SUM(C2:C10)
```

### **Q3：如何处理包含0值的平均值？**
```excel
包含0：=AVERAGE(A1:A10)
排除0：=AVERAGEIF(A1:A10,"<>0")
```

### **Q4：如何计算前N个最高值的平均值？**
```excel
前3名平均值：=AVERAGE(LARGE(A1:A100, {1,2,3}))
```

### **Q5：AVERAGE与AVERAGEA的区别？**
```excel
数据：A1=10, A2="缺考", A3=TRUE, A4=20

AVERAGE(A1:A4)    // 结果：15 (只计算10,20)
AVERAGEA(A1:A4)   // 结果：7.75 (10+0+1+20)/4
```

### **Q6：如何忽略错误值计算平均值？**
```excel
=AVERAGEIF(A1:A10,"<>#N/A")
=AGGREGATE(1, 6, A1:A10)  // 忽略所有错误
```

## 九、总结
**AVERAGE家族是Excel的"平均计算器"——AVERAGE是基础款，AVERAGEIF是智能款，AVERAGEIFS是专业款，满足你从简单到复杂的所有平均值计算需求！**

**记忆口诀**
```
AVERAGE求平均，数值相加除个数
AVERAGEIF加条件，筛选之后再平均
AVERAGEIFS多条件，复杂分析真方便
参数顺序要记清，区域大小要一致
```
