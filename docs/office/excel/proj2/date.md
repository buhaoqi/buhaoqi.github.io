# Excel日期时间函数详解

## **一、DATE() 函数：构造日期**

### **1. 基本语法**
```excel
=DATE(year, month, day)
```

### **2. 参数说明**
| 参数 | 说明 | 有效范围 |
|------|------|----------|
| **year** | 年份 | 1900-9999（Windows系统） |
| **month** | 月份 | 1-12（支持负数，可跨年计算） |
| **day** | 日 | 1-31（支持负数，可跨月计算） |

### **3. 使用示例**

#### **示例1：基本日期构造**
```excel
=DATE(2024, 12, 31)        // 2024年12月31日
=DATE(2024, 1, 1)          // 2024年1月1日
```

#### **示例2：自动调整功能**
```excel
=DATE(2024, 13, 1)         // 2025年1月1日（13月自动转为下一年1月）
=DATE(2024, 0, 15)         // 2023年12月15日（0月自动转为上一年12月）
=DATE(2024, 1, 32)         // 2024年2月1日（32日自动转为下个月）
=DATE(2024, 2, -1)         // 2024年1月30日（负数自动往前推算）
```

#### **示例3：动态日期构造**
```excel
=DATE(YEAR(TODAY()), MONTH(TODAY()), 1)  // 本月第一天
=DATE(YEAR(TODAY()), MONTH(TODAY())+1, 1) // 下个月第一天
=DATE(YEAR(TODAY())+1, 1, 1)            // 明年1月1日
```

#### **示例4：日期计算**
```excel
=DATE(2024, 3, 15) + 30    // 2024年4月14日（加30天）
=DATE(2024, 6, 1) - DATE(2024, 1, 1)  // 152（天数差）
```

### **4. 实际应用场景**

#### **场景1：生成固定日期序列**
```excel
A1: =DATE(2024, 1, 1)
A2: =A1 + 1                // 2024年1月2日
向下填充 → 生成连续日期
```

#### **场景2：计算合同到期日**
```excel
签约日：2024-03-15（A1）
合同期：12个月（B1）

到期日：=DATE(YEAR(A1), MONTH(A1)+B1, DAY(A1))  // 2025-03-15
```

#### **场景3：处理不规范的日期数据**
```excel
A1=2024, B1=13, C1=5      // 月份为13（不规范）
规范日期：=DATE(A1, B1, C1)  // 2025年1月5日（自动调整）
```

## **二、TODAY() 函数：获取当前日期**

### **1. 基本语法**
```excel
=TODAY()
```

### **2. 特性说明**
- **无参数**，直接使用
- **动态更新**：每次打开工作簿或按F9时更新
- **只含日期**，不含时间（时间部分为0:00）
- **格式**：显示为日期格式

### **3. 使用示例**

#### **示例1：基本使用**
```excel
=TODAY()                  // 返回当前日期，如：2024-01-02
```

#### **示例2：日期计算**
```excel
=TODAY() + 7              // 7天后
=TODAY() - 1              // 昨天
=TODAY() + 30             // 30天后
```

#### **示例3：条件判断**
```excel
=IF(A1 > TODAY(), "未到期", "已到期")
=IF(TODAY() - A1 > 365, "超一年", "一年内")
```

#### **示例4：生成日期标记**
```excel
=IF(TODAY() = A1, "今天到期", "")
=IF(A1 < TODAY(), "已过期", "有效")
```

### **4. 实际应用场景**

#### **场景1：动态标题**
```excel
="截至" & TEXT(TODAY(), "yyyy年mm月dd日") & "的销售报表"
// 结果：截至2024年01月02日的销售报表
```

#### **场景2：计算年龄/工龄**
```excel
生日：1990-05-20（A1）
年龄：=DATEDIF(A1, TODAY(), "Y") & "岁"
```

#### **场景3：倒计时**
```excel
截止日：2024-12-31（A1）
剩余天数：=A1 - TODAY()   // 自动更新
```

#### **场景4：自动标记即将到期项目**
```excel
到期日（A列），状态标记（B列）：
=IF(A1-TODAY()<=7, "即将到期", IF(A1<TODAY(), "已过期", "正常"))
```

## **三、NOW() 函数：获取当前日期时间**

### **1. 基本语法**
```excel
=NOW()
```

### **2. 特性说明**
- **无参数**，直接使用
- **动态更新**：每次打开、按F9或单元格计算时更新
- **包含日期和时间**
- **格式**：显示为日期时间格式

### **3. 使用示例**

#### **示例1：基本使用**
```excel
=NOW()                    // 如：2024-01-02 10:30:45
```

#### **示例2：提取日期或时间**
```excel
=INT(NOW())               // 只要日期部分
=NOW() - INT(NOW())       // 只要时间部分（小数形式）
=TEXT(NOW(), "hh:mm:ss")  // 只要时间部分（文本格式）
```

#### **示例3：时间计算**
```excel
=NOW() + 1/24             // 1小时后
=NOW() + 30/(24*60)       // 30分钟后
=NOW() + 1                // 1天后（含时间）
```

#### **示例4：记录时间戳**
在A1输入数据时，B1自动记录时间：
```excel
B1: =IF(A1<>"", NOW(), "")  // 但会不断更新，不是好方法
```

**正确的时间戳记录方法**：
```vba
' 使用VBA，Worksheet_Change事件
Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("A:A")) Is Nothing Then
        Target.Offset(0, 1).Value = Now
    End If
End Sub
```

### **4. 实际应用场景**

#### **场景1：动态报表时间**
```excel
="报表生成时间：" & TEXT(NOW(), "yyyy-mm-dd hh:mm:ss")
```

#### **场景2：计算耗时**
```excel
开始时间：2024-01-01 09:00:00（A1）
当前耗时：=NOW() - A1            // 天数差（含小数）
耗时小时：=(NOW()-A1)*24        // 转换为小时
```

#### **场景3：自动刷新数据**
结合其他函数实现定时刷新效果（需开启自动计算）。

## **四、YEAR()、MONTH()、DAY() 函数：提取日期部分**

### **1. YEAR() 函数**
```excel
=YEAR(serial_number)
```
**作用**：提取日期的年份部分

#### **示例：**
```excel
=YEAR("2024-12-31")        // 2024
=YEAR(TODAY())             // 当前年份
=YEAR(A1)                  // A1单元格日期的年份
```

### **2. MONTH() 函数**
```excel
=MONTH(serial_number)
```
**作用**：提取日期的月份部分（1-12）

#### **示例：**
```excel
=MONTH("2024-12-31")       // 12
=MONTH(TODAY())            // 当前月份
=MONTH(A1)                 // A1单元格日期的月份
```

### **3. DAY() 函数**
```excel
=DAY(serial_number)
```
**作用**：提取日期的日部分（1-31）

#### **示例：**
```excel
=DAY("2024-12-31")         // 31
=DAY(TODAY())              // 今天是几号
=DAY(A1)                   // A1单元格日期的日
```

### **4. 组合使用示例**

#### **示例1：计算季度**
```excel
=INT((MONTH(A1)-1)/3)+1    // 返回1-4季度
或
=LOOKUP(MONTH(A1), {1,4,7,10}, {1,2,3,4})
```

#### **示例2：计算年龄（精确）**
```excel
出生日期：1990-05-20（A1）
年龄：=YEAR(TODAY())-YEAR(A1)-IF(TODAY()<DATE(YEAR(TODAY()),MONTH(A1),DAY(A1)),1,0)
```

#### **示例3：生成月初月末日期**
```excel
任意日期：2024-06-15（A1）
月初：=DATE(YEAR(A1), MONTH(A1), 1)              // 2024-06-01
月末：=DATE(YEAR(A1), MONTH(A1)+1, 0)           // 2024-06-30
或
=EOMONTH(A1, 0)                                 // 月末（需要分析工具库）
```

#### **示例4：计算工作日**
```excel
开始日：A1，结束日：B1
工作日数：=NETWORKDAYS(A1, B1)
考虑假日：=NETWORKDAYS(A1, B1, 假日范围)
```

### **5. 实际应用场景**

#### **场景1：生日提醒**
```excel
生日列：A列（如：1990-05-20）
今日生日：=AND(MONTH(A1)=MONTH(TODAY()), DAY(A1)=DAY(TODAY()))
本月生日：=MONTH(A1)=MONTH(TODAY())
```

#### **场景2：按年月汇总数据**
```excel
日期列：A列，金额列：B列
各月合计：=SUMIFS(B:B, A:A, ">="&DATE(2024,1,1), A:A, "<="&DATE(2024,1,31))
或使用数据透视表
```

#### **场景3：计算服务年限**
```excel
入职日：2018-03-15（A1）
服务年限：=YEAR(TODAY())-YEAR(A1) & "年" & MONTH(TODAY())-MONTH(A1) & "个月"
```

#### **场景4：动态月份标题**
```excel
=TEXT(DATE(2024, SEQUENCE(12), 1), "mmmm")  // 生成1-12月名称
```

## **五、函数对比总结**

| 函数 | 功能 | 返回值 | 是否动态 | 示例 |
|------|------|--------|----------|------|
| **DATE()** | 构造日期 | 日期值 | 否 | =DATE(2024,12,31) |
| **TODAY()** | 当前日期 | 日期值 | 是 | =TODAY() |
| **NOW()** | 当前日期时间 | 日期时间值 | 是 | =NOW() |
| **YEAR()** | 提取年份 | 数字(1900-9999) | 取决于参数 | =YEAR(A1) |
| **MONTH()** | 提取月份 | 数字(1-12) | 取决于参数 | =MONTH(A1) |
| **DAY()** | 提取日 | 数字(1-31) | 取决于参数 | =DAY(A1) |

## **六、日期相关重要技巧**

### **1. 日期格式转换**
```excel
日期转文本：=TEXT(A1, "yyyy-mm-dd")     // 2024-01-02
文本转日期：=DATEVALUE("2024/1/2")      // 对应日期序列值
```

### **2. 日期序列数理解**
Excel中日期本质是序列数：
- 1900年1月1日 = 1
- 2024年1月2日 ≈ 45293
```excel
=DATE(1900,1,1)   // 显示1（需设为常规格式）
```

### **3. 工作日计算**
```excel
=WORKDAY(开始日期, 天数, [假期])        // 计算N个工作日后的日期
=NETWORKDAYS(开始日期, 结束日期, [假期]) // 计算两个日期间的工作日数
```

### **4. 日期函数组合**
```excel
// 计算上月同期
=DATE(YEAR(A1), MONTH(A1)-1, DAY(A1))

// 计算去年同月
=DATE(YEAR(A1)-1, MONTH(A1), DAY(A1))

// 计算本季度最后一天
=DATE(YEAR(A1), INT((MONTH(A1)-1)/3)*3+4, 0)
```

### **5. 常见日期错误处理**
```excel
// 检查是否为日期
=ISNUMBER(A1)   // 日期是数字，返回TRUE

// 处理文本型日期
=IF(ISNUMBER(A1), A1, DATEVALUE(A1))

// 防止#VALUE!错误
=IFERROR(DATE(year,month,day), "日期无效")
```

## **七、综合实战案例**

### **案例1：员工合同管理系统**

```excel
A列：姓名 | B列：入职日期 | C列：合同期限(月) | D列：到期日 | E列：状态

D2（到期日）：
=DATE(YEAR(B2), MONTH(B2)+C2, DAY(B2))

E2（状态）：
=IF(D2<TODAY(), "已过期",
     IF(D2-TODAY()<=30, "即将到期", "正常"))

F2（剩余天数）：
=D2-TODAY()
```

### **案例2：销售数据分析报表**

```excel
A列：销售日期 | B列：销售金额 | C列：销售员

本月销售总额：
=SUMIFS(B:B, A:A, ">="&DATE(YEAR(TODAY()), MONTH(TODAY()), 1),
             A:A, "<="&DATE(YEAR(TODAY()), MONTH(TODAY())+1, 0))

各月销售趋势（使用辅助列）：
E1：=TEXT(DATE(2024,1,1), "yyyy-mm")
F1：=SUMIFS(B:B, A:A, ">="&DATE(2024,1,1), A:A, "<="&DATE(2024,1,31))
```

### **案例3：项目进度跟踪表**

```excel
A列：任务名称 | B列：开始日期 | C列：计划天数 | D列：计划完成日 | E列：实际完成日 | F列：状态

D2（计划完成日）：
=B2+C2

F2（状态）：
=IF(E2<>"", "已完成",
     IF(TODAY()>D2, "延期",
        IF(TODAY()>=B2, "进行中", "未开始")))

G2（延期天数）：
=IF(AND(E2="", TODAY()>D2), TODAY()-D2, "")
```

### **案例4：生日提醒系统**

```excel
A列：姓名 | B列：生日 | C列：年龄 | D列：下次生日 | E列：提醒

C2（年龄）：
=YEAR(TODAY())-YEAR(B2)-IF(TODAY()<DATE(YEAR(TODAY()),MONTH(B2),DAY(B2)),1,0)

D2（下次生日）：
=DATE(YEAR(TODAY())+(DATE(YEAR(TODAY()),MONTH(B2),DAY(B2))<TODAY()), MONTH(B2), DAY(B2))

E2（提醒）：
=IF(D2-TODAY()=0, "今天生日",
     IF(D2-TODAY()<=7, "本周生日",
        IF(MONTH(D2)=MONTH(TODAY()), "本月生日", "")))
```

## **八、常见问题与解决方案**

### **Q1：为什么DATE函数返回#NUM!错误？**
```excel
原因：年份<1900或>9999
解决：=IF(OR(year<1900, year>9999), "年份无效", DATE(year,month,day))
```

### **Q2：TODAY()和NOW()不断变化怎么办？**
```excel
需求：记录固定时间点（如填写时间）
方案1：手动输入，或使用快捷键 Ctrl+;（日期）Ctrl+Shift+;（时间）
方案2：使用VBA Worksheet_Change事件
方案3：复制粘贴为值
```

### **Q3：如何冻结TODAY/NOW的值？**
```excel
步骤：
1. 输入 =TODAY()
2. 按F9（显示计算结果）
3. Ctrl+C复制
4. 右键 → 粘贴值
```

### **Q4：日期显示为数字怎么办？**
```excel
原因：单元格格式为"常规"
解决：
1. 选中单元格
2. Ctrl+1 → 数字 → 日期 → 选择格式
或
=TEXT(A1, "yyyy-mm-dd")  // 转为文本
```

### **Q5：如何计算两个日期之间的月数差？**
```excel
=DATEDIF(开始日期, 结束日期, "M")  // 总月数
=DATEDIF(开始日期, 结束日期, "YM") // 忽略年的月数差
```

### **Q6：如何获取当月的最后一天？**
```excel
=EOMONTH(日期, 0)                // Excel 2007+
=DATE(YEAR(日期), MONTH(日期)+1, 0)  // 所有版本
```

## **九、最佳实践建议**

### **1. 日期输入规范**
- 使用 `YYYY-MM-DD` 格式
- 或使用DATE函数构造
- 避免直接输入文本型日期

### **2. 公式优化**
- 频繁使用的日期计算可存入辅助单元格
- 复杂计算拆分为多步骤
- 使用名称管理器定义常用日期

### **3. 性能考虑**
- 大量TODAY()/NOW()函数会影响计算速度
- 可改为引用同一单元格：`=TODAY()` 放在A1，其他地方用 `=$A$1`

### **4. 版本兼容**
- Excel 2003支持所有基础函数
- EOMONTH、NETWORKDAYS等需要分析工具库（旧版本）
- Excel 365新增动态数组函数

## **十、记忆技巧**

### **函数名称记忆**
- **DATE**：构造日期（需要3个参数）
- **TODAY**：今天（无参数，想到"今天"）
- **NOW**：现在（无参数，包含时间）
- **YEAR/MONTH/DAY**：年月日（提取各部分）

### **实用口诀**
> 构造日期用DATE，三个参数别忘记  
> 今天日期TODAY，动态更新要留意  
> 包含时间用NOW，精确到秒真详细  
> 提取年月日不难，三个函数分别记  
> 日期计算很常用，组合使用出奇迹

## **六、参考答案提示**

### **关键函数使用示例**

```excel
-- 1. 计算工作日（排除周末）
=NETWORKDAYS(开始日期, 结束日期)

-- 2. 动态日期范围
本月第一天：=DATE(YEAR(TODAY()), MONTH(TODAY()), 1)
本月最后一天：=EOMONTH(TODAY(), 0)

-- 3. 年龄精确计算
=DATEDIF(出生日期, TODAY(), "Y") & "岁" & 
  DATEDIF(出生日期, TODAY(), "YM") & "个月"

-- 4. 时间差计算
上班时长：=(下班时间-上班时间)*24  -- 小时数
加班时间：=MAX(0, (下班时间-TIME(18,0,0))*24)

-- 5. 智能提醒
=IF(TODAY()>=到期日-7, "即将到期", "")
```
