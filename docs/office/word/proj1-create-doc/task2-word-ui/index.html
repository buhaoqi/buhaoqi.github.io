task2-word-ui

剪贴板组：​

粘贴：将剪切或复制的内容插入到光标处。

剪切：移除选中内容并将其暂存，以便移动。

复制：复制选中内容，以便重复使用。

格式刷：快速将选中文本的格式应用到其他文本。

字体组：​

设置文本的字体、字号、加粗、倾斜、下划线、颜色、突出显示等外观样式。

段落组：​

设置文本的对齐方式、行距、项目符号、编号、缩进、排序及显示编辑标记。

段落间距：

  • 行距：

    • 单倍行距：将行距设置为当前字体高度的1倍（约为字体高度的120%），是Word的默认标准间距。
    • 多倍行距：按指定的倍数（如1.2、2.5等）灵活调整单倍行距的倍数。
    • 固定值：为每行文字设置一个绝对高度（单位为磅），行高将严格固定，不随字体大小自动调整。
    • 最小值：设置一个最小的行高值（单位为磅），当字体高度小于此值时行距自动调整，大于此值时则以此值为准。
    • 1.5倍行距：将单倍行距扩大为1.5倍，使行距明显变宽。
    • 2倍行距：将单倍行距扩大为2倍，形成非常宽松的行间距。

  • 文档网格：一套排版规则和不可见的坐标系统，强制控制文本行数、字符数和字符间距。

文档网格和视图选项卡下的网格线是两个不同层面、功能迥异的概念。

特性文档网格（布局网格）网格线（视图参考线）
本质一套排版规则和不可见的坐标系统可见的视觉辅助线
作用强制控制文本行数、字符数和字符间距辅助对齐图形、文本框等对象
控制对象字符和段落（影响行距和字符间距）浮动对象（如图片、形状、文本框）
位置“布局”选项卡 → “页面设置” → “文档网格”选项卡“视图”选项卡 → “显示”组 → “网格线”
可见性永远不可见，只能通过文本排列效果感知可以显示或隐藏的虚线
类比建筑蓝图中的承重墙和柱网（决定结构）施工时在地面画的定位辅助线（帮助对齐）

  • 中文版式：

    1. 纵横混排：将选中的少量文字（通常用于数字或英文）在竖直排版中横向显示，或在横向排版中竖直显示，使其符合阅读习惯。例如将竖直排版中的“2024”数字躺倒排列，便于阅读。
    2. 合并字符：将最多6个字符在水平方向压缩合并成上下两行，形成一个类似双行合一的单字符效果，常用于制作特殊格式的短标题或抬头。
    3. 双行合一：将选中的文本（字符数较多）在原有行宽内排成上下两行，且这两行在逻辑上被视为一行。常用于制作公文的联合发文单位格式（如“XX部/XX局通知”的联合落款）。
    4. 适应文字：自动调整所选文本的字符间距，使其恰好撑满指定的字符宽度（如将5个字压缩到与3个字同宽），常用于标题或特定格式的精确对齐
    5. 文字缩放：按指定的百分比水平缩放所选文字的宽度（高度不变），从而改变字形的胖瘦比例。这是视觉上的字形变形，而非字符间距调整。

  • 排序：简单说，“排序”就是让Word帮你把杂乱的内容按顺序排好队。它不仅可以排文字，还能排数字和日期。“排序”按钮的本质是一个“文本数据整理器”，它能将选中的列表、段落或表格行，按照你设定的规则（是什么类型的数据，按升序还是降序）进行自动化重新排列。

样式组：​

为文本快速应用预设或自定义的格式集合（样式），以保持文档格式统一。

编辑组：​

查找/替换/选择：用于在文档中定位、修改或批量处理特定内容。

  • 加载项：装的第三方程序、插件，扩展Word的功能。
  • 编辑器：微软推出的高级校对工具，可以看作是Word原有“拼写和语法检查”功能的智能升级版，是一个基于人工智能的智能写作辅助和校对工具。
