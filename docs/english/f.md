### 2. **Function** [ˈfʌŋkʃn]  
**中文**：函数  
**短语**：  
- Call a function (调用函数)  
- Pure function (纯函数)  
**场景**：  
```javascript
// Define function
function calculateArea(width, height) {
  return width * height;
}
// Call function
const area = calculateArea(5, 4); // Returns 20
```
