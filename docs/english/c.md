### 7. **Class** [klɑːs]  
**中文**：类  
**短语**：  
- Abstract class (抽象类)  
- Class inheritance (类继承)  
**场景**：  
```java
// Define class
public class Vehicle {
  private String model;
  
  public void startEngine() {
    System.out.println("Engine started");
  }
}
```

### 4. **Conditional** [kənˈdɪʃənl]  
**中文**：条件语句  
**短语**：  
- Conditional statement (条件语句)  
- Ternary conditional (三元表达式)  
**场景**：  
```python
# Check user role
if role == "admin":
    show_admin_panel()
elif role == "user":
    show_user_dashboard()
else:
    show_error()
```
