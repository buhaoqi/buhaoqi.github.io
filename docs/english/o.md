### 6. **Object** [ˈɒbdʒɪkt]  
**中文**：对象  
**短语**：  
- JSON object (JSON对象)  
- Immutable object (不可变对象)  
**场景**：  
```javascript
// User object
const user = {
  id: 101,
  name: "Alice",
  login: function() { console.log("Logged in") }
};
user.login(); // Call object method
```
