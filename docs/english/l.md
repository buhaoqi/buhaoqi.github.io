### 3. **Loop** [luːp]  
**中文**：循环  
**短语**：  
- For loop (for循环)  
- Infinite loop (无限循环)  
**场景**：  
```java
// Loop through array
for (int i = 0; i < 5; i++) {
  System.out.println("Iteration: " + i);
}
```
