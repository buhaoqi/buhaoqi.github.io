    3.еҲӣе»әз¬¬дёҖдёӘзЁӢеәҸ

    ејҖеңә

    рҹҺҷпёҸ вҖңеӨ§е®¶еҘҪпјҒж¬ўиҝҺи§ӮзңӢгҖҠC#еҲқеӯҰиҖ…е…Ҙй—Ёж•ҷзЁӢгҖӢзҡ„з¬¬3иҜҫгҖҠеҲӣе»әз¬¬дёҖдёӘзЁӢеәҸгҖӢ

    еңЁдёҠжңҹи§Ҷйў‘дёӯпјҢжҲ‘д»¬дәҶи§Јзұ»дёҺеҜ№иұЎзҡ„жҰӮеҝөгҖӮ

    еңЁжң¬жңҹи§Ҷйў‘дёӯпјҡжҲ‘е°ҶеёҰйўҶдҪ дёҖжӯҘдёҖжӯҘеҲӣе»әдёҖдёӘжҺ§еҲ¶еҸ°еә”з”ЁзЁӢеәҸгҖӮ

    д»ҺеҲӣе»әйЎ№зӣ®з»“жһ„еҲ°иҝҗиЎҢзЁӢеәҸд»Јз ҒпјҢжҡӮж—¶дёҚйңҖиҰҒе®Ңе…ЁзҗҶи§ЈжҜҸдёҖжӯҘйӘӨзҡ„еҗ«д№үпјҢе…ҲеӯҰдјҡжҠҠзЁӢеәҸвҖңи·‘вҖқиө·е•ҰпјҢеҶҚеҺ»ж…ўж…ўвҖңзҗҶи§ЈвҖқзЁӢеәҸз»“жһ„гҖӮвҖқи·‘йҖҡзЁӢеәҸвҖңжҳҜд»ҠеӨ©зҡ„йҮҚзӮ№пјҢзҗҶи§ЈжҳҜдёӢиҠӮиҜҫзҡ„д»»еҠЎгҖӮ

    еҗҜеҠЁз•Ңйқў

    жү“ејҖVisual Studioд№ӢеҗҺпјҢйҰ–е…ҲзңӢеҲ°зҡ„жҳҜеҗҜеҠЁз•ҢйқўгҖӮ

    еңЁеҗҜеҠЁз•ҢйқўдёӢпјҢз®ҖеҚ•иҜҙпјҢжҲ‘д»¬еҸӘеҒҡдёӨд»¶дәӢпјҡ

    1. еҲӣе»әж–°йЎ№зӣ®
    2. жү“ејҖе·ІжңүйЎ№зӣ®пјҡж— и®әжҳҜе…ӢйҡҶеӯҳеӮЁеә“гҖҒжү“ејҖйЎ№зӣ®иҝҳжҳҜжү“ејҖжң¬ең°ж–Үд»¶еӨ№йғҪеұһдәҺжү“ејҖе·ІжңүйЎ№зӣ®гҖӮ

    еңЁиҝҷйҮҢпјҢжҲ‘д»¬йҖүжӢ©вҖңеҲӣе»әж–°йЎ№зӣ®вҖқгҖӮ

    еҲӣе»әж–°йЎ№зӣ®

    вҖңеҲӣе»әж–°йЎ№зӣ®вҖқзӘ—еҸЈжҸҗдҫӣдәҶеҗ„з§Қзұ»еһӢзҡ„йЎ№зӣ®жЁЎжқҝпјҢжҜҸз§ҚжЁЎжқҝйғҪжҳҜдёҖз»„йў„е®ҡд№үзҡ„вҖңйЎ№зӣ®з»“жһ„ж–Үд»¶вҖңпјҢз”ЁжқҘеё®жҲ‘д»¬еҝ«йҖҹеҲӣе»әеә”з”ЁзЁӢеәҸгҖӮ

    жіЁж„ҸзңӢеҸідёҠж–№зҡ„зӯӣйҖүжқЎд»¶вҖ”вҖ”жҲ‘д»¬йңҖиҰҒеңЁпјҡ

    • вҖҳжүҖжңүиҜӯиЁҖвҖҷдёӢжӢүжЎҶйҮҢйҖүжӢ©вҖҳC#вҖҷ
    • з„¶еҗҺеңЁвҖҳе№іеҸ°вҖҷйҮҢйҖүжӢ©вҖҳWindowsвҖҷ
    • жңҖеҗҺеңЁвҖҳйЎ№зӣ®зұ»еһӢвҖҷйҮҢжүҫеҲ°вҖҳжҺ§еҲ¶еҸ°вҖҷпјҡжҺ§еҲ¶еҸ°еә”з”ЁдёҚйңҖиҰҒеӣҫеҪўз•ҢйқўпјҢеҸӘеңЁжҺ§еҲ¶еҸ°дёӯеұ•зӨәжҲ‘д»¬зҡ„еә”з”ЁзЁӢеәҸпјҢйқһеёёйҖӮеҗҲеҲқеӯҰиҖ…еӯҰд№ C#иҜӯжі•гҖӮ

    вҖңиҝҷйҮҢеҲ—еҮәдәҶдёӨдёӘвҖқжҺ§еҲ¶еҸ°еә”з”ЁвҖңзҡ„жЁЎжқҝпјҡ

    дёҠйқўжЁЎжқҝж”ҜжҢҒи·Ёе№іеҸ°пјҢдёӢйқўд»…йҷҗWindowsвҖ”вҖ”жҲ‘д»¬йҖүдёӢйқўиҝҷдёӘпјҢзӮ№еҮ»дёӢдёҖжӯҘвҖқ иҝӣе…ҘвҖңй…ҚзҪ®ж–°йЎ№зӣ®вҖқзӘ—еҸЈгҖӮ

    й…ҚзҪ®ж–°йЎ№зӣ®

    й…ҚзҪ®ж–°йЎ№зӣ®зҡ„ж„ҸжҖқе°ұжҳҜпјҡй…ҚзҪ®ж–°йЎ№зӣ®зҡ„еӯҳеӮЁдҪҚзҪ®гҖӮ

    еҸҜд»ҘзңӢеҲ°пјҢеңЁй»ҳи®Өжғ…еҶөдёӢпјҢ

    • йЎ№зӣ®еҗҚз§°еҸ«пјҡConsoleApp1
    • и§ЈеҶіж–№жЎҲеҗҚз§°д№ҹеҸ«пјҡConsoleApp1
    • дҪҚзҪ®вҖҷеҶіе®ҡдәҶйЎ№зӣ®ж–Үд»¶еӨ№дҝқеӯҳеңЁдҪ з”өи„‘зҡ„е“ӘдёӘдҪҚзҪ®

    еңЁиҝҷйҮҢпјҡ

    • жҠҠйЎ№зӣ®еҗҚз§°дҝ®ж”№дёәпјҡLesson2 е®ғжҳҜеӯҳеӮЁйЎ№зӣ®ж–Үд»¶зҡ„ж–Үд»¶еӨ№гҖӮжіЁж„ҸпјҢеҗҚз§°дёӯжҜҸдёӘеҚ•иҜҚзҡ„йҰ–еӯ—жҜҚе»әи®®еӨ§еҶҷгҖӮ
    • жҠҠи§ЈеҶіж–№жЎҲзҡ„еҗҚз§°дҝ®ж”№дёәпјҡLearnCSharpпјҢе®ғжҳҜеӯҳеӮЁйЎ№зӣ®Lesson2зҡ„ж–Үд»¶еӨ№еҗҚз§°.
    • еҸҜд»ҘеңЁжңҖеә•йғЁзҡ„жҸҗзӨәдҝЎжҒҜдёӯзңӢеҲ°е®ғд»¬зҡ„е…ізі»

    и§ЈеҶіж–№жЎҲе’ҢйЎ№зӣ®д№Ӣй—ҙзҡ„е…ізі»жҳҜпјҡ

    • и§ЈеҶіж–№жЎҲLearnCSharpпјҡжҳҜдёҖдёӘе®№еҷЁпјҢе°ұеғҸд№ҰеҢ…гҖӮ
    • йЎ№зӣ®Lesson2: жҳҜи§ЈеҶіж–№жЎҲйҮҢйқўиЈ…зқҖзҡ„е…·дҪ“зҡ„йЎ№зӣ®пјҢLesson2е°ұеғҸдёҖжң¬д№ҰгҖӮ
    • еҚ•жң¬д№Ұж—¶пјҢжҲ‘д»¬еҸҜд»ҘдёҚйңҖиҰҒд№ҰеҢ…пјҢеӢҫйҖүвҖҳе°Ҷи§ЈеҶіж–№жЎҲе’ҢйЎ№зӣ®ж”ҫеңЁеҗҢдёҖзӣ®еҪ•дёӯвҖҷиҝҷж ·еҸҜд»ҘеҮҸе°‘ж–Үд»¶еӨ№еұӮзә§пјҢз»“жһ„жӣҙз®ҖжҙҒгҖӮ
    • дҪҚзҪ®: жҳҜеӯҳеӮЁи§ЈеҶіж–№жЎҲзҡ„жң¬ең°и·Ҝеҫ„гҖӮеңЁиҝҷйҮҢпјҢжҲ‘еёҢжңӣжҠҠйЎ№зӣ®еӯҳеӮЁеңЁжЎҢйқўдёҠпјҡзӮ№еҮ»зҡ„вҖңдҪҚзҪ®вҖқеҗҺйқўзҡ„вҖңдёүзӮ№вҖқеӣҫж ҮпјҢзӮ№еҮ»жЎҢйқўпјҢж–°е»әдёҖдёӘж–Үд»¶еӨ№пјҢе‘ҪеҗҚдёәвҖңprojectsвҖқпјҢзӮ№еҮ»вҖңйҖүжӢ©ж–Үд»¶еӨ№вҖқпјҢиҝҷж ·е°ұи®ҫзҪ®еҘҪдәҶи§ЈеҶіж–№жЎҲзҡ„еӯҳеӮЁдҪҚзҪ®гҖӮ
    • жЎҶжһ¶пјҡиҝҷйҮҢдҝқжҢҒй»ҳи®Өзҡ„.NET Framework 4.8.1пјҢиҝҷжҳҜжңҖзЁіе®ҡзҡ„Windowsе…је®№зүҲжң¬гҖӮ
