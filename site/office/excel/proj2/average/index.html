    AVERAGE函数

    Excel平均值函数详解：AVERAGE()、AVERAGEIF()、AVERAGEIFS()

    一、函数家族概述¶

    1. 核心函数对比表

    函数 语法 功能 条件数量 Excel版本
    AVERAGE() =AVERAGE(number1, [number2], ...) 计算算术平均值 无条件 所有版本
    AVERAGEIF() =AVERAGEIF(range, criteria, [average_range]) 单条件平均值 单条件 Excel 2007+
    AVERAGEIFS() =AVERAGEIFS(average_range, criteria_range1, criteria1, ...) 多条件平均值 多条件 Excel 2007+

    2. 重要特性

    • 忽略文本、逻辑值、空单元格
    • 最多255个参数（AVERAGE）
    • 最多127个条件对（AVERAGEIFS）
    • 支持通配符（* ? ~）
    • 返回数字类型

    一、AVERAGE() 函数¶

    用途：基础平均值计算

    基本语法

    =AVERAGE(number1, [number2], ...)

    参数特性

    • 忽略文本、逻辑值、空单元格（除非直接作为参数）
    • TRUE视为1，FALSE视为0（仅当直接作为参数时）
    • 支持最多255个参数

    使用示例

    示例1：直接数值计算

    =AVERAGE(10, 20, 30)          // 结果：20
=AVERAGE(10, 20, 30, 0)       // 结果：15

    示例2：单元格区域计算

    A1=10, A2=20, A3=30, A4="文本", A5为空
=AVERAGE(A1:A5)               // 结果：20 (只计算10,20,30，忽略文本和空单元格)

    示例3：混合参数计算

    =AVERAGE(A1:A3, 40, B1:B2)    // 区域+数值+区域

    示例4：特殊值处理

    A1=TRUE, A2=10, A3=FALSE
=AVERAGE(A1:A3)               // 结果：10 (只计算10，TRUE和FALSE被忽略)
=AVERAGE(TRUE, 10, FALSE)     // 结果：3.6667 (1+10+0)/3

    二、AVERAGEIF() 函数：¶

    用途：单条件平均值

    基本语法

    =AVERAGEIF(range, criteria, [average_range])

    参数详解

    参数 说明 必需
    range 条件判断区域 是
    criteria 条件表达式 是
    average_range 实际求平均的区域（可选，省略时对range求平均） 否

    文本条件

    =AVERAGEIF(A:A,"苹果",B:B)      // A列为"苹果"时，求B列平均值
=AVERAGEIF(A:A,"*果",B:B)       // 以"果"结尾
=AVERAGEIF(A:A,"苹*",B:B)       // 以"苹"开头
=AVERAGEIF(A:A,"<>苹果",B:B)    // 不等于"苹果"

    数字条件

    =AVERAGEIF(B:B,">60")           // 大于60的平均值
=AVERAGEIF(B:B,">=80")          // 大于等于80
=AVERAGEIF(B:B,"<>0")           // 不等于0
=AVERAGEIF(B:B,75)              // 等于75（通常无意义，但语法正确）

    日期条件

    =AVERAGEIF(C:C,">=2024-01-01",D:D)
=AVERAGEIF(C:C,">"&DATE(2024,6,1),D:D)

    单元格引用作为条件

    =AVERAGEIF(A:A,D2,B:B)          // D2的内容作为条件
=AVERAGEIF(B:B,">"&E2)          // 大于E2的值

    示例1：学生成绩分析

    A列：班级  B列：成绩

1班平均分：=AVERAGEIF(A:A,"1班",B:B)
及格(>=60)平均分：=AVERAGEIF(B:B,">=60")
姓"王"的学生平均分：=AVERAGEIF(A:A,"王*",B:B)
排除缺考(0分)：=AVERAGEIF(B:B,"<>0")

    示例2：销售数据分析

    A列：产品  B列：销售额  C列：利润

手机平均销售额：=AVERAGEIF(A:A,"手机",B:B)
高销售额(>10000)平均利润：=AVERAGEIF(B:B,">10000",C:C)
非促销产品平均销售额：=AVERAGEIF(D:D,"<>促销",B:B)

    5. 常见错误示例

    错误：=AVERAGEIF(A:A,">D2",B:B)      // D2被当作文本
正确：=AVERAGEIF(A:A,">"&D2,B:B)    // 使用&连接

错误：=AVERAGEIF(A1:A10,"条件",B1:B9) // 区域大小不一致
正确：=AVERAGEIF(A1:A10,"条件",B1:B10)

    三、AVERAGEIFS() 函数¶

    用途：多条件平均值

    基本语法

    =AVERAGEIFS(average_range, criteria_range1, criteria1, 
            [criteria_range2, criteria2], ...)

    参数说明

    参数 说明
    average_range 要求平均的实际区域（第一个参数！）
    criteria_range1 第一个条件的区域
    criteria1 第一个条件
    criteria_range2, criteria2, ... 可选，最多127个条件对

    示例1：双条件平均值

    A列：部门  B列：职级  C列：工资

求"技术部""高级"员工的平均工资：
=AVERAGEIFS(C:C, A:A,"技术部", B:B,"高级")

    示例2：多条件组合

    求"销售部"在"北京"且工龄>5年的平均业绩：
=AVERAGEIFS(业绩列, 部门列,"销售部", 地区列,"北京", 工龄列,">5")

    示例3：日期范围条件

    A列：日期  B列：温度

求2024年夏季(6-8月)平均温度：
=AVERAGEIFS(B:B, A:A,">=2024-06-01", A:A,"<=2024-08-31")

    场景1：员工绩效分析

    数据表：部门 | 职级 | 工龄 | 绩效分数 | 奖金

1. 技术部P7级平均绩效：
   =AVERAGEIFS(绩效分数列, 部门列,"技术部", 职级列,"P7")

2. 工龄3-5年且绩效>85的平均奖金：
   =AVERAGEIFS(奖金列, 工龄列,">=3", 工龄列,"<=5", 绩效分数列,">85")

3. 排除试用期员工的平均绩效：
   =AVERAGEIFS(绩效分数列, 状态列,"<>试用期")

    场景2：产品质量分析

    数据表：生产线 | 班次 | 日期 | 产品型号 | 合格率

1. A生产线早班的平均合格率：
   =AVERAGEIFS(合格率列, 生产线列,"A线", 班次列,"早班")

2. 2024年Q2型号X100的平均合格率：
   =AVERAGEIFS(合格率列, 产品型号列,"X100", 
               日期列,">=2024-04-01", 日期列,"<=2024-06-30")

3. 合格率>95%且产量>1000的平均数据：
   =AVERAGEIFS(某指标列, 合格率列,">0.95", 产量列,">1000")

    场景3：销售数据分析

    数据表：销售员 | 产品类别 | 季度 | 销售额 | 客户评分

1. 张三在Q2销售电器的平均销售额：
   =AVERAGEIFS(销售额列, 销售员列,"张三", 
               产品类别列,"电器", 季度列,"Q2")

2. 客户评分>4且销售额>10000的平均利润率：
   =AVERAGEIFS(利润率列, 客户评分列,">4", 销售额列,">10000")

3. 排除退货订单的平均交易额：
   =AVERAGEIFS(交易额列, 订单状态列,"<>退货")

    四、三函数对比总结¶

    特性 AVERAGE() AVERAGEIF() AVERAGEIFS()
    条件数量 无条件 单条件 多条件
    语法顺序 AVERAGE(求值区域) AVERAGEIF(条件区域,条件,求值区域) AVERAGEIFS(求值区域,条件区域1,条件1,...)
    求值区域位置 直接参数 第三个参数 第一个参数
    忽略的值 文本、空单元格、逻辑值（在区域中） 同AVERAGE 同AVERAGE
    通配符支持 不支持 支持 支持
    性能 最快 中等 较慢
    Excel版本 所有版本 Excel 2007+ Excel 2007+

    五、综合应用示例¶

    完整数据表示例

    A B C D E
    部门 姓名 职级 入职年份 月薪
    技术部 张三 P7 2018 25000
    销售部 李四 P6 2019 18000
    技术部 王五 P8 2017 30000
    人事部 赵六 P5 2020 15000
    技术部 孙七 P7 2019 23000
    销售部 周八 P6 2021 17000

    各种平均值计算

    1. 全体员工平均月薪：=AVERAGE(E:E)                    // 结果：21333

2. 技术部平均月薪：
   =AVERAGEIF(A:A,"技术部",E:E)                        // 结果：26000

3. P7级平均月薪：
   =AVERAGEIF(C:C,"P7",E:E)                           // 结果：24000

4. 技术部P7级平均月薪：
   =AVERAGEIFS(E:E, A:A,"技术部", C:C,"P7")           // 结果：24000

5. 2019年及以后入职的平均月薪：
   =AVERAGEIFS(E:E, D:D,">=2019")                     // 结果：18250

6. 技术部且职级>=P7的平均月薪：
   =AVERAGEIFS(E:E, A:A,"技术部", C:C,">=P7")         // 结果：26000

7. 排除人事部的平均月薪：
   =AVERAGEIF(A:A,"<>人事部",E:E)                     // 结果：22600

    AVERAGE()实际应用示例¶

    示例1：计算学生平均分

    在表1的K列（个人平均分）：
=AVERAGE(D2:H2)

结果：
张三：(85+92+78+88+90)/5 = 86.6
李四：(78+95+88+75+82)/5 = 83.6
注意：忽略文本、逻辑值、空单元格

    示例2：计算班级平均成绩

    计算1班语文平均分：
=AVERAGEIF(C2:C20, "1班", D2:D20)

先用AVERAGE计算所有学生语文平均：
=AVERAGE(D2:D20)

结果：计算D列所有数值的平均值

    示例3：计算多列平均值

    计算所有理科平均分（物理+化学）：
=AVERAGE(G2:H20)  // G列物理，H列化学

或者分列计算：
=(AVERAGE(G2:G20) + AVERAGE(H2:H20))/2

    示例4：计算员工季度平均业绩

    在表2的H列（季度平均）：
=AVERAGE(D2:F2)

结果：
张明：(158000+142000+165000)/3 = 155000
注意：已预计算好，演示公式

    AVERAGE的特殊情况处理

    忽略0值计算平均值

    假设有0值表示缺考，要忽略0值：
=AVERAGEIF(D2:D20, "<>0")

或者数组公式：
=AVERAGE(IF(D2:D20>0, D2:D20))

    处理错误值

    使用IFERROR：
=AVERAGE(IFERROR(D2:D20, ""))

或者AGGREGATE：
=AGGREGATE(1, 6, D2:D20)  // 1=平均值，6=忽略错误

    AVERAGEIF()实际应用示例¶

    示例5：计算班级平均分

range: C2:C20（班级列）

    AVERAGEIFS()实际应用示例¶

    示例12：多条件班级平均分

    计算1班且寄宿的学生语文平均分：
=AVERAGEIFS(D2:D20, C2:C20, "1班", I2:I20, "是")

解释：
average_range: D2:D20（语文成绩）
criteria_range1: C2:C20（班级），criteria1: "1班"
criteria_range2: I2:I20（是否寄宿），criteria2: "是"

    示例13：计算优秀学生平均分

    计算数学≥90且英语≥90的学生平均分：
=AVERAGEIFS(D2:D20, E2:E20, ">=90", F2:F20, ">=90")

结果：数学和英语都≥90的学生的语文平均分

    示例14：计算寄宿班干部平均成绩

    计算寄宿且是班干部的学生数学平均分：
=AVERAGEIFS(E2:E20, I2:I20, "是", J2:J20, "是")

    示例15：销售部高绩效员工平均

    在表2计算销售部且职级A的员工平均业绩：
=AVERAGEIFS(G2:G16, C2:C16, "销售部", H2:H16, "A")

解释：
average_range: G2:G16（季度平均）
criteria_range1: C2:C16（部门），criteria1: "销售部"
criteria_range2: H2:H16（职级），criteria2: "A"

    示例16：多条件产品分析

    在表3计算手机类别且促销中的产品平均售价：
=AVERAGEIFS(H2:H13, C2:C13, "手机", J2:J13, "是")

结果：类别为"手机"且正在促销的产品平均售价

    示例17：复杂条件平均值

    计算库存充足（>安全库存）且销量>100的产品平均成本：
=AVERAGEIFS(I2:I13, D2:D13, ">"&E2:E13, F2:F13, ">100")

注意：这里E2:E13需要逐行比较，实际应该为：
=AVERAGEIFS(I2:I13, D2:D13, ">="&E2:E13, F2:F13, ">100")
但Excel不支持数组条件，正确做法：
使用SUMPRODUCT或辅助列

    示例18：排除特定条件计算

    计算非食品类且非促销产品的平均库存：
=AVERAGEIFS(D2:D13, C2:C13, "<>食品", J2:J13, "否")

结果：类别不是食品且非促销的产品平均库存量

    示例19：多条件日期范围

    假设表2有月份列，计算Q1销售部业绩：
=AVERAGEIFS(业绩列, 部门列, "销售部", 日期列, ">=2024-01-01", 日期列, "<=2024-03-31")

    六、常见问题与解决方案¶

    Q1：为什么平均值显示为#DIV/0!？¶

    原因：没有满足条件的数值
解决：=IFERROR(AVERAGEIF(...), "无数据")

    Q2：如何计算加权平均值？¶

    普通平均值：=AVERAGE(B2:B10)
加权平均值：=SUMPRODUCT(B2:B10, C2:C10)/SUM(C2:C10)

    Q3：如何处理包含0值的平均值？¶

    包含0：=AVERAGE(A1:A10)
排除0：=AVERAGEIF(A1:A10,"<>0")

    Q4：如何计算前N个最高值的平均值？¶

    前3名平均值：=AVERAGE(LARGE(A1:A100, {1,2,3}))

    Q5：AVERAGE与AVERAGEA的区别？¶

    数据：A1=10, A2="缺考", A3=TRUE, A4=20

AVERAGE(A1:A4)    // 结果：15 (只计算10,20)
AVERAGEA(A1:A4)   // 结果：7.75 (10+0+1+20)/4

    Q6：如何忽略错误值计算平均值？¶

    =AVERAGEIF(A1:A10,"<>#N/A")
=AGGREGATE(1, 6, A1:A10)  // 忽略所有错误

    七、实际应用模板¶

    模板1：学生成绩管理系统¶

    A列：学号  B列：姓名  C列：班级  D列：语文  E列：数学  F列：英语

1. 各科全班平均分：
   语文：=AVERAGE(D:D)
   数学：=AVERAGE(E:E)
   英语：=AVERAGE(F:F)

   =AVERAGEIFS(E:E, C:C,"1班")

   语文：=AVERAGEIF(D:D,">=60")

   英语：=AVERAGEIF(F:F,">=60")

4. 1班且三科都及格的平均总分：
   =AVERAGEIFS(G:G, C:C,"1班", D:D,">=60", E:E,">=60", F:F,">=60")

    模板2：销售业绩分析系统¶

    A列：销售员  B列：区域  C列：季度  D列：销售额  E列：利润率

1. 各区域平均销售额：
   华东：=AVERAGEIFS(D:D, B:B,"华东")
   华南：=AVERAGEIFS(D:D, B:B,"华南")
   华北：=AVERAGEIFS(D:D, B:B,"华北")

2. Q2高销售额(>100000)的平均利润率：
   =AVERAGEIFS(E:E, C:C,"Q2", D:D,">100000")

3. 各销售员达标(销售额>目标)的平均超额比例：
   =AVERAGEIFS(超额比例列, D:D,">"&目标列)

    十、总结¶

    AVERAGE家族是Excel的"平均计算器"——AVERAGE是基础款，AVERAGEIF是智能款，AVERAGEIFS是专业款，满足你从简单到复杂的所有平均值计算需求！

    记忆口诀 

    AVERAGE求平均，数值相加除个数
AVERAGEIF加条件，筛选之后再平均
AVERAGEIFS多条件，复杂分析真方便
参数顺序要记清，区域大小要一致
