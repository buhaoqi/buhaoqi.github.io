    6月29日:IF练习(40道)

    关系判断题（10 题）

    1. 判断一个整数是否为正数。
    2. 判断一个整数是否是偶数。
    3. 判断某人是否成年（年龄 ≥ 18）。
    4. 判断成绩是否在 60～100 之间。
    5. 判断两个变量 a 和 b 是否相等。
    6. 判断一个布尔变量 isOnline 是否为 true。
    7. 判断一个字符是否是大写英文字母（A\~Z）。
    8. 判断一个字符串是否非空且长度超过 5。
    9. 判断某人是否是学生且年龄在 6\~18 岁之间。
    10. 判断是否不是工作日（isWeekend = true）且天气晴朗（isSunny = true）。

    关系判断题(答案)

    1. 判断一个整数是否为正数
    int number = 5;
if (number > 0)
    Console.WriteLine("是正数");
else
    Console.WriteLine("不是正数");

    ✅ 输出：是正数

    1. 判断一个整数是否是偶数
    int number = 6;
if (number % 2 == 0)
    Console.WriteLine("是偶数");
else
    Console.WriteLine("是奇数");

    ✅ 输出：是偶数

    1. 判断某人是否成年（年龄 ≥ 18）
    int age = 20;
if (age >= 18)
    Console.WriteLine("已成年");
else
    Console.WriteLine("未成年");


Console.Write("请输入出生年份：");
int birthYear = int.Parse(Console.ReadLine());

int currentYear = DateTime.Now.Year;
int age = currentYear - birthYear;

string result = (age >= 18) ? "已成年" : "未成年";
Console.WriteLine($"年龄：{age}，{result}");

    ✅ 输出：已成年

    1. 判断成绩是否在 60～100 之间
    int score = 85;
if (score >= 60 && score <= 100)
    Console.WriteLine("成绩合格");
else
    Console.WriteLine("成绩不合格");

    ✅ 输出：成绩合格

    1. 判断两个变量 a 和 b 是否相等
    int a = 10, b = 10;
if (a == b)
    Console.WriteLine("a 和 b 相等");
else
    Console.WriteLine("a 和 b 不相等");

    ✅ 输出：a 和 b 相等

    1. 判断一个布尔变量 isOnline 是否为 true
    bool isOnline = true;
if (isOnline)
    Console.WriteLine("用户在线");
else
    Console.WriteLine("用户离线");

    ✅ 输出：用户在线

    1. 判断一个字符是否是大写英文字母（A\~Z）
    char ch = 'G';
if (ch >= 'A' && ch <= 'Z')
    Console.WriteLine("是大写字母");
else
    Console.WriteLine("不是大写字母");

    ✅ 输出：是大写字母

    1. 判断一个字符串是否非空且长度超过 5
    string str = "ChatGPT";
if (!string.IsNullOrEmpty(str) && str.Length > 5)
    Console.WriteLine("字符串合法");
else
    Console.WriteLine("字符串不合法");

    ✅ 输出：字符串合法

    1. 判断某人是否是学生且年龄在 6\~18 岁之间
    bool isStudent = true;
int age = 12;
if (isStudent && age >= 6 && age <= 18)
    Console.WriteLine("是学生");
else
    Console.WriteLine("不是学生");

    ✅ 输出：是学生

    复合逻辑判断题（10 题）

    1. 判断一个数是否在 10\~20 之间，且不是 15。
    2. 判断一个数是否小于 0 或大于 100。
    3. 判断某人是否有会员资格（isVip = true）或者积分超过 1000。
    4. 判断某个商品是否缺货（库存为 0）或停售（isOnSale = false）。
    5. 判断用户是否既不是游客也不是管理员。
    6. 判断 a > b 且 b > c，从而判断是否递减。
    7. 判断某人是否具有访问权限（必须登录且权限等级为“管理员”）。
    8. 判断是否满足“仅一个条件为真”：hasCoupon ^ isDiscountDay。
    9. 判断输入密码是否正确且长度在 6\~12 位之间。
    10. 判断某设备是否在“运行状态”且未报警（isRunning && !isAlarm）。

    复合逻辑判断题（答案）

    1. 判断是否不是工作日且天气晴朗
    bool isWeekend = true;
bool isSunny = true;
if (isWeekend && isSunny)
    Console.WriteLine("适合出游");
else
    Console.WriteLine("不适合出游");

    ✅ 输出：适合出游

    1. 判断一个数是否在 10\~20 之间，且不是 15
    int num = 18;
if (num >= 10 && num <= 20 && num != 15)
    Console.WriteLine("符合条件");
else
    Console.WriteLine("不符合条件");

    ✅ 输出：符合条件

    1. 判断一个数是否小于 0 或大于 100
    int num = -5;
if (num < 0 || num > 100)
    Console.WriteLine("范围之外");
else
    Console.WriteLine("在范围内");

    ✅ 输出：范围之外

    1. 判断是否有会员资格（isVip）或者积分超过 1000
    bool isVip = false;
int points = 1200;
if (isVip || points > 1000)
    Console.WriteLine("有资格");
else
    Console.WriteLine("无资格");

    ✅ 输出：有资格

    1. 判断商品是否缺货（库存为0）或停售（isOnSale = false）
    int stock = 0;
bool isOnSale = true;
if (stock == 0 || !isOnSale)
    Console.WriteLine("不可购买");
else
    Console.WriteLine("可以购买");

    ✅ 输出：不可购买

    1. 判断用户不是游客也不是管理员
    bool isGuest = false;
bool isAdmin = false;
if (!isGuest && !isAdmin)
    Console.WriteLine("普通用户");
else
    Console.WriteLine("特殊身份");

    ✅ 输出：普通用户

    1. 判断 a > b 且 b > c 是否为递减序列
    int a = 10, b = 8, c = 5;
if (a > b && b > c)
    Console.WriteLine("递减序列");
else
    Console.WriteLine("不是递减序列");

    ✅ 输出：递减序列

    1. 判断是否已登录且权限为管理员
    bool isLoggedIn = true;
string role = "admin";
if (isLoggedIn && role == "admin")
    Console.WriteLine("有访问权限");
else
    Console.WriteLine("无访问权限");

    ✅ 输出：有访问权限

    1. 判断“仅一个条件为真”：hasCoupon ^ isDiscountDay
    bool hasCoupon = true;
bool isDiscountDay = false;
if (hasCoupon ^ isDiscountDay)
    Console.WriteLine("使用优惠");
else
    Console.WriteLine("不使用优惠");

    ✅ 输出：使用优惠

    1. 判断输入密码是否正确且长度在 6～12 位之间
    string inputPwd = "abc1234";
string correctPwd = "abc1234";
if (inputPwd == correctPwd && inputPwd.Length >= 6 && inputPwd.Length <= 12)
    Console.WriteLine("密码合法");
else
    Console.WriteLine("密码不合法");

    ✅ 输出：密码合法

    1. 判断设备是否在运行状态且未报警
    bool isRunning = true;
bool isAlarm = false;
if (isRunning && !isAlarm)
    Console.WriteLine("设备正常运行");
else
    Console.WriteLine("设备异常");

    ✅ 输出：设备正常运行

    基础题(5道)

    1. 温度判断
      输入一个温度值，如果高于30度输出"炎热"，低于10度输出"寒冷"，否则输出"舒适"

    2. 奇偶判断
      输入一个整数，判断并输出是奇数还是偶数

    3. 成绩等级
      输入成绩（0-100），输出等级：90+为A，80-89为B，70-79为C，60-69为D，低于60为F

    4. 登录验证
      预设用户名"admin"，密码"123456"。输入用户名密码，验证正确输出"登录成功"，否则输出"登录失败"

    5. 正负零判断
      输入一个数字，输出它是正数、负数还是零

    进阶题(5道)

    1. 闰年判断
      输入年份，判断是否为闰年（能被4整除但不能被100整除，或能被400整除）

    2. 三角形判断
      输入三个边长，判断是否能构成三角形（任意两边之和大于第三边）

    3. 最大数查找
      输入三个不同的整数，找出并输出最大值

    4. 字符类型识别
      输入一个字符，判断它是大写字母、小写字母、数字还是特殊字符

    5. 月份天数
      输入月份（1-12），输出该月天数（2月按28天计算）

    逻辑运算符应用(5道)

    1. 优惠折扣
      购物金额≥1000打9折，≥500打95折，否则不打折。输入金额计算实付金额

    2. BMI健康评估
      输入身高(m)和体重(kg)，计算BMI值：
      BMI < 18.5 → "过轻"
      18.5 ≤ BMI < 24 → "正常"
      24 ≤ BMI < 28 → "过重"
      BMI ≥ 28 → "肥胖"

    3. 二次方程根判断
      输入二次方程系数a,b,c，根据判别式输出：
      两个实根/一个实根/无实根（不考虑复数解）

    4. 三角形类型
      输入三角形三个边长，判断类型：等边、等腰、直角、普通或不能构成三角形

    5. 时间有效性
      输入时、分、秒，判断是否是有效时间（时0-23，分0-59，秒0-59）

    综合应用(5道)

    1. 成绩评语
      输入成绩和出勤率（0-100%），规则：

      • 成绩≥60且出勤≥80% → "通过"
      • 成绩≥60但出勤<80% → "补考"
      • 成绩<60 → "重修"

    2. 快递运费
      输入重量(kg)和距离(km)：

      • ≤1kg且≤100km：10元
      • ≤1kg但>100km：15元

      • 1kg：首重10元 + 续重5元/kg（不足1kg按1kg算）

    3. 日期合法性
      输入年、月、日，判断日期是否有效（考虑闰年和平年二月）

    4. 个人所得税
      输入月收入，按阶梯计算个税：

      • ≤5000：免税
      • 5001-8000：3%
      • 8001-17000：10%

      • 17000：20%

    5. 游戏角色状态
      角色有生命值(HP)、魔法值(MP)、状态(正常/中毒/冰冻)：

      • HP≤0 → "死亡"
      • 中毒且HP<20 → "危险"
      • 冰冻且MP<10 → "无法施法"
      • 其他情况 → "可战斗"
